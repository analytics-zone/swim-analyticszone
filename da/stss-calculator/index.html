Hvad er Svømme TSS (sTSS)?

Swimming Training Stress Score (sTSS) kvantificerer træningsbelastningen af en svømmetræning ved at kombinere intensitet og varighed. Det er tilpasset fra cyklings TSS-metodologi, ved at bruge din Critical Swim Speed (CSS) som tærskel-tempo. En 1-times træning ved CSS-tempo = 100 sTSS.

Træningsbelastnings Fortolkning

Sådan Beregnes sTSS

Formel

sTSS = (Varighed i timer) × (Intensitetsfaktor)² × 100

Hvor:

  • Intensitetsfaktor (IF) = CSS Tempo / Gennemsnitligt Trænings Tempo
  • Varighed = Total træningstid i timer
  • CSS Tempo = Dit tærskel-tempo fra CSS test

Udarbejdet Eksempel

Træningsdetaljer:

  • CSS Tempo: 1:49/100m (109 sekunder)
  • Træningsvarighed: 60 minutter (1 time)
  • Gennemsnitligt Tempo: 2:05/100m (125 sekunder)

Trin 1: Beregn Intensitetsfaktor

IF = CSS Tempo / Trænings Tempo
IF = 109 / 125
IF = 0.872

Trin 2: Beregn sTSS

sTSS = 1.0 timer × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

Fortolkning: Denne 60-minutters træning i let tempo (langsommere end CSS) genererede 76 sTSS - en moderat træningsbelastning velegnet til at opbygge aerob base.

Ugentlig sTSS Guide

Målrettet ugentlig sTSS afhænger af dit træningsniveau og mål:

Rekreative Svømmere

Ugentlig sTSS: 150-300

2-3 træninger per uge, 50-100 sTSS hver. Fokus på teknik og aerob base opbygning.

Fitness Svømmere / Triatleter

Ugentlig sTSS: 300-500

3-4 træninger per uge, 75-125 sTSS hver. Blanding af aerob udholdenhed og tærskel arbejde.

Konkurrence Masters Svømmere

Ugentlig sTSS: 500-800

4-6 træninger per uge, 80-150 sTSS hver. Struktureret træning med periodisering.

Elite / Universitets Svømmere

Ugentlig sTSS: 800-1200+

8-12 træninger per uge, dobbelt sessioner. Højt volumen med kritisk restitutionsstyring.

⚠️ Vigtige Noter

  • Kræver præcis CSS: Din CSS skal være opdateret (testet inden for 6-8 uger) for præcis sTSS.
  • Forenklet beregning: Denne beregner bruger gennemsnitligt tempo. Avanceret sTSS bruger Normalized Graded Pace (NGP) som tager højde for interval struktur.
  • Ikke til teknikarbejde: sTSS måler kun fysisk træningsstress, ikke færdighedsudvikling.
  • Individuel variation: Samme sTSS føles forskelligt for forskellige svømmere. Juster retningslinjer baseret på din restitution.

Hvorfor sTSS Betyder Noget

Training Stress Score er grundlaget for:

  • CTL (Chronic Training Load): Dit konditionsniveau - 42-dages eksponentielt vægtet gennemsnit af daglig sTSS
  • ATL (Acute Training Load): Din træthed - 7-dages eksponentielt vægtet gennemsnit af daglig sTSS
  • TSB (Training Stress Balance): Din form - TSB = CTL - ATL (positiv = frisk, negativ = træt)
  • Periodisering: Planlæg træningsfaser (base, opbygning, peak, taper) ved hjælp af målrettede CTL progressioner
  • Restitutionsstyring: Vid hvornår du skal presse på og hvornår du skal hvile baseret på TSB

Pro Tip: Spor Din CTL

Registrer daglig sTSS i et regneark eller træningsdagbog. Beregn dit 42-dages gennemsnit (CTL) ugentligt. Mål om 5-10 CTL point stigning per uge under base opbygning. Bevar eller reducer CTL lidt under taper (1-2 uger før konkurrence).

Ofte Stillede Spørgsmål

Hvad er Svømme TSS (sTSS)?

Swimming Training Stress Score (sTSS) er en metrik, der kvantificerer træningsbelastningen af en svømmetræning ved at kombinere både intensitet og varighed. Det er tilpasset fra cyklings TSS-metodologi, ved at bruge din Critical Swim Speed (CSS) som tærskel-tempo. En træning på 1 time ved CSS tempo svarer til 100 sTSS.

Hvordan beregner jeg min sTSS?

Brug beregneren ovenfor ved at indtaste dit CSS tempo (fra en CSS test), total træningsvarighed og gennemsnitligt tempo under træningen. Formlen er: sTSS = Varighed (timer) × Intensitetsfaktor² × 100, hvor Intensitetsfaktor = CSS Tempo / Gennemsnitligt Trænings Tempo.

Har jeg brug for CSS for at beregne sTSS?

Ja, din Critical Swim Speed (CSS) er påkrævet for at beregne Intensitetsfaktoren, som er essentiel for sTSS beregning. CSS repræsenterer dit tærskel-tempo og bør testes hver 6-8 uge. Du kan finde din CSS ved hjælp af vores CSS beregner.

Hvad er en god sTSS score for en træning?

Det afhænger af træningsintensitet: Lette træninger scorer typisk under 50 sTSS, moderate træninger 50-100 sTSS, hårde træninger 100-200 sTSS, og meget hårde træninger over 200 sTSS. Den passende score afhænger af dine træningsmål og nuværende fitnessniveau.

Hvor meget sTSS skal jeg lave om ugen?

Ugentlige sTSS mål varierer efter niveau: Rekreative svømmere: 150-300, Fitness svømmere/Triatleter: 300-500, Konkurrence Masters: 500-800, Elite/Universitets: 800-1200+. Start konservativt og øg gradvist for at undgå overtræning.

Er svømme-TSS det samme som cykel-TSS?

Konceptet og formlen er de samme, men sTSS er tilpasset svømning. I stedet for at bruge effekt (FTP) som cykel-TSS, bruger sTSS tempo med CSS som tærskel. Begge måler træningsbelastning ved at bruge Varighed × Intensitetsfaktor² × 100.

Kan jeg bruge sTSS til alle svømmeslag?

Ja, men din CSS bør være slagspecifik. De fleste svømmere bruger crawl CSS, da det er det mest trænede slag. Hvis du primært træner i et andet slag, udfør en CSS-test i det slag og brug det tempo til sTSS-beregning.

Hvad er forskellen mellem sTSS og CTL/ATL/TSB?

sTSS måler træningsbelastningen for en enkelt træning. CTL (Chronic Training Load) er din langsigtede form, ATL (Acute Training Load) er din nylige træthed, og TSB (Training Stress Balance) er din friskhed. Disse målinger bruger sTSS-værdier over tid til at spore din træningstilstand. Lær mere i vores Træningsbelastning guide.
