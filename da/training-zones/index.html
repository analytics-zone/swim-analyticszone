Svømmetræningszoner - CSS-baseret intensitet

7 zoner ud fra din CSS for tempo, restitution og raceforberedelse

Hvad er træningszoner?

Træningszoner er fart-/intensitetsområder baseret på din Critical Swim Speed (CSS). Hver zone målretter specifik fysiologisk adaptation (aerob base, tærskel, VO₂max).

Fordele ved CSS-baserede zoner

  • Personlig intensitet: Præcise farter ud fra din CSS
  • Kvantificeret belastning: Kernen til sTSS-beregning
  • Restitutionsstyring: Undgå overtræning, ram peak-timing

De 7 svømmezoner

Zone 4Tærskel95-102%Tærskeludvikling10×100 @ CSS, 15-20s pause
Zone 5VO₂max103-110%Maks O₂-optag, HIIT8×50 @ Z5, 30-45s pause
Zone 6Anaerob kapacitet111-120%Anaerob power, laktattolerance6×25 sprint, 45-60s pause
Zone 7Sprint> 120%Neuromuskulær power, start/turn12×15m undervandsdelfin + max accel

⚠️ Restitutionsregler

  • Zone 5: 48-72 timer mellem pas
  • Zone 6-7: Høj neuro/anaerob belastning – hold frekvensen lav
  • Undgå back-to-back højintensitet

CSS-test → personlige zoner

1) CSS-test

400m + 200m time trial med 5-10 min pause.

Se startguiden for protokol.

2) CSS-beregning

Indtast tider – appen beregner CSS og genererer 7 zoner automatisk.

3) Træn efter zoner

  • Zone 2: Aerob base (60-70% af ugen)
  • Zone 3-4: Tempo/tærskel for stabil fart
  • Zone 5+: Højintensivt i begrænset mængde

4) sTSS-kobling

IF og sTSS for hvert pas baseres på CSS/zoner. CSS-opdatering recalculerer de sidste 90 dages sTSS.

💡 Retest CSS

Base/build: hver 6-8 uge; peak: 4-6 uge; efter sygdom/skade/pause eller hvis zoner føles vedvarende for let/svære.

Praktisk fordeling

Begynder/Fitness

  • Ugevolumen: 6.000-12.000m (2-3 pas)
  • Zone 1: 15%
  • Zone 2: 55-65%
  • Zone 3: 10-15%
  • Zone 4: 5-10%
  • Zone 5+: 0-5%

Mellem/konkurrence

  • Ugevolumen: 12.000-20.000m (3-5 pas)
  • Zone 1: 10-15%
  • Zone 2: 45-55%
  • Zone 3: 15-20%
  • Zone 4: 10-15%
  • Zone 5+: 5-10%

Avanceret/elite

  • Ugevolumen: 20.000-35.000m (5-8 pas)
  • Zone 1: 10-15%
  • Zone 2: 40-50%
  • Zone 3: 15-20%
  • Zone 4: 10-15%
  • Zone 5+: 10-15%

IM/andre stilarter

CSS er typisk baseret på crawl. For IM/ryg/bryst: brug crawl-CSS som reference og justér efter følelse, eller lav stilartsspecifik CSS-test.

⚠️ Restitution

Efter Zone 5: 48-72t. Efter Zone 6-7 kan længere restitution være nødvendig. Undgå back-to-back højintensitet.

CSS-test og zoneberegning

  1. CSS-test: Tag tider på 400m + 200m.
  2. CSS-pace: (T400 - T200) / 2 → sek/100m.
  3. CSS-hastighed: 100m / CSS-pace (min) → m/min.
  4. Zonegrænser: CSS-hastighed × zoneprocenter.
