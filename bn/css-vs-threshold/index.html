সমালোচনামূলক সাঁতারের গতি (CSS) বনাম থ্রেশহোল্ড পেস: কোনটি গুরুত্বপূর্ণ?

কয়েক বছর ধরে, "থ্রেশহোল্ড পেস" (টি-পেস) ছিল সোনার মান। 2026 সালে, ক্রিটিক্যাল সুইম স্পীড (CSS)-এ স্থানান্তর করা হচ্ছে কীভাবে ক্রীড়াবিদরা তাদের সেশনে গতি আনতে পারে এবং রেস-ডে ক্লান্তির পূর্বাভাস দেয়।

🎯 দ্রুত উত্তর

টি-পেস (থ্রেশহোল্ড পেস) এটি একটি সাধারণ অনুমান যে গতি আপনি এক ঘন্টা ধরে রাখতে পারেন, প্রায়শই বিস্তৃত প্রশিক্ষণ অঞ্চল সেট করার জন্য ব্যবহৃত হয়। এটি একটি "ব্যবহারিক" কিন্তু কম দানাদার মেট্রিক।

CSS (গুরুত্বপূর্ণ সাঁতারের গতি) দুটি ভিন্ন দূরত্ব (যেমন, 400m এবং 200m) থেকে গাণিতিকভাবে প্রাপ্ত "কার্যকর" থ্রেশহোল্ড। এটি তাত্ত্বিক গতি যা আপনি "ওয়াল" আঘাত না করেই বজায় রাখতে পারেন। আপনি যদি একজন ধৈর্যশীল সাঁতারু বা ট্রায়াথলিট হন, আপনার প্রশিক্ষণ প্রোগ্রামে আপনার CSS হল সবচেয়ে গুরুত্বপূর্ণ নম্বর।

2026 শিফট: প্রচেষ্টা থেকে ফিজিওলজিতে

সাইকেল চালানো বা দৌড়ানোর তুলনায় পুলে ক্লান্তি নিয়ন্ত্রণ করা খুবই কঠিন। একটি পাওয়ার মিটার ছাড়া, গতি আমাদের তীব্রতার একমাত্র প্রক্সি। যদিও অনেক সাঁতারু এখনও পুরানো-স্কুল 1500 মিটার "টাইম-ট্রায়াল" গতির উপর নির্ভর করে, আধুনিক কোচিং চলে গেছে দুই-দূরত্বের CSS মডেল.

প্রধান পার্থক্য হল নির্ভুলতা বনাম অনুভূত প্রচেষ্টা. একটি দীর্ঘ প্রচেষ্টার উপর ভিত্তি করে একটি টি-পেস প্রায়শই আপনার বিপাকীয় থ্রেশহোল্ডের চেয়ে আপনার মানসিক দৃঢ়তাকে বেশি ক্যাপচার করে। CSS, আপনার "পেস-ডিউরেশন কার্ভ"-এর দুটি স্বতন্ত্র পয়েন্ট থেকে গণনা করা হয়, যা আপনার ইঞ্জিনের প্রকৃত বায়বীয় সীমা প্রকাশ করে।

মেট্রিক তুলনা

থ্রেশহোল্ড পেস (টি-পেস)

"সহজ" মেট্রিক

T-Pace সাধারণত একটি একক 1500m বা 1000m সর্বাধিক প্রচেষ্টা থেকে উদ্ভূত হয়। এটি একটি অবিচ্ছিন্ন সাঁতারের জন্য আপনার "রেডলাইন" প্রতিনিধিত্ব করে।

  • সুবিধা: বুঝতে সহজ. এক পরীক্ষা, এক নম্বর। দীর্ঘ, অবিচ্ছিন্ন সেটের জন্য দুর্দান্ত।
  • অসুবিধা: পেসিং ত্রুটির জন্য অত্যন্ত সংবেদনশীল। আপনি ক্লান্তিতে কত দ্রুত ম্লান হন তার হিসাব নেই। প্রত্যেক সাঁতারুকে একই শারীরবৃত্তীয় প্রোফাইলের মতো একই 1500 মিটার গতির সাথে আচরণ করে।

ক্রিটিক্যাল সাঁতারের গতি (CSS)

"কার্যকর" মেট্রিক

CSS হল দুটি দূরত্বের (সাধারণত একটি 400m এবং একটি 200m সর্বাধিক প্রচেষ্টা) এর উপর ভিত্তি করে একটি গাণিতিক ডেরিভেশন।

  • সুবিধা: পেসিং পক্ষপাত দূর করে। ব্যবধানে আপনার আচরণের পূর্বাভাস দেয়। একটি উদ্দেশ্য প্রদান করে "বায়বীয় সিলিং।"
  • অসুবিধা: দুটি পৃথক পরীক্ষা প্রয়োজন। কিছুটা জটিল গণিত (সাঁতারের বিশ্লেষণে স্বয়ংক্রিয়ভাবে গণনা করা হয়)। নিয়মিত বিশ্রামের সাথে পরীক্ষাগুলি করা প্রয়োজন।

কেন ডেল্টা ব্যাপার

"ডেল্টা" হল আপনার 400 মিটার গতি এবং আপনার 200 মিটার গতির মধ্যে পার্থক্য৷ দু'জন সাঁতারুর উভয়েরই টি-গতি 1:40/100m হতে পারে, কিন্তু তাদের CSS প্রোফাইলগুলি এইরকম দেখতে পারে:

সাঁতারু এ ("দ্য ডিজেল")

200m এবং 400m এর মধ্যে ছোট ব-দ্বীপ। এই সাঁতারুর একটি খুব উচ্চ বায়বীয় দক্ষতা আছে কিন্তু কম স্প্রিন্ট গতি। তাদের CSS তাদের T-Pace এর খুব কাছাকাছি।

সাঁতারু বি ("দ্য স্প্রিন্টার")

বড় ব-দ্বীপ। দুর্দান্ত 200 মিটার গতি কিন্তু 400 মিটারে উল্লেখযোগ্যভাবে কমে যায়। তাদের CSS তাদের T-Pace পরামর্শের চেয়ে কম হবে। যদি তারা একটি সাধারণ "টি-পেস" এ প্রশিক্ষণের চেষ্টা করে, তবে তারা ক্রমাগত তাদের অ্যানেরোবিক সিস্টেমকে অতিরিক্ত রান্না করবে।

📈 সিএসএস সূত্র

CSS = (D1 - D2) / (T1 - T2)

যেখানে D1/D2 হল দুটি দূরত্ব এবং T1/T2 হল সেকেন্ডের সময়। এটি আপনার প্রাথমিক অ্যানেরোবিক ঢেউয়ের শব্দ ছাড়াই আপনার টেকসই গতির "ঢাল" প্রকাশ করে।

সিএসএস পরীক্ষা কীভাবে সম্পাদন করবেন (ধাপে ধাপে)

আপনার CSS গণনা করতে, আপনাকে একটি স্ট্যান্ডার্ড পুলে বিভিন্ন দিনে (বা এর মধ্যে সম্পূর্ণ পুনরুদ্ধার সহ) দুটি সর্বাধিক ট্রায়াল করতে হবে:

  1. ওয়ার্ম আপ: আপনার শরীরকে প্রস্তুত করতে একটি পুঙ্খানুপুঙ্খ, প্রমিত ওয়ার্ম-আপ করুন (সাধারণত 400 মি থেকে 600 মি মিশ্র স্ট্রোক এবং বিল্ড)।
  2. ট্রায়াল 1 (400 মি সর্বোচ্চ প্রচেষ্টা): যত দ্রুত সম্ভব 400 মিটার সাঁতার কাটুন। সঠিক সময় রেকর্ড করুন। নিজেকে গতি দিন যাতে আপনি প্রথম 100 মিটারে পুড়ে না যান।
  3. সক্রিয় পুনরুদ্ধার: 100 মিটার থেকে 200 মিটার সাঁতার কাটুন খুব সহজে ল্যাকটিক অ্যাসিড ফ্লাশ করুন, তারপরে 5-10 মিনিট বিশ্রাম নিন।
  4. ট্রায়াল 2 (200 মি সর্বোচ্চ প্রচেষ্টা): পরম সর্বোচ্চ প্রচেষ্টায় 200 মিটার সাঁতার কাটুন। সঠিক সময় রেকর্ড করুন।
  5. গণনা করুন: আপনার সময়কে সেকেন্ডে রূপান্তর করুন এবং উপরের সূত্রটি প্রয়োগ করুন, অথবা আপনার 100 মিটার গতির গতি পেতে সুইম অ্যানালিটিক্সে CSS ক্যালকুলেটর টুল ব্যবহার করুন।

আপনি কোনটি ব্যবহার করা উচিত?

থ্রেশহোল্ড পেস (টি-পেস) ব্যবহার করুন যদি:

  • আপনি শুধু গতি-ভিত্তিক প্রশিক্ষণ দিয়ে শুরু করছেন।
  • আপনি শুধুমাত্র দীর্ঘ, একটানা সাঁতার কাটান (খোলা জল 3 কিমি+)।
  • আপনি একাধিক পরীক্ষার সেশন মোকাবেলা করতে চান না।

ক্রিটিক্যাল সুইম স্পিড (CSS) ব্যবহার করুন যদি:

  • ট্রায়াথলেট এবং ম্যারাথন সাঁতারু: একটি দৌড়ে খুব তাড়াতাড়ি "মোমবাতি জ্বালানো" এড়াতে।
  • ইন্টারভাল ট্রেনিং: নির্দিষ্ট CSS + 2s বা CSS - 1s জোন সেট করতে যা আসলে কাজ করে।
  • ট্র্যাকিং অগ্রগতি: CSS একটি জেনেরিক 1000m পরীক্ষার চেয়ে বায়বীয় উন্নতির জন্য বেশি সংবেদনশীল।

থ্রেশহোল্ড পেস বনাম CSS FAQ

সিএসএস কি থ্রেশহোল্ড গতির মতো?

না। তারা সম্পর্কিত থ্রেশহোল্ড ধারণা, কিন্তু তারা একই পরীক্ষা নয়। থ্রেশহোল্ড গতি সাধারণত একটি দীর্ঘ সাঁতার থেকে আসে, যখন CSS দুটি প্রচেষ্টা থেকে গণনা করা হয় এবং বিরতির কাজের জন্য আরও পুনরাবৃত্তিযোগ্য অ্যাঙ্কর দেয়।

ট্রায়াথলন এবং ওপেন-ওয়াটার পেসিংয়ের জন্য কোনটি ভাল?

সিএসএস সাধারণত ভাল শুরুর পয়েন্ট কারণ এটি অত্যধিক পুরস্কৃত স্বল্পমেয়াদী গতি ছাড়াই টেকসই গতি প্রতিফলিত করে। থ্রেশহোল্ড গতি এখনও দীর্ঘ স্থির সাঁতারের জন্য উপযোগী হতে পারে, তবে CSS সাধারণত পেসিং লক্ষ্য এবং জোন কাজে রূপান্তর করা সহজ।

আমি কিভাবে থ্রেশহোল্ড পেস থেকে CSS-ভিত্তিক সাঁতারের অঞ্চলে যেতে পারি?

CSS সেট করতে আপনার পরবর্তী 400m এবং 200m পরীক্ষা ব্যবহার করুন, তারপর সেই সংখ্যার চারপাশে অ্যাঙ্কর ইন্টারভাল কাজ করুন। আপনি যদি ইতিমধ্যেই টি-পেস দ্বারা প্রশিক্ষণ নেন, তাহলে উভয় সংখ্যার তুলনা করুন এবং পুনরাবৃত্তিযোগ্য অ্যারোবিক গতির চারপাশে আপনার সেটগুলিকে পুনর্নির্মাণের জন্য CSS গাইড এবং সাঁতারের প্রশিক্ষণ অঞ্চলগুলি ব্যবহার করুন৷

আপনার পুল জোন অনুমান বন্ধ করুন.

সুইম অ্যানালিটিক্স আপনাকে CSS কে ব্যবহারিক পুল জোন, ইন্টারভাল পেসিং, এবং সময়ের সাথে সাথে আপনার অ্যারোবিক সিলিংকে আরও পরিষ্কার করতে সাহায্য করে।
