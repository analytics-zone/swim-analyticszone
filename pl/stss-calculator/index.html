Darmowy Kalkulator TSS dla Pływania

Oblicz Training Stress Score dla treningów pływackich - Jedyny darmowy kalkulator sTSS

Czym jest TSS dla Pływania (sTSS)?

Swimming Training Stress Score (sTSS) kwantyfikuje obciążenie treningowe sesji pływackiej, łącząc intensywność i czas trwania. Jest to adaptacja metodologii TSS z kolarstwa, wykorzystująca Critical Swim Speed (CSS) jako tempo progowe. Trening o czasie 1 godziny w tempie CSS = 100 sTSS.

Interpretacja Obciążenia Treningowego

Jak Obliczany Jest sTSS

Formuła

sTSS = (Czas trwania w godzinach) × (Współczynnik Intensywności)² × 100

Gdzie:

  • Współczynnik Intensywności (IF) = Tempo CSS / Średnie Tempo Treningu
  • Czas trwania = Całkowity czas treningu w godzinach
  • Tempo CSS = Tempo progowe z testu CSS

Praktyczny Przykład

Szczegóły Treningu:

  • Tempo CSS: 1:49/100m (109 sekund)
  • Czas Trwania Treningu: 60 minut (1 godzina)
  • Średnie Tempo: 2:05/100m (125 sekund)

Krok 1: Oblicz Współczynnik Intensywności

IF = Tempo CSS / Tempo Treningu
IF = 109 / 125
IF = 0.872

Krok 2: Oblicz sTSS

sTSS = 1.0 godziny × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

Interpretacja: Ten 60-minutowy trening w łagodnym tempie (wolniejszym niż CSS) wygenerował 76 sTSS - umiarkowane obciążenie treningowe odpowiednie do budowania bazy tlenowej.

Interpretacja Wartości sTSS

Zakres sTSS Obciążenie Treningowe Czas Regeneracji Przykład Treningu
< 50 Niskie Tego samego dnia Łagodne pływanie 30 min, ćwiczenia techniczne
50-100 Umiarkowane 1 dzień Wytrzymałość 60 min, stałe tempo
100-200 Wysokie 1-2 dni Serie progowe 90 min, interwały w tempie wyścigowym
200-300 Bardzo Wysokie 2-3 dni Intensywny trening 2 godziny, wielokrotne bloki progowe

Przewodnik Tygodniowego sTSS

Docelowy tygodniowy sTSS zależy od poziomu treningu i celów:

Pływacy Rekreacyjni

Tygodniowy sTSS: 150-300

2-3 treningi tygodniowo, 50-100 sTSS każdy. Skupienie na technice i budowaniu bazy tlenowej.

Pływacy Fitness / Triathloniści

Tygodniowy sTSS: 300-500

3-4 treningi tygodniowo, 75-125 sTSS każdy. Mieszanka wytrzymałości tlenowej i pracy progowej.

Pływacy Masters Wyczynowi

Tygodniowy sTSS: 500-800

4-6 treningów tygodniowo, 80-150 sTSS każdy. Ustrukturyzowany trening z periodyzacją.

Pływacy Elitarni / Uniwersyteccy

Tygodniowy sTSS: 800-1200+

8-12 treningów tygodniowo, sesje podwójne. Wysoki wolumen z krytycznym zarządzaniem regeneracją.

⚠️ Ważne Uwagi

  • Wymaga dokładnego CSS: CSS powinno być aktualne (testowane w ciągu 6-8 tygodni) dla precyzyjnego sTSS.
  • Uproszczone obliczenia: Ten kalkulator używa średniego tempa. Zaawansowany sTSS używa Normalized Graded Pace (NGP), który uwzględnia strukturę interwałów.
  • Nie dla pracy technicznej: sTSS mierzy tylko fizyczny stres treningu, nie rozwój umiejętności.
  • Indywidualna zmienność: Ten sam sTSS różnie odczuwany przez różnych pływaków. Dostosuj wytyczne według regeneracji.

Dlaczego sTSS Ma Znaczenie

Training Stress Score jest podstawą dla:

  • CTL (Chronic Training Load): Poziom formy fizycznej - wykładnicza średnia ważona z 42 dni dziennego sTSS
  • ATL (Acute Training Load): Zmęczenie - wykładnicza średnia ważona z 7 dni dziennego sTSS
  • TSB (Training Stress Balance): Stan formy - TSB = CTL - ATL (dodatni = świeży, ujemny = zmęczony)
  • Periodyzacja: Planowanie faz treningowych (baza, rozwój, szczyt, tapering) używając docelowych progresji CTL
  • Zarządzanie Regeneracją: Wiedza, kiedy forsować, a kiedy odpoczywać na podstawie TSB

Porada Pro: Śledź CTL

Zapisuj dzienne sTSS w arkuszu kalkulacyjnym lub dzienniku treningowym. Obliczaj średnią z 42 dni (CTL) co tydzień. Celuj w wzrost 5-10 punktów CTL tygodniowo podczas budowania bazy. Utrzymuj lub lekko redukuj CTL podczas taperingu (1-2 tygodnie przed zawodami).

Często Zadawane Pytania

Czym jest TSS dla Pływania (sTSS)?

Swimming Training Stress Score (sTSS) to metryka, która kwantyfikuje obciążenie treningowe sesji pływackiej, łącząc zarówno intensywność, jak i czas trwania. Jest to adaptacja metodologii TSS z kolarstwa, wykorzystująca Critical Swim Speed (CSS) jako tempo progowe. Trening o czasie 1 godziny w tempie CSS równa się 100 sTSS.

Jak obliczyć mój sTSS?

Użyj kalkulatora powyżej, wprowadzając swoje tempo CSS (z testu CSS), całkowity czas trwania treningu i średnie tempo podczas treningu. Formuła to: sTSS = Czas trwania (godziny) × Współczynnik Intensywności² × 100, gdzie Współczynnik Intensywności = Tempo CSS / Średnie Tempo Treningu.

Czy potrzebuję CSS do obliczenia sTSS?

Tak, Twoja Krytyczna Prędkość Pływania (CSS) jest wymagana do obliczenia Współczynnika Intensywności, który jest niezbędny do obliczenia sTSS. CSS reprezentuje Twoje tempo progowe i powinno być testowane co 6-8 tygodni. Możesz znaleźć swoje CSS używając naszego kalkulatora CSS.

Jaki jest dobry wynik sTSS dla jednego treningu?

To zależy od intensywności treningu: Łatwe treningi zazwyczaj uzyskują wynik poniżej 50 sTSS, umiarkowane treningi 50-100 sTSS, ciężkie treningi 100-200 sTSS, a bardzo ciężkie treningi powyżej 200 sTSS. Odpowiedni wynik zależy od celów treningowych i obecnego poziomu kondycji.

Ile sTSS powinienem robić tygodniowo?

Tygodniowe cele sTSS różnią się w zależności od poziomu: Pływacy rekreacyjni: 150-300, Pływacy fitness/Triathloniści: 300-500, Pływacy Masters Wyczynowi: 500-800, Elita/Uniwersyteccy: 800-1200+. Zacznij zachowawczo i zwiększaj stopniowo, aby uniknąć przetrenowania.

Czy pływacki TSS jest taki sam jak kolarski TSS?

Koncepcja i formuła są takie same, ale sTSS jest dostosowany do pływania. Zamiast używać mocy (FTP) jak kolarski TSS, sTSS używa tempa z CSS jako progiem. Oba mierzą obciążenie treningowe używając Czas trwania × Współczynnik Intensywności² × 100.

Czy mogę używać sTSS dla wszystkich stylów pływackich?

Tak, ale Twój CSS powinien być specyficzny dla stylu. Większość pływaków używa CSS stylem dowolnym, ponieważ jest to najczęściej trenowany styl. Jeśli trenujesz głównie innym stylem, wykonaj test CSS w tym stylu i użyj tego tempa do obliczania sTSS.

Jaka jest różnica między sTSS a CTL/ATL/TSB?

sTSS mierzy obciążenie treningowe pojedynczego treningu. CTL (Chronic Training Load) to Twoja długoterminowa forma, ATL (Acute Training Load) to Twoje niedawne zmęczenie, a TSB (Training Stress Balance) to Twoja świeżość.
