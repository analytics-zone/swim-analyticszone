Co je TSS pro Plavání (sTSS)?

Swimming Training Stress Score (sTSS) kvantifikuje tréninkovou zátěž plaveckého sezení kombinací intenzity a délky trvání. Je to adaptace metodologie TSS z cyklistiky využívající Critical Swim Speed (CSS) jako práhové tempo. Trénink po dobu 1 hodiny v tempu CSS = 100 sTSS.

Interpretace Tréninkové Zátěže

Jak se Vypočítává sTSS

Vzorec

sTSS = (Délka v hodinách) × (Faktor Intenzity)² × 100

Kde:

  • Faktor Intenzity (IF) = Tempo CSS / Průměrné Tempo Tréninku
  • Délka = Celková doba tréninku v hodinách
  • Tempo CSS = Práhové tempo z testu CSS

Praktický Příklad

Detail Tréninku:

  • Tempo CSS: 1:49/100m (109 sekund)
  • Délka Tréninku: 60 minut (1 hodina)
  • Průměrné Tempo: 2:05/100m (125 sekund)

Krok 1: Vypočítat Faktor Intenzity

IF = Tempo CSS / Tempo Tréninku
IF = 109 / 125
IF = 0.872

Krok 2: Vypočítat sTSS

sTSS = 1.0 hodiny × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

Interpretace: Tento 60minutový trénink v lehkém tempu (pomalejším než CSS) vygeneroval 76 sTSS - mírnou tréninkovou zátěž vhodnou pro budování aerobní základny.

Interpretace Hodnot sTSS

Rozsah sTSS Tréninková Zátěž Doba Zotavení Příklad Tréninku

Průvodce Týdenního sTSS

Cílový týdenní sTSS závisí na úrovni tréninku a cílech:

Rekreační Plavci

Týdenní sTSS: 150-300

2-3 tréninky týdně, 50-100 sTSS každý. Zaměření na techniku a budování aerobní základny.

Fitness Plavci / Triatlonisté

Týdenní sTSS: 300-500

3-4 tréninky týdně, 75-125 sTSS každý. Směs aerobní vytrvalosti a práhové práce.

Závodní Masters Plavci

Týdenní sTSS: 500-800

4-6 tréninků týdně, 80-150 sTSS každý. Strukturovaný trénink s periodizací.

Elitní / Univerzitní Plavci

Týdenní sTSS: 800-1200+

8-12 tréninků týdně, dvojitá sezení. Vysoký objem s kritickým řízením zotavení.

⚠️ Důležité Poznámky

  • Vyžaduje přesné CSS: CSS by mělo být aktuální (testováno během 6-8 týdnů) pro přesné sTSS.
  • Zjednodušený výpočet: Tato kalkulačka používá průměrné tempo. Pokročilé sTSS používá Normalized Graded Pace (NGP), které zohledňuje strukturu intervalů.
  • Ne pro technickou práci: sTSS měří pouze fyzický stres tréninku, ne rozvoj dovedností.
  • Individuální variabilita: Stejné sTSS pociťováno různě různými plavci. Upravte pokyny podle zotavení.

Proč sTSS Záleží

Training Stress Score je základem pro:

  • CTL (Chronic Training Load): Úroveň fyzické kondice - exponenciálně vážený 42denní průměr denního sTSS
  • ATL (Acute Training Load): Únava - exponenciálně vážený 7denní průměr denního sTSS
  • TSB (Training Stress Balance): Stav formy - TSB = CTL - ATL (pozitivní = svěží, negativní = unavený)
  • Periodizace: Plánování tréninkových fází (základna, rozvoj, vrchol, tapering) pomocí cílových CTL progresí
  • Řízení Zotavení: Vědomí, kdy tlačit a kdy odpočívat na základě TSB

Profesionální Tip: Sledujte CTL

Zaznamenávejte denní sTSS v tabulkovém procesoru nebo tréninkové deníku. Vypočítejte 42denní průměr (CTL) každý týden. Cílte na nárůst CTL o 5-10 bodů týdně během budování základny. Udržujte nebo mírně snižujte CTL během taperingu (1-2 týdny před závodem).

Často Kladené Otázky

Co je Swimming TSS (sTSS)?

Swimming Training Stress Score (sTSS) je metrika, která kvantifikuje tréninkovou zátěž plaveckého tréninku kombinací intenzity a délky trvání. Je adaptována z metodologie TSS cyklistiky, používající vaši Kritickou Rychlost Plavání (CSS) jako prahové tempo. Trénink 1 hodiny v tempu CSS se rovná 100 sTSS.

Jak vypočítám své sTSS?

Použijte kalkulačku výše zadáním vašeho CSS tempa (z CSS testu), celkové délky tréninku a průměrného tempa během tréninku. Vzorec je: sTSS = Délka (hodiny) × Faktor Intenzity² × 100, kde Faktor Intenzity = CSS Tempo / Průměrné Tempo Tréninku.

Potřebuji CSS pro výpočet sTSS?

Ano, vaše Kritická Rychlost Plavání (CSS) je vyžadována pro výpočet Faktoru Intenzity, který je nezbytný pro výpočet sTSS. CSS představuje vaše prahové tempo a mělo by být testováno každých 6-8 týdnů. Své CSS můžete najít pomocí naší CSS kalkulačky.

Jaké je dobré sTSS skóre pro jeden trénink?

Závisí na intenzitě tréninku: Lehké tréninky typicky dosahují pod 50 sTSS, mírné tréninky 50-100 sTSS, těžké tréninky 100-200 sTSS a velmi těžké tréninky nad 200 sTSS. Vhodné skóre závisí na vašich tréninkových cílech a současné úrovni kondice.

Kolik sTSS bych měl dělat týdně?

Týdenní cíle sTSS se liší podle úrovně: Rekreační plavci: 150-300, Fitness plavci/Triatlonisté: 300-500, Závodní Masters: 500-800, Elitní/Univerzitní: 800-1200+. Začněte konzervativně a zvyšujte postupně, abyste předešli přetrénování.

Je plavecké TSS stejné jako cyklistické TSS?

Koncept a vzorec jsou stejné, ale sTSS je adaptováno pro plavání. Místo použití výkonu (FTP) jako cyklistické TSS, sTSS používá tempo s CSS jako prahem. Obě měří tréninkovou zátěž pomocí Délka × Faktor Intenzity² × 100.

Mohu použít sTSS pro všechny plavecké styly?

Ano, ale vaše CSS by mělo být specifické pro styl. Většina plavců používá CSS kraulu, protože je to nejčastěji trénovaný styl. Pokud trénujete primárně jiný styl, proveďte CSS test v tom stylu a použijte to tempo pro výpočet sTSS.

Jaký je rozdíl mezi sTSS a CTL/ATL/TSB?

sTSS měří tréninkovou zátěž jednoho tréninku. CTL (Chronická Tréninková Zátěž) je vaše dlouhodobá kondice, ATL (Akutní Tréninková Zátěž) je vaše nedávná únava a TSB (Rovnováha Tréninkového Stresu) je vaše svěžest. Tyto metriky používají hodnoty sTSS v čase ke sledování vašeho tréninkového stavu. Více se dozvíte v našem průvodci Tréninkovou Zátěží.
