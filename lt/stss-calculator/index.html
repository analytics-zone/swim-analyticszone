Nemokama plaukimo TSS skaičiuoklė

Apskaičiuokite plaukimo treniruočių streso balą – vienintelė nemokama sTSS skaičiuoklė

Kas yra plaukimo TSS (sTSS)?

Plaukimo treniruočių streso balas (sTSS) kiekybiškai įvertina plaukimo treniruotės krūvį, derindamas intensyvumą ir trukmę. Jis pritaikytas pagal dviračių sporto TSS metodiką, kaip slenksčio tempą naudojant jūsų kritinį plaukimo greitį (CSS). 1 valandos treniruotė CSS tempu = 100 sTSS.

Treniruočių krūvio interpretacija

Kaip skaičiuojamas sTSS

Formulė

sTSS = (Trukmė valandomis) × (Intensyvumo faktorius)² × 100

Kur:

  • Intensyvumo faktorius (IF) = CSS tempas / vidutinis treniruotės tempas.
  • Trukmė = bendras treniruotės laikas valandomis.
  • CSS tempas = jūsų slenksčio tempas iš CSS testo.

Skaičiavimo pavyzdys

Treniruotės informacija:

  • CSS tempas: 1:49/100 m (109 sekundės).
  • Trukmė: 60 minučių (1 valanda).
  • Vidutinis tempas: 2:05/100 m (125 sekundės).

1 žingsnis: intensyvumo faktoriaus apskaičiavimas

IF = CSS tempas / treniruotės tempas
IF = 109 / 125
IF = 0,872

2 žingsnis: sTSS apskaičiavimas

sTSS = 1,0 val. × (0,872)² × 100
sTSS = 1,0 × 0,760 × 100
sTSS = 76

Interpretacija: ši 60 minučių treniruotė lengvu tempu (lėčiau nei CSS) sugeneravo 76 sTSS – vidutinį treniruočių krūvį, tinkamą aerobinės bazės stiprinimui.

Savaitės sTSS gairės

Tikslinis savaitės sTSS priklauso nuo jūsų lygio ir tikslų:

Mėgėjai

Savaitės sTSS: 150–300

2–3 treniruotės per savaitę, po 50–100 sTSS kiekviena. Fokusas į techniką ir aerobinės bazės kūrimą.

Sveikatingumo plaukikai / triatlonininkai

Savaitės sTSS: 300–500

3–4 treniruotės per savaitę, po 75–125 sTSS kiekviena. Aerobinės ištvermės ir slenksčio darbo derinys.

Konkurencingi meistrai plaukikai

Savaitės sTSS: 500–800

4–6 treniruotės per savaitę, po 80–150 sTSS kiekviena. Struktūrizuotos treniruotės su periodizacija.

Elito / studentų rinktinių plaukikai

Savaitės sTSS: 800–1200+

8–12 treniruočių per savaitę, dvigubos treniruotės. Didelis tūris, itin svarbus atsigavimo valdymas.

⚠️ Svarbios pastabos

  • Reikalingas tikslus CSS: jūsų CSS turi būti aktualus (testuotas per paskutines 6–8 savaites), kad sTSS būtų tikslus.
  • Supaprastintas skaičiavimas: ši skaičiuoklė naudoja vidutinį tempą. Pažangus sTSS naudoja normalizuotą tempą (NGP), kuris atsižvelgia į intervalų struktūrą.
  • Netinka technikos darbui: sTSS matuoja tik fizinį stresą, o ne įgūdžių lavinimą.
  • Individualūs skirtumai: tas pats sTSS skirtingiems plaukikams gali jaustis kitaip. Reguliuokite krūvį pagal savo atsistatymą.

Kodėl sTSS yra svarbus

Treniruočių streso balas yra pagrindas šiems rodikliams:

  • CTL (ilgalaikis treniruočių krūvis): jūsų fizinis pasirengimas – 42 dienų sTSS eksponentinis vidurkis.
  • ATL (trumpalaikis treniruočių krūvis): jūsų nuovargis – 7 dienų sTSS eksponentinis vidurkis.
  • TSB (treniruotumo balas): jūsų sportinė forma – TSB = CTL - ATL (teigiamas = esate pailsėjęs, neigiamas = pavargęs).
  • Periodizacija: planuokite treniruočių fazes (bazinė, stiprinimo, piko, krūvio mažinimo) naudodami tikslinį CTL kilimą.
  • Atsigavimo valdymas: žinokite, kada paspausti, o kada pailsėti remiantis TSB rodikliu.

Patarimas: stebėkite savo CTL

Užsirašykite dienos sTSS į lentelę ar treniruočių žurnalą. Kiekvieną savaitę apskaičiuokite savo 42 dienų vidurkį (CTL). Bazinio periodo metu siekite 5–10 CTL taškų prieaugio per savaitę. Krūvio mažinimo (taper) metu (1–2 savaites prieš varžybas) palaikykite arba šiek tiek sumažinkite CTL.

Dažniausiai užduodami klausimai

Kas yra plaukimo TSS (sTSS)?

Plaukimo treniruočių streso balas (sTSS) yra metrika, kiekybiškai įvertinanti plaukimo treniruotės krūvį pagal jos intensyvumą ir trukmę. Jis pritaikytas iš dviračių sporto TSS metodikos, slenksčio tempu laikant jūsų kritinį plaukimo greitį (CSS). 1 valandos treniruotė CSS tempu lygi 100 sTSS.

Kaip apskaičiuoti savo sTSS?

Naudokite aukščiau pateiktą skaičiuoklę įvesdami savo CSS tempą (iš CSS testo), bendrą treniruotės trukmę ir vidutinį visos treniruotės tempą. Formulė: sTSS = trukmė (val.) × intensyvumo faktorius² × 100, kur intensyvumo faktorius = CSS tempas / vidutinis treniruotės tempas.

Ar man reikia CSS, kad apskaičiuočiau sTSS?

Taip, jūsų kritinis plaukimo greitis (CSS) yra būtinas norint apskaičiuoti intensyvumo faktorių, kuris yra esminis sTSS skaičiavimui. CSS atspindi jūsų slenksčio tempą ir turėtų būti testuojamas kas 6–8 savaites. Savo CSS galite rasti naudodami mūsų CSS skaičiuoklę.

Koks sTSS balas vienai treniruotei yra geras?

Tai priklauso nuo treniruotės intensyvumo: lengvos treniruotės paprastai siekia iki 50 sTSS, vidutinės – 50–100 sTSS, sunkios – 100–200 sTSS, o labai sunkios – virš 200 sTSS. Tinkamas balas priklauso nuo jūsų tikslų ir esamos formos.

Kiek sTSS turėčiau sukaupti per savaitę?

Savaitės sTSS tikslai skiriasi pagal lygį: mėgėjams – 150–300, sveikatingumo plaukikams/triatlonininkams – 300–500, meistrams – 500–800, elitui – 800–1200+. Pradėkite atsargiai ir didinkite krūvį pamažu, kad išvengtumėte pervargimo.

Ar plaukimo TSS yra toks pats kaip dviračių sporto TSS?

Koncepcija ir formulė yra tokios pačios, tačiau sTSS pritaikytas plaukimui. Vietoj galios (FTP), kaip dviračių sporte, sTSS naudoja tempą, o kaip slenksčio rodiklį – CSS. Abu matuoja krūvį naudojant trukmės, intensyvumo faktoriaus kvadrato ir 100 sandaugą.

Ar galiu naudoti sTSS visiems plaukimo būdams?

Taip, bet jūsų CSS turėtų būti pritaikytas konkrečiam būdui. Dauguma naudoja kraulio CSS, nes tai pagrindinis treniruojamas būdas. Jei plaukiate daugiausia kitu būdu, atlikite CSS testą tuo būdu ir naudokite gautą tempą sTSS skaičiavimui.

Kuo skiriasi sTSS ir CTL/ATL/TSB?

sTSS matuoja vienos treniruotės krūvį. CTL (ilgalaikis krūvis) parodo jūsų bendrą formą, ATL (trumpalaikis krūvis) – paskutinių dienų nuovargį, o TSB (balas) – jūsų šviežumą (pailsėjimo lygį). Šios metrikos naudoja sTSS reikšmes laiko tėkmėje. Daugiau sužinokite mūsų treniruočių krūvio vadove.
