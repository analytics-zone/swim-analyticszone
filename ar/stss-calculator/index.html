حاسبة TSS المجانية للسباحة

احسب درجة إجهاد التدريب لجلسات السباحة - الحاسبة المجانية الوحيدة لـ sTSS

ما هو TSS للسباحة (sTSS)؟

درجة إجهاد التدريب للسباحة (sTSS) تحدد كمياً حمل التدريب لجلسة السباحة من خلال الجمع بين الشدة والمدة. إنها مقتبسة من منهجية TSS من ركوب الدراجات، باستخدام سرعتك الحرجة للسباحة (CSS) كسرعة العتبة. جلسة تدريب لمدة ساعة واحدة بسرعة CSS = 100 sTSS.

تفسير حمل التدريب

كيف يتم حساب sTSS

الصيغة

sTSS = (المدة بالساعات) × (عامل الشدة)² × 100

حيث:

  • عامل الشدة (IF) = سرعة CSS / السرعة المتوسطة للتدريب
  • المدة = إجمالي وقت التدريب بالساعات
  • سرعة CSS = سرعة العتبة الخاصة بك من اختبار CSS

مثال عملي

تفاصيل التدريب:

  • سرعة CSS: 1:49/100م (109 ثانية)
  • مدة التدريب: 60 دقيقة (ساعة واحدة)
  • السرعة المتوسطة: 2:05/100م (125 ثانية)

الخطوة 1: حساب عامل الشدة

IF = سرعة CSS / سرعة التدريب
IF = 109 / 125
IF = 0.872

الخطوة 2: حساب sTSS

sTSS = 1.0 ساعة × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

التفسير: هذه الجلسة التدريبية لمدة 60 دقيقة بسرعة سهلة (أبطأ من CSS) أنتجت 76 sTSS - حمل تدريب معتدل مناسب لبناء القاعدة الهوائية.

تفسير قيم sTSS

نطاق sTSS حمل التدريب وقت الاستشفاء مثال على التدريب
< 50 منخفض نفس اليوم سباحة سهلة 30 دقيقة، تمارين تقنية
50-100 معتدل يوم واحد تحمل 60 دقيقة، سرعة ثابتة
100-200 عالي 1-2 يوم مجموعات عتبة 90 دقيقة، فترات بسرعة السباق
200-300 عالي جداً 2-3 أيام تدريب شاق ساعتين، كتل عتبة متعددة
> 300 شديد 3+ أيام سباحة طويلة (>ساعتين)، تحمل فائق

دليل sTSS الأسبوعي

يعتمد هدف sTSS الأسبوعي على مستوى تدريبك وأهدافك:

السباحون الترفيهيون

sTSS أسبوعي: 150-300

2-3 جلسات تدريب أسبوعياً، 50-100 sTSS لكل منها. التركيز على التقنية وبناء القاعدة الهوائية.

سباحو اللياقة / الترياثليت

sTSS أسبوعي: 300-500

3-4 جلسات تدريب أسبوعياً، 75-125 sTSS لكل منها. مزيج من التحمل الهوائي وعمل العتبة.

سباحو الماسترز التنافسيون

sTSS أسبوعي: 500-800

4-6 جلسات تدريب أسبوعياً، 80-150 sTSS لكل منها. تدريب منظم مع دورية.

السباحون النخبة / الجامعيون

sTSS أسبوعي: 800-1200+

8-12 جلسة تدريب أسبوعياً، جلسات مزدوجة. حجم عالٍ مع إدارة استشفاء حرجة.

⚠️ ملاحظات هامة

  • يتطلب CSS دقيق: يجب أن يكون CSS الخاص بك محدثاً (تم اختباره خلال 6-8 أسابيع) للحصول على sTSS دقيق.
  • حساب مبسط: تستخدم هذه الحاسبة السرعة المتوسطة. sTSS المتقدم يستخدم Normalized Graded Pace (NGP) الذي يأخذ في الاعتبار هيكل الفترات.
  • ليس للعمل التقني: sTSS يقيس فقط الإجهاد البدني للتدريب، وليس تطوير المهارات.
  • التباين الفردي: نفس sTSS يشعر به السباحون المختلفون بشكل مختلف. اضبط الإرشادات وفقاً لاستشفائك.

لماذا يهم sTSS

درجة إجهاد التدريب هي الأساس لـ:

  • CTL (حمل التدريب المزمن): مستوى لياقتك - متوسط مرجح أسياً لـ 42 يوماً من sTSS اليومي
  • ATL (حمل التدريب الحاد): إرهاقك - متوسط مرجح أسياً لـ 7 أيام من sTSS اليومي
  • TSB (توازن إجهاد التدريب): حالة الشكل الخاصة بك - TSB = CTL - ATL (إيجابي = منتعش، سلبي = متعب)
  • الدورية: تخطيط مراحل التدريب (القاعدة، التطوير، الذروة، الانخفاض) باستخدام تطورات CTL المستهدفة
  • إدارة الاستشفاء: معرفة متى تدفع ومتى ترتاح بناءً على TSB

نصيحة احترافية: تتبع CTL الخاص بك

سجل sTSS اليومي في جدول بيانات أو دفتر تدريب. احسب متوسط 42 يوماً (CTL) أسبوعياً. الهدف زيادة 5-10 نقاط CTL أسبوعياً خلال بناء القاعدة. حافظ على أو قلل CTL قليلاً خلال الانخفاض (1-2 أسبوع قبل المنافسة).

الأسئلة الشائعة

ما هو TSS للسباحة (sTSS)؟

درجة إجهاد التدريب للسباحة (sTSS) هي مقياس يحدد كمياً حمل التدريب لجلسة السباحة من خلال الجمع بين الشدة والمدة. إنها مقتبسة من منهجية TSS من ركوب الدراجات، باستخدام سرعتك الحرجة للسباحة (CSS) كسرعة العتبة. جلسة تدريب لمدة ساعة واحدة بسرعة CSS تساوي 100 sTSS.

كيف أحسب sTSS الخاص بي؟

استخدم الحاسبة أعلاه بإدخال سرعة CSS الخاصة بك (من اختبار CSS)، ومدة التدريب الإجمالية، والسرعة المتوسطة أثناء التدريب. الصيغة هي: sTSS = المدة (ساعات) × عامل الشدة² × 100، حيث عامل الشدة = سرعة CSS / السرعة المتوسطة للتدريب.

هل أحتاج CSS لحساب sTSS؟

نعم، سرعتك الحرجة للسباحة (CSS) مطلوبة لحساب عامل الشدة، والذي هو أساسي لحساب sTSS. يمثل CSS سرعة العتبة الخاصة بك ويجب اختباره كل 6-8 أسابيع. يمكنك العثور على CSS الخاص بك باستخدام حاسبة CSS الخاصة بنا.

ما هي درجة sTSS الجيدة لتدريب واحد؟

يعتمد على شدة التدريب: التدريبات السهلة عادة تسجل أقل من 50 sTSS، التدريبات المعتدلة 50-100 sTSS، التدريبات الصعبة 100-200 sTSS، والتدريبات الصعبة جداً أكثر من 200 sTSS. الدرجة المناسبة تعتمد على أهداف التدريب ومستوى اللياقة الحالي.

كم sTSS يجب أن أفعل في الأسبوع؟

أهداف sTSS الأسبوعية تختلف حسب المستوى: السباحون الترفيهيون: 150-300، سباحو اللياقة/الترياثليت: 300-500، سباحو الماسترز التنافسيون: 500-800، النخبة/الجامعيون: 800-1200+. ابدأ بحذر وزد تدريجياً لتجنب الإفراط في التدريب.

هل TSS السباحة نفس TSS ركوب الدراجات؟

المفهوم والصيغة متشابهان، لكن sTSS مُكيّف للسباحة. بدلاً من استخدام القوة (FTP) مثل TSS ركوب الدراجات، يستخدم sTSS السرعة مع CSS كعتبة. كلاهما يقيس حمل التدريب باستخدام المدة × عامل الشدة² × 100.

هل يمكنني استخدام sTSS لجميع أنماط السباحة؟

نعم، لكن CSS الخاص بك يجب أن يكون محدداً حسب النمط. معظم السباحين يستخدمون CSS النمط الحر لأنه النمط الأكثر تدريباً. إذا كنت تتدرب بشكل أساسي في نمط آخر، قم بإجراء اختبار CSS في ذلك النمط واستخدم تلك السرعة لحساب sTSS.

ما الفرق بين sTSS و CTL/ATL/TSB؟

sTSS يقيس حمل التدريب لجلسة واحدة. CTL (حمل التدريب المزمن) هو لياقتك طويلة الأمد، ATL (حمل التدريب الحاد) هو إرهاقك الأخير، و TSB (توازن إجهاد التدريب) هو انتعاشك. هذه المقاييس تستخدم قيم sTSS عبر الزمن لتتبع حالة التدريب. تعلم المزيد في دليل حمل التدريب الخاص بنا.
