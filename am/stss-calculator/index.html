ነጻ የዋና TSS ካልኩሌተር

ለዋና ልምምዶች የስልጠና ጭንቀት ነጥብ ያስሉ - ብቸኛው ነጻ sTSS ካልኩሌተር

የዋና TSS (sTSS) ምንድን ነው?

የዋና ስልጠና ጭንቀት ነጥብ (sTSS) ጥንካሬ እና ቆይታን በማጣመር የዋና ልምምድ የስልጠና ጫናን ይቆጥራል። ወሳኝ የዋና ፍጥነትዎን (CSS) እንደ ገደብ ፍጥነት በመጠቀም ከብስክሌት TSS ዘዴ የተስተካከለ ነው። በCSS ፍጥነት የ1-ሰዓት ልምምድ = 100 sTSS።

ተወዳዳሪ ማስተርስ ዋኞች

ልሂቅ / የኮሌጅ ዋኞች

ሳምንታዊ sTSS: 800-1200+

⚠️ አስፈላጊ ማስታወሻዎች

  • ትክክለኛ CSS ያስፈልጋል: CSS ዎ ለትክክለኛ sTSS ወቅታዊ መሆን አለበት (በ6-8 ሳምንታት ውስጥ የተሞከረ)።
  • ቀለል ያለ ስሌት: ይህ ካልኩሌተር አማካይ ፍጥነት ይጠቀማል። የላቀ sTSS የኢንተርቫል መዋቅርን የሚያካትት Normalized Graded Pace (NGP) ይጠቀማል።
  • ለቴክኒክ ስራ አይደለም: sTSS አካላዊ የስልጠና ጭንቀትን ብቻ ይለካል፣ የክህሎት ልማትን አይደለም።
  • የግለሰብ ልዩነት: ተመሳሳይ sTSS ለተለያዩ ዋኞች የተለየ ይሰማል። መመሪያዎችን በማገገሚያዎ ላይ ያስተካክሉ።

sTSS ለምን አስፈላጊ ነው

የስልጠና ጭንቀት ነጥብ ለሚከተሉት መሰረት ነው:

  • CTL (ሥር የሰደደ የስልጠና ጫና): የአካል ብቃት ደረጃዎ - የዕለታዊ sTSS 42-ቀን ኤክስፖኔንሺያል ክብደት ያለው አማካይ
  • ATL (አጣዳፊ የስልጠና ጫና): ድካምዎ - የዕለታዊ sTSS 7-ቀን ኤክስፖኔንሺያል ክብደት ያለው አማካይ
  • TSB (የስልጠና ጭንቀት ሚዛን): ቅርጽዎ - TSB = CTL - ATL (አዎንታዊ = ትኩስ፣ አሉታዊ = ደክሟል)
  • ፔሪዮዳይዜሽን: ዒላማ CTL እድገቶችን በመጠቀም የስልጠና ደረጃዎችን (መሰረት፣ ግንባታ፣ ጫፍ፣ ቴፐር) ያቅዱ
  • የማገገሚያ አስተዳደር: በTSB ላይ በመመስረት መቼ መግፋት እና መቼ ማረፍ እንዳለብዎ ይወቁ

ፕሮ ምክር: CTL ዎን ይከታተሉ

ዕለታዊ sTSS ን በስፕሬድሺት ወይም የስልጠና ምዝገባ ውስጥ ይመዝግቡ። የ42-ቀን አማካይዎን (CTL) በየሳምንቱ ያስሉ። በመሰረት ግንባታ ወቅት በሳምንት 5-10 CTL ነጥቦች ጭማሪ ያነጣጥሩ። በቴፐር ወቅት (ከውድድር 1-2 ሳምንታት በፊት) CTL ን ይጠብቁ ወይም ትንሽ ይቀንሱ።

በተደጋጋሚ የሚጠየቁ ጥያቄዎች

የዋና TSS (sTSS) ምንድን ነው?

የዋና ስልጠና ጭንቀት ነጥብ (sTSS) ጥንካሬ እና ቆይታን በማጣመር የዋና ልምምድ የስልጠና ጫናን የሚቆጥር መለኪያ ነው። ወሳኝ የዋና ፍጥነትዎን (CSS) እንደ ገደብ ፍጥነት በመጠቀም ከብስክሌት TSS ዘዴ የተስተካከለ ነው። በCSS ፍጥነት የ1 ሰዓት ልምምድ 100 sTSS ይሆናል።

sTSS ን እንዴት አስላለሁ?

CSS ፍጥነትዎን (ከCSS ሙከራ)፣ ጠቅላላ የልምምድ ቆይታ እና በልምምድ ወቅት አማካይ ፍጥነት በማስገባት ከላይ ያለውን ካልኩሌተር ይጠቀሙ። ቀመሩ: sTSS = ቆይታ (ሰዓታት) × የጥንካሬ ፋክተር³ × 100፣ የት የጥንካሬ ፋክተር = CSS ፍጥነት / አማካይ የልምምድ ፍጥነት።

sTSS ለማስላት CSS ያስፈልገኛል?

አዎ፣ ወሳኝ የዋና ፍጥነትዎ (CSS) ለsTSS ስሌት አስፈላጊ የሆነውን የጥንካሬ ፋክተር ለማስላት ያስፈልጋል። CSS የገደብ ፍጥነትዎን ይወክላል እና በየ6-8 ሳምንታት መሞከር አለበት። CSS ዎን በCSS ካልኩሌተራችን ማግኘት ይችላሉ።

ለአንድ ልምምድ ጥሩ sTSS ነጥብ ምንድን ነው?

በልምምድ ጥንካሬ ላይ ይወሰናል: ቀላል ልምምዶች በተለምዶ ከ50 sTSS በታች ነጥብ ያስመዘግባሉ፣ መካከለኛ ልምምዶች 50-100 sTSS፣ ከባድ ልምምዶች 100-200 sTSS እና በጣም ከባድ ልምምዶች ከ200 sTSS በላይ። ተስማሚ ነጥብ በስልጠና ግቦችዎ እና የአሁኑ የአካል ብቃት ደረጃ ላይ ይወሰናል።

በሳምንት ምን ያህል sTSS ማድረግ አለብኝ?

ሳምንታዊ sTSS ዒላማዎች በደረጃ ይለያያሉ: መዝናኛ ዋኞች: 150-300፣ የአካል ብቃት ዋኞች/ትሪያትሌቶች: 300-500፣ ተወዳዳሪ ማስተርስ: 500-800፣ ልሂቅ/ኮሌጅ: 800-1200+። ከመጠን በላይ ስልጠናን ለማስወገድ በጥንቃቄ ይጀምሩ እና ቀስ በቀስ ይጨምሩ።

የዋና TSS ከብስክሌት TSS ጋር ተመሳሳይ ነው?

ጽንሰ-ሐሳቡ እና ቀመሩ ተመሳሳይ ናቸው፣ ግን sTSS ለዋና የተስተካከለ ነው። እንደ ብስክሌት TSS ኃይል (FTP) ከመጠቀም ይልቅ፣ sTSS CSS ን እንደ ገደብ ከፍጥነት ጋር ይጠቀማል። ሁለቱም ቆይታ × የጥንካሬ ፋክተር³ × 100 በመጠቀም የስልጠና ጫናን ይለካሉ።

sTSS ን ለሁሉም የዋና ስትሮኮች መጠቀም እችላለሁ?

አዎ፣ ግን CSS ዎ ለስትሮክ-ልዩ መሆን አለበት። አብዛኛዎቹ ዋኞች ፍሪስታይል CSS ይጠቀማሉ ምክንያቱም በጣም የተለመደ የሰለጠነ ስትሮክ ስለሆነ። በሌላ ስትሮክ በዋናነት የሚሰለጥኑ ከሆነ፣ በዚያ ስትሮክ CSS ሙከራ ያድርጉ እና ለsTSS ስሌት ያንን ፍጥነት ይጠቀሙ።

በsTSS እና CTL/ATL/TSB መካከል ያለው ልዩነት ምንድን ነው?

sTSS የአንድ ልምምድ የስልጠና ጫናን ይለካል። CTL (ሥር የሰደደ የስልጠና ጫና) የረጅም ጊዜ አካል ብቃትዎ ነው፣ ATL (አጣዳፊ የስልጠና ጫና) የቅርብ ጊዜ ድካምዎ ነው እና TSB (የስልጠና ጭንቀት ሚዛን) ትኩስነትዎ ነው። እነዚህ መለኪያዎች የስልጠና ሁኔታዎን ለመከታተል በጊዜ ሂደት sTSS እሴቶችን ይጠቀማሉ። በየስልጠና ጫና መመሪያችን ውስጥ ተጨማሪ ይወቁ።
