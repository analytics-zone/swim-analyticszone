Izračunajte ocjenu stresa pri vježbanju plivanja - Jedini besplatni sTSS kalkulator

Što je Swimming TSS (sTSS)?

Rezultat stresa tijekom treninga plivanja (sTSS)kvantificira opterećenje vježbanja plivanja kombinacijom intenziteta i trajanja. Prilagođen je biciklističkoj metodologiji TSS, koristeći vašu kritičnu brzinu plivanja (CSS) kao granični tempo. 1-satno vježbanje CSS tempom = 100 sTSS.

Kako se sTSS izračunava

Formula

sTSS = (Trajanje u satima) × (Faktor intenziteta)³ × 100

Gdje:

  • Faktor intenziteta (IF)= CSS tempo / prosječni tempo vježbanja
  • Trajanje= Ukupno vrijeme vježbanja u satima
  • Tempo CSS-a= Vaš tempo praga iz CSS testa

Primjer rada

Detalji vježbanja:

  • Tempo CSS-a:1:49/100m(109 sekundi)
  • Trajanje treninga:60 minuta(1 sat)
  • Prosječni tempo:2:05/100m(125 sekundi)

Korak 1: Izračunajte faktor intenziteta

IF = CSS tempo / tempo vježbanja
ZAŠTITI15X = 109 / 125
ZAŠTITI15X =0,872

Korak 2: Izračunajte sTSS

sTSS = 1,0 sat × (0,872)² × 100
sTSS = 1,0 × 0,760 × 100
ZAŠTITI13X =76

Tumačenje:Ovaj 60-minutni trening laganim tempom (sporijim od CSS-a) generirao je 76 sTSS - umjereno opterećenje pri treningu prikladno za izgradnju aerobne baze.

Tjedne smjernice sTSS

Ciljani tjedni sTSS ovisi o vašoj razini treninga i ciljevima:

Plivači rekreativci

Tjedni sTSS:150-300 (prikaz, ostalo).

2-3 treninga tjedno, 50-100 sTSS svaki. Usredotočite se na tehniku ​​i izgradnju aerobne baze.

Kondicijski plivači/triatlonci

Tjedni sTSS:300-500

3-4 treninga tjedno, 75-125 sTSS svaki. Mješavina aerobne izdržljivosti i rada na pragu.

Natjecateljski majstori u plivanju

Tjedni sTSS:500-800

4-6 treninga tjedno, 80-150 sTSS svaki. Strukturirani trening s periodizacijom.

Elitni/sveučilišni plivači

Tjedni sTSS:800-1200+

8-12 treninga tjedno, dvodnevni. Velika količina s kritičnim upravljanjem oporavkom.

⚠️ Važne napomene

  • Zahtijeva točan CSS:Vaš CSS mora biti aktualan (testiran unutar 6-8 tjedana) za točan sTSS.
  • Pojednostavljeni izračun:Ovaj kalkulator koristi prosječni tempo. Advanced sTSS koristi normalizirani stupnjevani tempo (NGP) koji uzima u obzir strukturu intervala.
  • Nije za tehnički rad:sTSS mjeri samo stres pri tjelesnom treningu, ne i razvoj vještina.
  • Individualna varijacija:Isti sTSS djeluje drugačije za različite plivače. Prilagodite smjernice na temelju vašeg oporavka.

Zašto je sTSS važan

Trening Stres Score temelj je za:

  • CTL (kronično opterećenje tijekom treninga):Vaša razina kondicije - 42-dnevni eksponencijalno ponderirani prosjek dnevnog sTSS
  • ATL (akutno opterećenje tijekom treninga):Vaš umor - 7-dnevni eksponencijalno ponderirani prosjek dnevnog sTSS
  • TSB (Ravnoteža stresa pri vježbanju):Vaša forma - TSB = CTL - ATL (pozitivno = svježe, negativno = umorno)
  • Periodizacija:Planirajte faze treninga (osnova, izgradnja, vrhunac, sužavanje) koristeći ciljane CTL progresije
  • Upravljanje oporavkom:Znajte kada gurati, a kada se odmoriti na temelju TSB

Stručni savjet: Pratite svoj CTL

Zabilježite dnevni sTSS u proračunsku tablicu ili dnevnik treninga. Izračunajte svoj 42-dnevni prosjek (CTL) tjedno. Ciljajte povećanje od 5-10 CTL bodova tjedno tijekom izgradnje baze. Zadržite ili malo smanjite CTL tijekom sužavanja (1-2 tjedna prije utrke).

Često postavljana pitanja

Što je Swimming TSS (sTSS)?

Stres rezultat treninga plivanja (sTSS) metrika je koja kvantificira opterećenje treninga plivanja kombinacijom intenziteta i trajanja. Prilagođen je biciklističkoj metodologiji TSS, koristeći vašu kritičnu brzinu plivanja (CSS) kao granični tempo. Vježba od 1 sata CSS tempom jednaka je 100 sTSS.

Kako mogu izračunati svoj sTSS?

Upotrijebite gornji kalkulator tako da unesete svoj CSS tempo (iz CSS testa), ukupno trajanje vježbanja i prosječni tempo tijekom vježbanja. Formula je: sTSS = trajanje (sati) × faktor intenziteta³ × 100, gdje je faktor intenziteta = CSS tempo / prosječni tempo vježbanja.

Trebam li CSS za izračun sTSS?

Da, vaša kritična brzina plivanja (CSS) potrebna je za izračun faktora intenziteta, koji je neophodan za izračun sTSS. CSS predstavlja vaš prag tempa i treba ga testirati svakih 6-8 tjedana. Svoj CSS možete pronaći koristeći našCSS kalkulator.

Koji je dobar sTSS rezultat za jedan trening?

Ovisi o intenzitetu vježbanja: lagani treninzi obično imaju rezultat ispod 50 sTSS, umjereni treninzi 50-100 sTSS, teški treninzi 100-200 sTSS, a vrlo teški treninzi iznad 200 sTSS. Odgovarajući rezultat ovisi o vašim ciljevima treninga i trenutnoj razini kondicije.

Koliko sTSS trebam raditi tjedno?

Tjedni ciljevi sTSS razlikuju se prema razini: rekreativni plivači: 150-300, fitness plivači/triatlonci: 300-500, natjecateljski masters: 500-800, elitni/sveučilišni: 800-1200+. Počnite konzervativno i postupno povećavajte kako biste izbjegli pretreniranost.

Je li plivanje TSS isto što i vožnja bicikla TSS?

Koncept i formula su isti, ali je sTSS prilagođen plivanju. Umjesto korištenja snage (FTP) kao što je ciklički TSS, sTSS koristi tempo s CSS-om kao pragom. Oba mjere opterećenje vježbanjem pomoću Trajanja × faktora intenziteta³ × 100.

Mogu li koristiti sTSS za sve stilove plivanja?

Da, ali vaš CSS bi trebao biti specifičan za crte. Većina plivača koristi CSS slobodnim stilom jer je to stil koji se najčešće trenira. Ako prvenstveno trenirate u drugom zaveslaju, izvedite CSS test u tom zaveslaju i koristite taj tempo za sTSS izračun.

Koja je razlika između sTSS i CTL/ATL/TSB?

sTSS mjeri opterećenje treninga jednog treninga. CTL (Chronic Training Load) je vaša dugoročna kondicija, ATL (Acute Training Load) je vaš nedavni umor, a TSB (Training Stress Balance) vaša je svježina. Ove metrike koriste sTSS vrijednosti tijekom vremena za praćenje vašeg stanja treniranja. Saznajte više u našemVodič za opterećenje pri vježbanju.
