முக்கியமான நீச்சல் வேகம் (CSS) எதிராக த்ரெஷோல்ட் வேகம்: எது முக்கியம்?

பல ஆண்டுகளாக, "த்ரெஷோல்ட் பேஸ்" (டி-பேஸ்) தங்கத் தரமாக இருந்தது. 2026 ஆம் ஆண்டில், கிரிட்டிகல் ஸ்விம் ஸ்பீட் (CSS) க்கு மாறுவது, விளையாட்டு வீரர்கள் தங்கள் அமர்வுகளை எவ்வாறு வேகப்படுத்துகிறார்கள் மற்றும் ரேஸ்-டே களைப்பை எவ்வாறு கணிக்கிறார்கள் என்பதை மறுவரையறை செய்கிறது.

🎯 விரைவான பதில்

டி-பேஸ் (த்ரெஷோல்ட் பேஸ்) ஒரு மணிநேரத்திற்கு நீங்கள் பராமரிக்கக்கூடிய வேகத்தின் பொதுவான மதிப்பீடாகும், இது பெரும்பாலும் பரந்த பயிற்சி மண்டலங்களை அமைப்பதற்குப் பயன்படுத்தப்படுகிறது. இது ஒரு "நடைமுறை" ஆனால் குறைவான சிறுமணி அளவீடு ஆகும்.

CSS (முக்கியமான நீச்சல் வேகம்) இரண்டு வெவ்வேறு தூரங்களிலிருந்து (எ.கா., 400மீ மற்றும் 200மீ) கணித ரீதியாக பெறப்பட்ட "செயல்பாட்டு" வரம்பு ஆகும். இது "சுவரை" தாக்காமல் நீங்கள் தக்கவைக்கக்கூடிய தத்துவார்த்த வேகம். நீங்கள் சகிப்புத்தன்மை நீச்சல் வீரராகவோ அல்லது ட்ரையத்லெட்டாகவோ இருந்தால், உங்கள் பயிற்சித் திட்டத்தில் உங்கள் CSS மிக முக்கியமான எண்ணாகும்.

2026 மாற்றம்: முயற்சியிலிருந்து உடலியல் வரை

குளத்தில் சோர்வை நிர்வகிப்பது சைக்கிள் ஓட்டுதல் அல்லது ஓடுவதை விட மிகவும் கடினமானது. மின் மீட்டர் இல்லாமல், வேகம் மட்டுமே தீவிரத்திற்கான எங்களின் ஒரே ப்ராக்ஸி. பல நீச்சல் வீரர்கள் இன்னும் பழைய பள்ளி 1500 மீ "டைம்-ட்ரையல்" வேகத்தை நம்பியிருந்தாலும், நவீன பயிற்சியானது இரண்டு தூர CSS மாதிரி.

முக்கிய வேறுபாடு துல்லியம் எதிராக உணரப்பட்ட முயற்சி. ஒரு நீண்ட முயற்சியை அடிப்படையாகக் கொண்ட டி-பேஸ் பெரும்பாலும் உங்கள் வளர்சிதை மாற்ற வரம்பைக் காட்டிலும் உங்கள் மன அழுத்தத்தைப் பிடிக்கிறது. CSS, உங்கள் "பேஸ்-டூரேஷன் வளைவில்" இரண்டு வெவ்வேறு புள்ளிகளிலிருந்து கணக்கிடப்படுகிறது, இது உங்கள் இயந்திரத்தின் உண்மையான ஏரோபிக் வரம்பை வெளிப்படுத்துகிறது.

மெட்ரிக் ஒப்பீடு

த்ரெஷோல்ட் பேஸ் (டி-பேஸ்)

"எளிய" மெட்ரிக்

T-Pace பொதுவாக 1500m அல்லது 1000m அதிகபட்ச முயற்சியில் இருந்து பெறப்படுகிறது. இது தொடர்ச்சியான நீச்சலுக்கான உங்கள் "ரெட்லைனை" குறிக்கிறது.

  • நன்மை: புரிந்து கொள்ள எளிமையானது. ஒரு சோதனை, ஒரு எண். நீண்ட, தொடர்ச்சியான தொகுப்புகளுக்கு சிறந்தது.
  • பாதகம்: வேகக்கட்டுப்பாடு பிழைகளுக்கு அதிக உணர்திறன். சோர்வின் கீழ் நீங்கள் எவ்வளவு விரைவாக மங்குகிறீர்கள் என்பதைக் கணக்கிடவில்லை. ஒவ்வொரு நீச்சல் வீரரையும் அதே 1500மீ வேகத்துடன் ஒரே உடலியல் சுயவிவரத்துடன் நடத்துகிறது.

முக்கியமான நீச்சல் வேகம் (CSS)

"செயல்பாட்டு" மெட்ரிக்

CSS என்பது இரண்டு தூரங்களை அடிப்படையாகக் கொண்ட ஒரு கணித வழித்தோன்றலாகும் (பொதுவாக 400மீ மற்றும் 200மீ அதிகபட்ச முயற்சி).

  • நன்மை: வேகக்கட்டுப்பாட்டை நீக்குகிறது. உங்கள் நடத்தையை இடைவெளியில் சிறப்பாகக் கணிப்பது. ஒரு புறநிலை "ஏரோபிக் உச்சவரம்பு" வழங்குகிறது.
  • பாதகம்: இரண்டு தனித்தனி சோதனைகள் தேவை. சற்று சிக்கலான கணிதம் (நீச்சல் பகுப்பாய்வில் தானாக கணக்கிடப்பட்டது). சோதனைகள் சீரான ஓய்வுடன் செய்யப்பட வேண்டும்.

டெல்டா ஏன் முக்கியமானது

"டெல்டா" என்பது உங்கள் 400மீ வேகத்திற்கும் 200மீ வேகத்திற்கும் உள்ள வித்தியாசம். இரண்டு நீச்சல் வீரர்கள் இருவரும் 1:40/100மீ டி-வேகத்தைக் கொண்டிருக்கலாம், ஆனால் அவர்களின் CSS சுயவிவரங்கள் இப்படி இருக்கலாம்:

நீச்சல் வீரர் ஏ ("தி டீசல்")

200மீ முதல் 400மீ வரையிலான சிறிய டெல்டா. இந்த நீச்சல் வீரர் மிக அதிக ஏரோபிக் திறன் கொண்டவர், ஆனால் வேகம் குறைவாக உள்ளது. அவர்களின் CSS அவர்களின் T-Pace க்கு மிக அருகில் உள்ளது.

நீச்சல் வீரர் பி ("தி ஸ்ப்ரிண்டர்")

பெரிய டெல்டா. சிறந்த 200மீ வேகம் ஆனால் 400மீட்டரில் கணிசமாக குறைகிறது. அவர்களின் CSS அவர்களின் T-Pace பரிந்துரைகளை விட குறைவாக இருக்கும். அவர்கள் ஒரு பொதுவான "டி-பேஸ்" இல் பயிற்சி பெற முயற்சித்தால், அவர்கள் தொடர்ந்து தங்கள் காற்றில்லா அமைப்பை அதிகமாக சமைப்பார்கள்.

📈 CSS ஃபார்முலா

CSS = (D1 - D2) / (T1 - T2)

D1/D2 என்பது இரண்டு தூரங்கள் மற்றும் T1/T2 என்பது நொடிகளில் நேரமாகும். இது உங்கள் ஆரம்ப காற்றில்லா எழுச்சியின் சத்தம் இல்லாமல் உங்கள் நிலையான வேகத்தின் "சாய்வை" வெளிப்படுத்துகிறது.

CSS தேர்வை எவ்வாறு செய்வது (படிப்படியாக)

உங்கள் CSSஐக் கணக்கிட, ஒரு நிலையான குளத்தில் வெவ்வேறு நாட்களில் (அல்லது இடையில் முழு மீட்புடன்) இரண்டு அதிகபட்ச சோதனைகளைச் செய்ய வேண்டும்:

  1. வார்ம்-அப்: உங்கள் உடலை தயார்படுத்த ஒரு முழுமையான, தரப்படுத்தப்பட்ட வார்ம்-அப் (பொதுவாக 400மீ முதல் 600மீ வரையிலான கலவையான பக்கவாதம் மற்றும் உருவாக்கம்) செய்யவும்.
  2. சோதனை 1 (400மீ அதிகபட்ச முயற்சி): முடிந்தவரை வேகமாக 400 மீ நீந்தவும். சரியான நேரத்தை பதிவு செய்யவும். முதல் 100மீட்டரில் எரிந்துவிடாமல் நீங்களே வேகமாகச் செல்லுங்கள்.
  3. செயலில் மீட்பு: லாக்டிக் அமிலத்தை வெளியேற்றுவதற்கு 100 மீ முதல் 200 மீ வரை நீந்தவும், அதன் பிறகு 5-10 நிமிட ஓய்வு.
  4. சோதனை 2 (200மீ அதிகபட்ச முயற்சி): அதிகபட்ச முயற்சியில் 200 மீ நீந்தவும். சரியான நேரத்தை பதிவு செய்யவும்.
  5. கணக்கிடு: உங்கள் நேரத்தை வினாடிகளாக மாற்றி மேலே உள்ள சூத்திரத்தைப் பயன்படுத்தவும் அல்லது உங்கள் 100மீ வேகத்தைப் பெற நீச்சல் பகுப்பாய்வுகளில் CSS கால்குலேட்டர் கருவியைப் பயன்படுத்தவும்.

நீங்கள் எதைப் பயன்படுத்த வேண்டும்?

த்ரெஷோல்ட் பேஸ் (டி-பேஸ்) பயன்படுத்தினால்:

  • நீங்கள் வேக அடிப்படையிலான பயிற்சியுடன் தொடங்குகிறீர்கள்.
  • நீங்கள் நீண்ட, தொடர்ச்சியான நீச்சல்களை மட்டுமே செய்கிறீர்கள் (திறந்த நீர் 3 கிமீ+).
  • பல சோதனை அமர்வுகளை நீங்கள் சமாளிக்க விரும்பவில்லை.

கிரிட்டிகல் நீச்சல் வேகத்தை (CSS) பயன்படுத்தினால்:

  • டிரைத்லெட்ஸ் & மராத்தான் நீச்சல் வீரர்கள்: பந்தயத்தில் சீக்கிரம் "மெழுகுவர்த்தியை எரிப்பதை" தவிர்க்க.
  • இடைவெளி பயிற்சி: உண்மையில் வேலை செய்யும் குறிப்பிட்ட CSS + 2s அல்லது CSS - 1s மண்டலங்களை அமைக்க.
  • கண்காணிப்பு முன்னேற்றம்: பொதுவான 1000மீ சோதனையை விட ஏரோபிக் மேம்பாடுகளுக்கு CSS அதிக உணர்திறன் கொண்டது.

த்ரெஷோல்ட் பேஸ் vs CSS FAQ

CSS என்பது வாசல் வேகம் ஒன்றா?

இல்லை. அவை தொடர்புடைய வாசல் கருத்துக்கள், ஆனால் அவை ஒரே சோதனை அல்ல. த்ரெஷோல்ட் வேகம் வழக்கமாக ஒரு நீண்ட நீச்சலில் இருந்து வருகிறது, அதே சமயம் CSS ஆனது இரண்டு முயற்சிகளில் இருந்து கணக்கிடப்படுகிறது மற்றும் இடைவெளி வேலைகளுக்கு மீண்டும் மீண்டும் செய்யக்கூடிய நங்கூரத்தை அளிக்கிறது.

டிரையத்லான் மற்றும் ஓபன்-வாட்டர் பேஸிங்கிற்கு எது சிறந்தது?

CSS பொதுவாக சிறந்த தொடக்க புள்ளியாகும், ஏனெனில் இது குறுகிய கால வேகத்திற்கு அதிக வெகுமதி அளிக்காமல் நிலையான வேகத்தை பிரதிபலிக்கிறது. நீண்ட நிலையான நீச்சல்களுக்கு த்ரெஷோல்ட் வேகம் இன்னும் பயனுள்ளதாக இருக்கும், ஆனால் CSS பொதுவாக வேகமான இலக்குகளாகவும் மண்டல வேலைகளாகவும் மாற்றுவது எளிது.

வாசல் வேகத்தில் இருந்து CSS அடிப்படையிலான நீச்சல் மண்டலங்களுக்கு நான் எப்படி நகர்வது?

CSS ஐ அமைக்க, உங்கள் அடுத்த 400மீ மற்றும் 200மீ சோதனையைப் பயன்படுத்தவும், பின்னர் அந்த எண்ணைச் சுற்றி நங்கூர இடைவெளி வேலை செய்யவும். நீங்கள் ஏற்கனவே T-Pace மூலம் பயிற்சி பெற்றிருந்தால், இரண்டு எண்களையும் ஒப்பிட்டு, CSS வழிகாட்டி மற்றும் நீச்சல் பயிற்சி மண்டலங்களைப் பயன்படுத்தி மீண்டும் மீண்டும் செய்யக்கூடிய ஏரோபிக் வேகத்தில் உங்கள் செட்களை மீண்டும் உருவாக்கவும்.

உங்கள் பூல் மண்டலங்களை யூகிப்பதை நிறுத்துங்கள்.

நீச்சல் அனலிட்டிக்ஸ், CSSஐ நடைமுறைக் குளம் மண்டலங்களாக மாற்ற உதவுகிறது, இடைவெளி வேகம் மற்றும் காலப்போக்கில் உங்கள் ஏரோபிக் உச்சவரம்பு பற்றிய தெளிவான பார்வை.
