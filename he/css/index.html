מהירות שחייה קריטית (CSS)

הקרן לאימוני שחייה מבוססי נתונים

מהי מהירות שחייה קריטית (CSS)?

מהירות שחייה קריטית (CSS)היא מהירות השחייה המקסימלית התיאורטית שתוכל לשמור ללא תשישות. הוא מייצג את קצב הסף האירובי שלך, המתאים בדרך כלל ל-4 mmol/L לקטאט בדם ובר קיימא למשך כ-30 דקות. CSS מחושב באמצעות ניסוי זמן של 400 ו-200 מטר כדי לקבוע אזורי אימון מותאמים אישית.

מהירות שחייה קריטית (CSS) מייצגת את מהירות השחייה המקסימלית התיאורטית שתוכל לשמור ברציפות ללא תשישות. זה הסף האירובי שלך במים - העוצמה שבה ייצור לקטט שווה פינוי לקטט.

🎯 משמעות פיזיולוגית

CSS מתאים מאוד ל:

  • סף לקטט 2 (LT2)- סף הנשמה שני
  • מצב יציב מקסימלי של לקטט (MLSS)- רמת לקטט בת קיימא הגבוהה ביותר
  • קצב סף פונקציונלי (FTP)- שחייה שווה ערך לרכיבה על אופניים FTP
  • ~4 ממול/ליטר לקטט בדם- סמן OBLA מסורתי

למה CSS חשוב

CSS הוא המדד יסודשפותח את כל ניתוח עומס האימון המתקדם:

  • אזורי אימון:התאמה אישית של אזורי עוצמה בהתבסס על הפיזיולוגיה שלך
  • חישוב sTSS:מאפשר כימות מדויק של ציון מתח אימון
  • CTL/ATL/TSB:נדרש עבור מדדי תרשים ניהול ביצועים
  • מעקב אחר התקדמות:מדד אובייקטיבי לשיפור הכושר האירובי
⚠️ תלות קריטית:ללא מבחן CSS חוקי, לא ניתן לחשב מדדי עומס אימון מתקדמים (sTSS, CTL, ATL, TSB). CSS לא מדויק ישחית את כל ניתוח ההדרכה שלאחר מכן.

כיצד לחשב CSS ב-4 שלבים פשוטים

📋 פרוטוקול סטנדרטי

  1. חימום

    300-800 מ' שחייה קלה, תרגילים והצטברות מתקדמת כדי להתכונן למאמץ מירבי.

  2. מבחן זמן 400 מ'

    מאמץ מתמשך מרבי מהתחלת דחיפה (ללא צלילה). שיא זמן עד השני. המטרה: 400 מ' בר-קיימא המהיר ביותר.

  3. שחזור מלא

    5-10 דקותשל שחייה קלה או מנוחה מלאה. זה קריטי לתוצאות מדויקות.

  4. מבחן זמן 200 מ'

    מקסימום מאמץ מהתחלת דחיפה. שיא זמן מדויק. זה צריך להיות מהיר יותר לכל 100 מ' מאשר 400 מ'.

⚠️ טעויות נפוצות

התאוששות לא מספקת

בעיה:עייפות מאטה באופן מלאכותי זמן של 200 מטר

תוצאה:CSS מחושב הופךמהיר יותר מהמציאות, מה שמוביל לאזורים מאומנים יתר על המידה

פתרון:נח עד שהקצב יורד מתחת ל-120 פעימות לדקה או עד שהנשימה תתאושש לחלוטין

קצב גרוע ב-400 מטר

בעיה:התחלה מהירה מדי גורמת להאטה דרמטית

תוצאה:זמן 400 מטר אינו משקף קצב בר-קיימא אמיתי

פתרון:כוון לפיצולים זוגיים או פיצול שלילי (200 מטר שני ≤ 200 מטר ראשונים)

שימוש בהתחלות צלילה

בעיה:מוסיף יתרון של ~0.5-1.5 שניות, הטיית חישובים

פתרון:השתמש תמיד בדחיפה מהקיר

🔄 תדירות בדיקה חוזרת

בדוק שוב CSS כל6-8 שבועותלעדכן את אזורי האימון ככל שהכושר משתפר. האזורים שלך צריכים להיות מהירים יותר ככל שאתה מסתגל לאימון.

נוסחת חישוב CSS

נוסחה

CSS (m/s) = (D₂ - D₁) / (T₂ - T₁)

איפה:

  • D₁ = 200 מטר
  • D₂ = 400 מטר
  • T₁ = זמן ל-200 מ' (בשניות)
  • T₂ = זמן ל-400 מ' (בשניות)

מפושט עבור קצב ל-100 מטר

קצב CSS/100 מטר (שניות) = (T₄₀₀ - T₂₀₀) / 2

דוגמה עובדת

תוצאות הבדיקה:

  • זמן 400 מ':6:08(368 שניות)
  • זמן 200 מ':2:30(150 שניות)

שלב 1: חשב CSS ב-m/s

CSS = (400 - 200) / (368 - 150)
CSS = 200 / 218
CSS =0.917 מ/שניה

שלב 2: המר לקצב ל-100 מטר

קצב = 100 / 0.917
קצב = 109 שניות
קצב =1:49 ל-100 מ'

אזורי אימון המבוססים על CSS

הערה:בשחייה, הקצב נמדד כמוזמן למרחק. לכן, אאחוז גבוה יותר = קצב איטי יותר, וכן אאחוז נמוך יותר = קצב מהיר יותר. זה הפוך לרכיבה/ריצה כאשר אחוז גבוה יותר = מאמץ קשה יותר.

🎯 הטבות אימון מבוסס אזור

שימוש באזורים מבוססי CSS הופך אימון "תחושה" סובייקטיבי לאימונים אובייקטיביים שניתן לחזור עליהם. כל אזור מתמקד בהתאמות פיזיולוגיות ספציפיות:

  • אזור 2:בניית מנוע אירובי (60-70% מהנפח השבועי)
  • אזור 3:שפר את היעילות בקצב המירוץ (15-20% מהנפח)
  • אזור 4:דחוף את סף הלקטט גבוה יותר (10-15% מהנפח)
  • אזור 5:פתח מהירות ועוצמה מעולים (5-10% מהנפח)

ערכי CSS טיפוסיים לפי רמה

אימות מדעי

Wakayoshi et al. (1992-1993) - מחקר יסודי

המחקרים המכריעים של Kohji Wakayoshi באוניברסיטת אוסקה קבעו את CSS כחלופה תקפה ומעשית לבדיקת לקטט במעבדה:

  • מתאם חזק עם VO₂ בסף אנאירובי(r = 0.818)
  • מתאם מצוין עם מהירות ב-OBLA(r = 0.949)
  • מנבא ביצועים של 400 מ'(r = 0.864)
  • מתאים ל-4 mmol/L לקטט בדם- מצב יציב מקסימלי של לקטט
  • קשר ליניארי בין מרחק וזמן(r² > 0.998)

מאמרי מפתח:

  1. Wakayoshi K, et al. (1992). "קביעה ותוקף של מהירות קריטית כמדד לביצועי שחייה אצל השחיין התחרותי".כתב העת האירופי לפיזיולוגיה יישומית, 64(2), 153-157.
  2. Wakayoshi K, et al. (1992). "שיטה פשוטה לקביעת מהירות קריטית כסף עייפות שחייה בשחייה תחרותית".כתב העת הבינלאומי לרפואת ספורט, 13(5), 367-371.
  3. Wakayoshi K, et al. (1993). "האם מהירות שחייה קריטית מייצגת את עוצמת הפעילות הגופנית במצב יציב מקסימלי של לקטט?"כתב העת האירופי לפיזיולוגיה יישומית, 66(1), 90-95.

🔬 למה CSS עובד

CSS מייצג את הגבול ביןתחומי פעילות גופנית כבדים וחמורים. מתחת ל-CSS, ייצור ופינוי לקטט נשארים מאוזנים - אתה יכול לשחות לתקופות ממושכות. מעל CSS, לקטט מצטבר בהדרגה עד לתשישות תוך 20-40 דקות.

זה הופך את ה-CSS לעוצמה המושלמת עבור:

  • קביעת קצבי מירוץ ברי קיימא לאירועי 800-1500 מ'
  • אימון אינטרוולים של מרשם סף
  • ניטור שיפורי כושר אירובי
  • חישוב עומס אימונים וצרכי התאוששות

כיצד לבדוק את ה-CSS שלך

שלב 1: התחממו כראוי

השלם 300-800 מטר של שחייה קלה, כולל תרגילים והצטברות מתקדמת. זה מכין את הגוף למאמץ מירבי ומונע פציעה.

שלב 2: בצע ניסוי זמן של 400 מטר

שחו 400 מטר במאמץ מתמשך מרבי מהתחלת דחיפה (ללא צלילה). רשום את הזמן שלך לשנייה. כוון לפיצולים זוגיים או לפיצול שלילי (200 מ' השני שווה ל-200 מ' הראשונים או מהיר יותר).

שלב 3: התאושש לחלוטין

מנוחה 5-10 דקות עם שחייה קלה או מנוחה מלאה. ההתאוששות הזו היא קריטית. המתן עד שהדופק שלך יירד מתחת ל-120 פעימות לדקה והנשימה תתאושש במלואה לפני שתמשיך.

שלב 4: בצע ניסוי זמן של 200 מטר

שחו 200 מטר במאמץ מרבי מהתחלת דחיפה. רשום את הזמן שלך במדויק. זה אמור להיות מהיר יותר ל-100 מ' מקצב ה-400 מ' שלך.

שלב 5: חשב את ה-CSS שלך

השתמש בנוסחה: CSS Pace/100m = (T400 - T200) / 2. דוגמה: (368s - 150s) / 2 = 109s = 1:49/100m. או השתמשו במחשבון שבעמוד זה כדי לקבל תוצאות מיידיות עם אזורי אימון מותאמים אישית.

יישומים מעשיים

1️⃣ ביטול נעילת מדדי עומס אימון

CSS הוא המכנה בחישוב Intensity Factor עבור sTSS. בלעדיו, אינך יכול לכמת מתח באימון או לעקוב אחר מגמות כושר/עייפות.

2️⃣ התאמה אישית של אזורי אימון

תרשימי קצב כלליים אינם מתייחסים לפיזיולוגיה אינדיבידואלית. אזורים מבוססי CSS מבטיחים שכל שחיין מתאמן בעצימות האופטימלית שלהם.

3️⃣ מעקב אחר התקדמות הכושר

בדיקה חוזרת כל 6-8 שבועות. שיפור CSS (קצב מהיר יותר) מעיד על הסתגלות אירובית מוצלחת. קיפאון CSS מצביע על צורך בהתאמת האימון.

4️⃣ חזה את ביצועי המירוץ

קצב ה-CSS מקורב לקצב המירוץ בר-קיימא של 30 דקות. השתמש בו כדי להגדיר יעדים ריאליים לאירועי 800 מ', 1500 מ' ואירועי מים פתוחים.

5️⃣ אימוני סף עיצוב

ערכות CSS קלאסיות: 8×100 @ קצב CSS (מנוחה של 15 שניות), 5×200 @ 101% CSS (מנוחה של 20 שניות), 3×400 @ 103% CSS (מנוחה של 30 שניות). בניית יכולת פינוי לקטט.

6️⃣ Optimize Taper Strategy

עקוב אחר CSS לפני ואחרי ה-Taper. טייפר מוצלח שומר או משפר מעט את ה-CSS תוך הפחתת עייפות (TSB מוגבר).
