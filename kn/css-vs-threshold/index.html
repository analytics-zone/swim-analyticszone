ಕ್ರಿಟಿಕಲ್ ಸ್ವಿಮ್ ಸ್ಪೀಡ್ (CSS) ವಿರುದ್ಧ ಥ್ರೆಶೋಲ್ಡ್ ಪೇಸ್: ಯಾವುದು ಮುಖ್ಯ?

ವರ್ಷಗಳವರೆಗೆ, "ಥ್ರೆಶೋಲ್ಡ್ ಪೇಸ್" (ಟಿ-ಪೇಸ್) ಚಿನ್ನದ ಮಾನದಂಡವಾಗಿತ್ತು. 2026 ರಲ್ಲಿ, ಕ್ರಿಟಿಕಲ್ ಸ್ವಿಮ್ ಸ್ಪೀಡ್ (CSS) ಗೆ ಬದಲಾವಣೆಯು ಕ್ರೀಡಾಪಟುಗಳು ತಮ್ಮ ಅವಧಿಗಳನ್ನು ಹೇಗೆ ವೇಗಗೊಳಿಸುತ್ತಾರೆ ಮತ್ತು ಓಟದ ದಿನದ ಆಯಾಸವನ್ನು ಹೇಗೆ ಊಹಿಸುತ್ತಾರೆ ಎಂಬುದನ್ನು ಮರು ವ್ಯಾಖ್ಯಾನಿಸುತ್ತಿದೆ.

🎯 ತ್ವರಿತ ಉತ್ತರ

ಟಿ-ಪೇಸ್ (ಥ್ರೆಶೋಲ್ಡ್ ಪೇಸ್) ನೀವು ಒಂದು ಗಂಟೆಯವರೆಗೆ ನಿರ್ವಹಿಸಬಹುದಾದ ವೇಗದ ಸಾಮಾನ್ಯ ಅಂದಾಜು, ಇದನ್ನು ಸಾಮಾನ್ಯವಾಗಿ ವಿಶಾಲವಾದ ತರಬೇತಿ ವಲಯಗಳನ್ನು ಹೊಂದಿಸಲು ಬಳಸಲಾಗುತ್ತದೆ. ಇದು "ಪ್ರಾಯೋಗಿಕ" ಆದರೆ ಕಡಿಮೆ ಗ್ರ್ಯಾನ್ಯುಲರ್ ಮೆಟ್ರಿಕ್ ಆಗಿದೆ.

CSS (ನಿರ್ಣಾಯಕ ಈಜು ವೇಗ) ಎರಡು ವಿಭಿನ್ನ ಅಂತರಗಳಿಂದ (ಉದಾ., 400ಮೀ ಮತ್ತು 200ಮೀ) ಗಣಿತೀಯವಾಗಿ ಪಡೆದ "ಕ್ರಿಯಾತ್ಮಕ" ಮಿತಿಯಾಗಿದೆ. ಇದು "ವಾಲ್" ಅನ್ನು ಹೊಡೆಯದೆಯೇ ನೀವು ಉಳಿಸಿಕೊಳ್ಳಬಹುದಾದ ಸೈದ್ಧಾಂತಿಕ ವೇಗವಾಗಿದೆ. ನೀವು ಸಹಿಷ್ಣುತೆ ಈಜುಗಾರ ಅಥವಾ ಟ್ರಯಥ್ಲೀಟ್ ಆಗಿದ್ದರೆ, ನಿಮ್ಮ ತರಬೇತಿ ಕಾರ್ಯಕ್ರಮದಲ್ಲಿ ನಿಮ್ಮ CSS ಪ್ರಮುಖ ಸಂಖ್ಯೆಯಾಗಿದೆ.

2026 ರ ಶಿಫ್ಟ್: ಪ್ರಯತ್ನದಿಂದ ಶರೀರಶಾಸ್ತ್ರಕ್ಕೆ

ಸೈಕ್ಲಿಂಗ್ ಅಥವಾ ಓಟಕ್ಕೆ ಹೋಲಿಸಿದರೆ ಕೊಳದಲ್ಲಿ ಆಯಾಸವನ್ನು ನಿರ್ವಹಿಸುವುದು ಕುಖ್ಯಾತವಾಗಿ ಕಷ್ಟಕರವಾಗಿದೆ. ವಿದ್ಯುತ್ ಮೀಟರ್ ಇಲ್ಲದೆ, ವೇಗವು ತೀವ್ರತೆಗೆ ನಮ್ಮ ಏಕೈಕ ಪ್ರಾಕ್ಸಿಯಾಗಿದೆ. ಅನೇಕ ಈಜುಗಾರರು ಇನ್ನೂ ಹಳೆಯ-ಶಾಲೆಯ 1500m "ಸಮಯ-ಪ್ರಯೋಗ" ವೇಗವನ್ನು ಅವಲಂಬಿಸಿದ್ದಾರೆ, ಆಧುನಿಕ ತರಬೇತಿಯು ಎರಡು-ದೂರ CSS ಮಾದರಿ.

ಮುಖ್ಯ ವ್ಯತ್ಯಾಸವೆಂದರೆ ನಿಖರತೆ ವಿರುದ್ಧ ಗ್ರಹಿಸಿದ ಪ್ರಯತ್ನ. ಒಂದೇ ದೀರ್ಘ ಪ್ರಯತ್ನದ ಆಧಾರದ ಮೇಲೆ ಟಿ-ಪೇಸ್ ನಿಮ್ಮ ಮೆಟಾಬಾಲಿಕ್ ಮಿತಿಗಿಂತ ಹೆಚ್ಚಾಗಿ ನಿಮ್ಮ ಮಾನಸಿಕ ಗ್ರಿಟ್ ಅನ್ನು ಸೆರೆಹಿಡಿಯುತ್ತದೆ. CSS, ನಿಮ್ಮ "ಪೇಸ್-ಡುರೇಶನ್ ಕರ್ವ್" ನಲ್ಲಿ ಎರಡು ವಿಭಿನ್ನ ಬಿಂದುಗಳಿಂದ ಲೆಕ್ಕಹಾಕಲಾಗಿದೆ, ನಿಮ್ಮ ಎಂಜಿನ್‌ನ ನಿಜವಾದ ಏರೋಬಿಕ್ ಮಿತಿಯನ್ನು ಬಹಿರಂಗಪಡಿಸುತ್ತದೆ.

ಮೆಟ್ರಿಕ್ ಹೋಲಿಕೆ

ಥ್ರೆಶೋಲ್ಡ್ ಪೇಸ್ (ಟಿ-ಪೇಸ್)

"ಸರಳ" ಮೆಟ್ರಿಕ್

T-Pace ಅನ್ನು ಸಾಮಾನ್ಯವಾಗಿ 1500m ಅಥವಾ 1000m ಗರಿಷ್ಠ ಪ್ರಯತ್ನದಿಂದ ಪಡೆಯಲಾಗುತ್ತದೆ. ಇದು ನಿರಂತರ ಈಜುಗಾಗಿ ನಿಮ್ಮ "ರೆಡ್‌ಲೈನ್" ಅನ್ನು ಪ್ರತಿನಿಧಿಸುತ್ತದೆ.

  • ಸಾಧಕ: ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಸರಳ. ಒಂದು ಪರೀಕ್ಷೆ, ಒಂದು ಸಂಖ್ಯೆ. ದೀರ್ಘ, ನಿರಂತರ ಸೆಟ್‌ಗಳಿಗೆ ಅದ್ಭುತವಾಗಿದೆ.
  • ಕಾನ್ಸ್: ಪೇಸಿಂಗ್ ದೋಷಗಳಿಗೆ ಹೆಚ್ಚು ಸೂಕ್ಷ್ಮ. ಆಯಾಸದ ಅಡಿಯಲ್ಲಿ ನೀವು ಎಷ್ಟು ವೇಗವಾಗಿ ಮಸುಕಾಗುತ್ತೀರಿ ಎಂಬುದನ್ನು ಲೆಕ್ಕಿಸುವುದಿಲ್ಲ. ಪ್ರತಿ ಈಜುಗಾರನನ್ನು ಅದೇ 1500ಮೀ ವೇಗದಲ್ಲಿ ಒಂದೇ ರೀತಿಯ ಶಾರೀರಿಕ ಪ್ರೊಫೈಲ್ ಹೊಂದಿರುವಂತೆ ಪರಿಗಣಿಸುತ್ತದೆ.

ಕ್ರಿಟಿಕಲ್ ಸ್ವಿಮ್ ಸ್ಪೀಡ್ (CSS)

"ಕ್ರಿಯಾತ್ಮಕ" ಮೆಟ್ರಿಕ್

CSS ಎನ್ನುವುದು ಎರಡು ಅಂತರಗಳ ಆಧಾರದ ಮೇಲೆ ಗಣಿತದ ವ್ಯುತ್ಪನ್ನವಾಗಿದೆ (ಸಾಮಾನ್ಯವಾಗಿ 400m ಮತ್ತು 200m ಗರಿಷ್ಠ ಪ್ರಯತ್ನ).

  • ಸಾಧಕ: ಪೇಸಿಂಗ್ ಪಕ್ಷಪಾತವನ್ನು ತೆಗೆದುಹಾಕುತ್ತದೆ. ಮಧ್ಯಂತರದಲ್ಲಿ ನಿಮ್ಮ ನಡವಳಿಕೆಯನ್ನು ಉತ್ತಮವಾಗಿ ಮುನ್ಸೂಚಿಸುತ್ತದೆ. ವಸ್ತುನಿಷ್ಠ "ಏರೋಬಿಕ್ ಸೀಲಿಂಗ್" ಅನ್ನು ಒದಗಿಸುತ್ತದೆ.
  • ಕಾನ್ಸ್: ಎರಡು ಪ್ರತ್ಯೇಕ ಪರೀಕ್ಷೆಗಳ ಅಗತ್ಯವಿದೆ. ಸ್ವಲ್ಪ ಹೆಚ್ಚು ಸಂಕೀರ್ಣವಾದ ಗಣಿತ (ಸ್ವಿಮ್ ಅನಾಲಿಟಿಕ್ಸ್‌ನಲ್ಲಿ ಸ್ವಯಂ ಲೆಕ್ಕಾಚಾರ). ಸ್ಥಿರವಾದ ವಿಶ್ರಾಂತಿಯೊಂದಿಗೆ ಪರೀಕ್ಷೆಗಳನ್ನು ಮಾಡಬೇಕಾಗಿದೆ.

ಏಕೆ ಡೆಲ್ಟಾ ವಿಷಯಗಳು

"ಡೆಲ್ಟಾ" ನಿಮ್ಮ 400ಮೀ ವೇಗ ಮತ್ತು ನಿಮ್ಮ 200ಮೀ ವೇಗದ ನಡುವಿನ ವ್ಯತ್ಯಾಸವಾಗಿದೆ. ಇಬ್ಬರು ಈಜುಗಾರರು 1:40/100m ನ T-ಪೇಸ್ ಅನ್ನು ಹೊಂದಿರಬಹುದು, ಆದರೆ ಅವರ CSS ಪ್ರೊಫೈಲ್‌ಗಳು ಈ ರೀತಿ ಕಾಣಿಸಬಹುದು:

ಈಜುಗಾರ ಎ ​​("ದಿ ಡೀಸೆಲ್")

200m ಮತ್ತು 400m ನಡುವಿನ ಸಣ್ಣ ಡೆಲ್ಟಾ. ಈ ಈಜುಗಾರ ಹೆಚ್ಚಿನ ಏರೋಬಿಕ್ ದಕ್ಷತೆಯನ್ನು ಹೊಂದಿದೆ ಆದರೆ ಕಡಿಮೆ ಸ್ಪ್ರಿಂಟ್ ವೇಗವನ್ನು ಹೊಂದಿದೆ. ಅವರ ಸಿಎಸ್ಎಸ್ ಅವರ ಟಿ-ಪೇಸ್‌ಗೆ ಬಹಳ ಹತ್ತಿರದಲ್ಲಿದೆ.

ಈಜುಗಾರ ಬಿ ("ದಿ ಸ್ಪ್ರಿಂಟರ್")

ದೊಡ್ಡ ಡೆಲ್ಟಾ. ಗ್ರೇಟ್ 200m ವೇಗ ಆದರೆ 400m ನಲ್ಲಿ ಗಮನಾರ್ಹವಾಗಿ ಇಳಿಯುತ್ತದೆ. ಅವರ ಸಿಎಸ್ಎಸ್ ಅವರ ಟಿ-ಪೇಸ್ ಸೂಚಿಸುವುದಕ್ಕಿಂತ ಕಡಿಮೆ ಇರುತ್ತದೆ. ಅವರು ಜೆನೆರಿಕ್ "ಟಿ-ಪೇಸ್" ನಲ್ಲಿ ತರಬೇತಿ ನೀಡಲು ಪ್ರಯತ್ನಿಸಿದರೆ, ಅವರು ನಿರಂತರವಾಗಿ ತಮ್ಮ ಆಮ್ಲಜನಕರಹಿತ ವ್ಯವಸ್ಥೆಯನ್ನು ಅತಿಯಾಗಿ ಬೇಯಿಸುತ್ತಾರೆ.

📈 CSS ಫಾರ್ಮುಲಾ

CSS = (D1 - D2) / (T1 - T2)

ಇಲ್ಲಿ D1/D2 ಎರಡು ದೂರಗಳು ಮತ್ತು T1/T2 ಸೆಕೆಂಡುಗಳಲ್ಲಿ ಸಮಯ. ಇದು ನಿಮ್ಮ ಆರಂಭಿಕ ಆಮ್ಲಜನಕರಹಿತ ಉಲ್ಬಣದ ಶಬ್ದವಿಲ್ಲದೆ ನಿಮ್ಮ ಸಮರ್ಥನೀಯ ವೇಗದ "ಇಳಿಜಾರು" ಅನ್ನು ಬಹಿರಂಗಪಡಿಸುತ್ತದೆ.

CSS ಪರೀಕ್ಷೆಯನ್ನು ಹೇಗೆ ಮಾಡುವುದು (ಹಂತ-ಹಂತ)

ನಿಮ್ಮ CSS ಅನ್ನು ಲೆಕ್ಕಾಚಾರ ಮಾಡಲು, ನೀವು ಪ್ರಮಾಣಿತ ಪೂಲ್‌ನಲ್ಲಿ ವಿವಿಧ ದಿನಗಳಲ್ಲಿ (ಅಥವಾ ಪೂರ್ಣ ಚೇತರಿಕೆಯೊಂದಿಗೆ) ಎರಡು ಗರಿಷ್ಠ ಪ್ರಯೋಗಗಳನ್ನು ನಿರ್ವಹಿಸಬೇಕಾಗುತ್ತದೆ:

  1. ವಾರ್ಮ್ ಅಪ್: ನಿಮ್ಮ ದೇಹವನ್ನು ಸಿದ್ಧಪಡಿಸಲು ಸಂಪೂರ್ಣವಾದ, ಪ್ರಮಾಣೀಕೃತ ಅಭ್ಯಾಸವನ್ನು (ಸಾಮಾನ್ಯವಾಗಿ 400m ನಿಂದ 600m ಮಿಶ್ರಿತ ಸ್ಟ್ರೋಕ್‌ಗಳು ಮತ್ತು ನಿರ್ಮಾಣಗಳು) ಮಾಡಿ.
  2. ಪ್ರಯೋಗ 1 (400 ಮೀ ಗರಿಷ್ಠ ಪ್ರಯತ್ನ): ಸಾಧ್ಯವಾದಷ್ಟು ವೇಗವಾಗಿ 400 ಮೀ ಈಜಿಕೊಳ್ಳಿ. ನಿಖರವಾದ ಸಮಯವನ್ನು ರೆಕಾರ್ಡ್ ಮಾಡಿ. ಮೊದಲ 100 ಮೀಟರ್‌ನಲ್ಲಿ ನೀವು ಸುಟ್ಟು ಹೋಗದಂತೆ ನೀವೇ ವೇಗಗೊಳಿಸಿ.
  3. ಸಕ್ರಿಯ ಚೇತರಿಕೆ: ಲ್ಯಾಕ್ಟಿಕ್ ಆಮ್ಲವನ್ನು ಹೊರಹಾಕಲು 100m ನಿಂದ 200m ಈಜುವುದು ತುಂಬಾ ಸುಲಭ, ನಂತರ 5-10 ನಿಮಿಷಗಳ ವಿಶ್ರಾಂತಿ.
  4. ಪ್ರಯೋಗ 2 (200ಮೀ ಗರಿಷ್ಠ ಪ್ರಯತ್ನ): ಸಂಪೂರ್ಣ ಗರಿಷ್ಠ ಪ್ರಯತ್ನದಲ್ಲಿ 200 ಮೀ ಈಜಿಕೊಳ್ಳಿ. ನಿಖರವಾದ ಸಮಯವನ್ನು ರೆಕಾರ್ಡ್ ಮಾಡಿ.
  5. ಲೆಕ್ಕಾಚಾರ: ನಿಮ್ಮ ಸಮಯವನ್ನು ಸೆಕೆಂಡುಗಳಿಗೆ ಪರಿವರ್ತಿಸಿ ಮತ್ತು ಮೇಲಿನ ಸೂತ್ರವನ್ನು ಅನ್ವಯಿಸಿ ಅಥವಾ ನಿಮ್ಮ 100ಮೀ ವೇಗವನ್ನು ಪಡೆಯಲು ಸ್ವಿಮ್ ಅನಾಲಿಟಿಕ್ಸ್‌ನಲ್ಲಿ CSS ಕ್ಯಾಲ್ಕುಲೇಟರ್ ಉಪಕರಣವನ್ನು ಬಳಸಿ.

ನೀವು ಯಾವುದನ್ನು ಬಳಸಬೇಕು?

ಒಂದು ವೇಳೆ ಥ್ರೆಶೋಲ್ಡ್ ಪೇಸ್ (ಟಿ-ಪೇಸ್) ಬಳಸಿ:

  • ನೀವು ವೇಗ ಆಧಾರಿತ ತರಬೇತಿಯೊಂದಿಗೆ ಪ್ರಾರಂಭಿಸುತ್ತಿದ್ದೀರಿ.
  • ನೀವು ದೀರ್ಘ, ನಿರಂತರ ಈಜುಗಳನ್ನು ಮಾತ್ರ ನಿರ್ವಹಿಸುತ್ತೀರಿ (ತೆರೆದ ನೀರು 3km+).
  • ನೀವು ಬಹು ಪರೀಕ್ಷಾ ಅವಧಿಗಳೊಂದಿಗೆ ವ್ಯವಹರಿಸಲು ಬಯಸುವುದಿಲ್ಲ.

ಒಂದು ವೇಳೆ ಕ್ರಿಟಿಕಲ್ ಸ್ವಿಮ್ ಸ್ಪೀಡ್ (CSS) ಬಳಸಿ:

  • ಟ್ರಯಥ್ಲೆಟ್ಸ್ ಮತ್ತು ಮ್ಯಾರಥಾನ್ ಈಜುಗಾರರು: ಓಟದ ಆರಂಭದಲ್ಲಿ "ಮೇಣದಬತ್ತಿಯನ್ನು ಸುಡುವುದನ್ನು" ತಪ್ಪಿಸಲು.
  • ಮಧ್ಯಂತರ ತರಬೇತಿ: ನಿರ್ದಿಷ್ಟ CSS + 2s ಅಥವಾ CSS ಹೊಂದಿಸಲು - 1s ವಲಯಗಳು ನಿಜವಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತವೆ.
  • ಟ್ರ್ಯಾಕಿಂಗ್ ಪ್ರಗತಿ: CSS ಸಾಮಾನ್ಯ 1000m ಪರೀಕ್ಷೆಗಿಂತ ಏರೋಬಿಕ್ ಸುಧಾರಣೆಗಳಿಗೆ ಹೆಚ್ಚು ಸೂಕ್ಷ್ಮವಾಗಿರುತ್ತದೆ.

ಥ್ರೆಶೋಲ್ಡ್ ಪೇಸ್ ವಿರುದ್ಧ CSS FAQ

CSS ಮಿತಿ ವೇಗದಂತೆಯೇ ಇದೆಯೇ?

ಇಲ್ಲ. ಅವು ಸಂಬಂಧಿತ ಮಿತಿ ಪರಿಕಲ್ಪನೆಗಳು, ಆದರೆ ಅವು ಒಂದೇ ಪರೀಕ್ಷೆಯಲ್ಲ. ಥ್ರೆಶೋಲ್ಡ್ ಪೇಸ್ ಸಾಮಾನ್ಯವಾಗಿ ಒಂದು ದೀರ್ಘವಾದ ಈಜುವಿಕೆಯಿಂದ ಬರುತ್ತದೆ, ಆದರೆ CSS ಅನ್ನು ಎರಡು ಪ್ರಯತ್ನಗಳಿಂದ ಲೆಕ್ಕಹಾಕಲಾಗುತ್ತದೆ ಮತ್ತು ಮಧ್ಯಂತರ ಕೆಲಸಕ್ಕಾಗಿ ಹೆಚ್ಚು ಪುನರಾವರ್ತನೀಯ ಆಂಕರ್ ಅನ್ನು ನೀಡುತ್ತದೆ.

ಟ್ರೈಯಥ್ಲಾನ್ ಮತ್ತು ಓಪನ್-ವಾಟರ್ ಪೇಸಿಂಗ್‌ಗೆ ಯಾವುದು ಉತ್ತಮ?

CSS ಸಾಮಾನ್ಯವಾಗಿ ಉತ್ತಮ ಆರಂಭದ ಹಂತವಾಗಿದೆ ಏಕೆಂದರೆ ಇದು ಅಲ್ಪಾವಧಿಯ ವೇಗವನ್ನು ಹೆಚ್ಚು ಪ್ರತಿಫಲ ನೀಡದೆ ಸಮರ್ಥನೀಯ ವೇಗವನ್ನು ಪ್ರತಿಬಿಂಬಿಸುತ್ತದೆ. ದೀರ್ಘ ಸ್ಥಿರವಾದ ಈಜುಗಳಿಗೆ ಥ್ರೆಶೋಲ್ಡ್ ಪೇಸ್ ಇನ್ನೂ ಉಪಯುಕ್ತವಾಗಿದೆ, ಆದರೆ ಸಿಎಸ್ಎಸ್ ಸಾಮಾನ್ಯವಾಗಿ ಪೇಸಿಂಗ್ ಟಾರ್ಗೆಟ್‌ಗಳು ಮತ್ತು ಝೋನ್ ವರ್ಕ್ ಆಗಿ ಪರಿವರ್ತಿಸಲು ಸುಲಭವಾಗಿದೆ.

ನಾನು ಥ್ರೆಶೋಲ್ಡ್ ಪೇಸ್‌ನಿಂದ CSS-ಆಧಾರಿತ ಈಜು ವಲಯಗಳಿಗೆ ಹೇಗೆ ಚಲಿಸುವುದು?

CSS ಅನ್ನು ಹೊಂದಿಸಲು ನಿಮ್ಮ ಮುಂದಿನ 400m ಮತ್ತು 200m ಪರೀಕ್ಷೆಯನ್ನು ಬಳಸಿ, ನಂತರ ಆ ಸಂಖ್ಯೆಯ ಸುತ್ತಲೂ ಆಂಕರ್ ಮಧ್ಯಂತರ ಕೆಲಸ ಮಾಡಿ. ನೀವು ಈಗಾಗಲೇ T-Pace ಮೂಲಕ ತರಬೇತಿ ಪಡೆದಿದ್ದರೆ, ಎರಡೂ ಸಂಖ್ಯೆಗಳನ್ನು ಹೋಲಿಕೆ ಮಾಡಿ ಮತ್ತು ಪುನರಾವರ್ತನೀಯ ಏರೋಬಿಕ್ ವೇಗದ ಸುತ್ತಲೂ ನಿಮ್ಮ ಸೆಟ್‌ಗಳನ್ನು ಮರುನಿರ್ಮಾಣ ಮಾಡಲು CSS ಮಾರ್ಗದರ್ಶಿ ಜೊತೆಗೆ ಈಜು ತರಬೇತಿ ವಲಯಗಳನ್ನು ಬಳಸಿ.

ನಿಮ್ಮ ಪೂಲ್ ವಲಯಗಳನ್ನು ಊಹಿಸುವುದನ್ನು ನಿಲ್ಲಿಸಿ.

ಸ್ವಿಮ್ ಅನಾಲಿಟಿಕ್ಸ್ ನಿಮಗೆ CSS ಅನ್ನು ಪ್ರಾಯೋಗಿಕ ಪೂಲ್ ವಲಯಗಳಾಗಿ ಪರಿವರ್ತಿಸಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ, ಮಧ್ಯಂತರ ಪೇಸಿಂಗ್ ಮತ್ತು ಕಾಲಾನಂತರದಲ್ಲಿ ನಿಮ್ಮ ಏರೋಬಿಕ್ ಸೀಲಿಂಗ್‌ನ ಸ್ಪಷ್ಟ ನೋಟವನ್ನು ನೀಡುತ್ತದೆ.
