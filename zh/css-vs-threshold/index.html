临界游泳速度 (CSS) 与阈值配速：哪一个重要？

多年来，“阈值速度”(T-Pace) 一直是黄金标准。 2026 年，向临界游泳速度 (CSS) 的转变将重新定义运动员如何调整训练节奏和预测比赛日疲劳程度。

🎯 快速回答

T 步速（阈值步速） 是对您可以维持一小时的配速的一般估计，通常用于设置广泛的训练区域。这是一个“实用”但不太细粒度的指标。

CSS（临界游泳速度） 是从两个不同距离（例如 400m 和 200m）以数学方式得出的“功能”阈值。这是您在不撞到“墙”的情况下可以维持的理论速度。如果您是耐力游泳运动员或铁人三项运动员，CSS 是您训练计划中最重要的数字。

2026 年的转变：从努力到生理

与骑自行车或跑步相比，在泳池中控制疲劳是众所周知的困难。如果没有功率计，配速是我们衡量强度的唯一指标。虽然许多游泳运动员仍然依赖老式的 1500 米“计时赛”配速，但现代教练已经转向 两距离 CSS 模型.

主要区别是 精确度与感知努力。基于单次长时间努力的 T-Pace 往往比新陈代谢阈值更能激发您的精神毅力。 CSS 根据“步速-持续时间曲线”上的两个不同点计算得出，揭示了引擎的实际有氧极限。

指标比较

阈值配速（T 配速）

“简单”指标

T-Pace 通常源自单次 1500m 或 1000m 最大努力。它代表您连续游泳的“红线”。

  • 优点： 简单易懂。一项测试，一个数字。非常适合长时间、连续的训练。
  • 缺点： 对起搏错误高度敏感。不考虑您在疲劳情况下褪色的速度。以相同的 1500m 配速对待每位游泳运动员，使其具有相同的生理特征。

临界游泳速度 (CSS)

“功能”指标

CSS 是基于两个距离（通常是 400m 和 200m 最大努力）的数学推导。

  • 优点： 消除节奏偏差。更好地预测您的间歇行为。提供客观的“有氧上限”。
  • 缺点： 需要两个单独的测试。稍微复杂的数学（在游泳分析中自动计算）。要求在持续休息的情况下进行测试。

为什么三角洲很重要

“Delta”是 400m 配速与 200m 配速之间的差异。两名游泳运动员的 T 步速可能均为 1:40/100m，但他们的 CSS 配置文件可能如下所示：

游泳运动员A（“柴油机”）

200m 至 400m 之间的小三角洲。这位游泳运动员的有氧效率非常高，但冲刺速度较低。他们的 CSS 非常接近他们的 T-Pace。

游泳运动员B（“短跑运动员”）

大三角洲。 200m 速度很好，但 400m 速度明显下降。他们的 CSS 将低于 T-Pace 建议的水平。如果他们尝试以通用的“T-Pace”进行训练，他们的无氧系统就会不断过度运转。

📈 CSS 公式

CSS = (D1 - D2) / (T1 - T2)

其中 D1/D2 是两个距离，T1/T2 是以秒为单位的时间。这揭示了您可持续配速的“斜率”，而没有最初无氧激增的噪音。

如何执行 CSS 测试（分步）

要计算您的 CSS，您需要在不同的日子（或之间完全恢复）在标准池中执行两次最大试验：

  1. 热身： 进行彻底、标准化的热身（通常为 400m 至 600m 的混合划水和训练），为您的身体做好准备。
  2. 试验 1（400m 最大努力）： 尽可能快地游400m。记录准确的时间。调整自己的节奏，以免在前 100 米中精疲力尽。
  3. 主动恢复： 游泳100m到200m很容易排出乳酸，然后休息5-10分钟。
  4. 试验 2（200m 最大努力）： 以绝对最大努力游200m。记录准确的时间。

您应该使用哪个？

如果出现以下情况，请使用阈值配速 (T-Pace)：

  • 您刚刚开始基于配速的训练。
  • 您只能进行长时间、连续的游泳（开放水域 3 公里以上）。
  • 您不想处理多个测试会话。

在以下情况下使用临界游泳速度 (CSS)：

  • 铁人三项运动员和马拉松游泳运动员： 避免在比赛中过早“燃烧蜡烛”。
  • 间歇训练： 设置实际有效的特定 CSS + 2s 或 CSS - 1s 区域。
  • 跟踪进度： CSS 对有氧运动的改善比一般的 1000m 测试更敏感。

阈值速度与 CSS 常见问题解答

CSS 与阈值速度相同吗？

不。它们是相关的阈值概念，但不是同一个测试。阈值配速通常来自一次较长的游泳，而 CSS 则根据两次努力计算得出，并为间歇训练提供了更具可重复性的锚点。

铁人三项和开放水域配速哪个更好？

CSS 通常是更好的起点，因为它反映了可持续的步伐，而不会过度奖励短期速度。阈值配速对于长时间稳定游泳仍然有用，但 CSS 通常更容易转换为配速目标和区域工作。

如何从阈值速度转移到基于 CSS 的游泳区域？

使用下一个 400m 和 200m 测试来设置 CSS，然后围绕该数字设置锚定间隔。如果您已经通过 T-Pace 进行训练，请比较这两个数字并使用 CSS 指南和游泳训练区围绕可重复的有氧配速重建您的组数。

别再猜测你的泳池区域了。
