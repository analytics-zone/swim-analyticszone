什么是游泳 TSS (sTSS)？

游泳训练压力评分 (sTSS) 通过结合强度和持续时间来量化游泳训练的负荷。它改编自自行车的 TSS 方法论，使用您的临界游泳速度 (CSS) 作为阈值配速。以 CSS 配速进行 1 小时的训练 = 100 sTSS。

训练负荷解读

sTSS 是如何计算的

公式

sTSS = (小时持续时间) × (强度系数)² × 100

其中:

  • 强度系数 (IF) = CSS 配速 / 平均训练配速
  • 持续时间 = 以小时为单位的总训练时间
  • CSS 配速 = 来自 CSS 测试的阈值配速

计算示例

训练详情:

  • CSS 配速: 1:49/100m (109 秒)
  • 训练持续时间: 60 分钟 (1 小时)
  • 平均配速: 2:05/100m (125 秒)

步骤 1: 计算强度系数

IF = CSS 配速 / 训练配速
IF = 109 / 125
IF = 0.872

步骤 2: 计算 sTSS

sTSS = 1.0 小时 × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

解读: 这次以轻松配速 (慢于 CSS) 进行的 60 分钟训练产生了 76 sTSS - 适合建立有氧基础的中等训练负荷。

理解 sTSS 值

< 50 低 当天 轻松 30 分钟游泳，技术练习
50-100 中等 1 天 60 分钟耐力，稳定配速

每周 sTSS: 800-1200+

每周 8-12 次训练，双练。高训练量，恢复管理至关重要。

⚠️ 重要注意事项

  • 需要准确的 CSS: 您的 CSS 必须是最新的 (在 6-8 周内测试) 才能获得准确的 sTSS。
  • 简化计算: 此计算器使用平均配速。高级 sTSS 使用标准化分级配速 (NGP)，它考虑了间歇结构。
  • 不适用于技术训练: sTSS 仅测量身体训练压力，不测量技能发展。
  • 个体差异: 相同的 sTSS 对不同的游泳者感觉不同。根据您的恢复情况调整指南。

为什么 sTSS 很重要

训练压力评分是以下内容的基础:

  • CTL (慢性训练负荷): 您的体能水平 - 每日 sTSS 的 42 天指数加权平均值
  • ATL (急性训练负荷): 您的疲劳度 - 每日 sTSS 的 7 天指数加权平均值
  • TSB (训练压力平衡): 您的状态 - TSB = CTL - ATL (正值 = 清新，负值 = 疲劳)
  • 周期化: 使用目标 CTL 进展计划训练阶段 (基础，构建，巅峰，减量)
  • 恢复管理: 根据 TSB 知道何时该努力，何时该休息

专业提示: 跟踪您的 CTL

在电子表格或训练日志中记录每日 sTSS。每周计算您的 42 天平均值 (CTL)。在基础构建期间，目标是每周增加 5-10 个 CTL 点。在减量期间 (比赛前 1-2 周)，保持或略微降低 CTL。

常见问题

什么是游泳 TSS (sTSS)？

游泳训练压力评分 (sTSS) 是一项通过结合强度和持续时间来量化游泳训练负荷的指标。它改编自自行车的 TSS 方法论，使用您的临界游泳速度 (CSS) 作为阈值配速。以 CSS 配速进行 1 小时的训练等于 100 sTSS。

如何计算我的 sTSS？

使用上面的计算器，输入您的 CSS 配速（来自 CSS 测试）、总训练持续时间和训练期间的平均配速。公式为：sTSS = 持续时间（小时）× 强度系数² × 100，其中强度系数 = CSS 配速 / 平均训练配速。

我需要 CSS 来计算 sTSS 吗？

是的，计算强度系数需要您的临界游泳速度 (CSS)，这对 sTSS 计算至关重要。CSS 代表您的阈值配速，应每 6-8 周测试一次。您可以使用我们的 CSS 计算器找到您的 CSS。

一次训练的良好 sTSS 分数是多少？

这取决于训练强度：轻松的训练通常得分低于 50 sTSS，中等训练 50-100 sTSS，艰苦训练 100-200 sTSS，非常艰苦的训练超过 200 sTSS。合适的分数取决于您的训练目标和当前的体能水平。

我每周应该做多少 sTSS？

每周 sTSS 目标因水平而异：休闲游泳者：150-300，健身游泳者/铁人三项运动员：300-500，竞技大师：500-800，精英/大学生：800-1200+。保守开始并逐渐增加以避免过度训练。

游泳TSS与骑行TSS相同吗？

概念和公式相同，但sTSS是为游泳而调整的。sTSS不像骑行TSS那样使用功率（FTP），而是使用以CSS为阈值的配速。两者都使用持续时间 × 强度系数² × 100来测量训练负荷。

我可以对所有游泳姿势使用sTSS吗？

可以，但您的CSS应该是特定于姿势的。大多数游泳者使用自由泳CSS，因为这是最常训练的姿势。如果您主要训练其他姿势，请在该姿势中进行CSS测试，并使用该配速进行sTSS计算。

sTSS与CTL/ATL/TSB有什么区别？

sTSS测量单次训练的训练负荷。CTL（慢性训练负荷）是您的长期体能，ATL（急性训练负荷）是您最近的疲劳，TSB（训练压力平衡）是您的新鲜度。这些指标使用随时间变化的sTSS值来跟踪您的训练状态。在我们的训练负荷指南中了解更多。
