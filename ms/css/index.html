Kelajuan Renang Kritikal (CSS)

Asas Latihan Renang Dipacu Data

Apakah Kelajuan Renang Kritikal (CSS)?

Kelajuan Renang Kritikal (CSS)ialah halaju renang maksimum teori yang anda boleh kekalkan tanpa keletihan. Ia mewakili kadar ambang aerobik anda, biasanya sepadan dengan laktat darah 4 mmol/L dan mampan selama kira-kira 30 minit. CSS dikira menggunakan percubaan masa 400m dan 200m untuk menentukan zon latihan yang diperibadikan.

Kelajuan Renang Kritikal (CSS) mewakili halaju renang maksimum teori yang anda boleh kekalkan secara berterusan tanpa keletihan. Ia adalah ambang aerobik anda dalam air—keamatan pengeluaran laktat sama dengan pelepasan laktat.

🎯 Kepentingan Fisiologi

CSS berpadanan rapat dengan:

  • Ambang Laktat 2 (LT2)- Ambang pengudaraan kedua
  • Keadaan Tetap Laktat Maksimum (MLSS)- Paras laktat mampan tertinggi
  • Kelajuan Ambang Fungsian (FTP)- Berenang setaraf dengan berbasikal FTP
  • ~4 mmol/L darah laktat- Penanda OBLA tradisional

Mengapa CSS Penting

CSS ialahmetrik asasyang membuka kunci semua analisis beban latihan lanjutan:

  • Zon Latihan:Memperibadikan zon intensiti berdasarkan fisiologi anda
  • sTSS Pengiraan:Membolehkan kuantifikasi Skor Tekanan Latihan yang tepat
  • CTL/ATL/TSB:Diperlukan untuk metrik Carta Pengurusan Prestasi
  • Penjejakan Kemajuan:Ukuran objektif peningkatan kecergasan aerobik
⚠️ Kebergantungan Kritikal:Tanpa ujian CSS yang sah, metrik beban latihan lanjutan (sTSS, CTL, ATL, TSB) tidak boleh dikira. CSS yang tidak tepat akan merosakkan semua analisis latihan seterusnya.

Cara Mengira CSS dalam 4 Langkah Mudah

📋 Protokol Standard

  1. Memanaskan badan

    300-800m berenang mudah, latih tubi dan binaan progresif untuk bersedia untuk usaha maksimum.

  2. Percubaan Masa 400m

    Usaha berterusan maksimum dari permulaan tolak (tiada menyelam). Catat masa ke detik. Matlamat: 400m mampan terpantas.

  3. Pemulihan Lengkap

    5-10 minitberenang mudah atau berehat sepenuhnya. Ini adalah KRITIKAL untuk hasil yang tepat.

  4. Percubaan Masa 200m

    Usaha maksimum dari permulaan tolak. Catat masa dengan tepat. Ini sepatutnya lebih laju setiap 100m daripada 400m.

⚠️ Kesilapan Biasa

Pemulihan Tidak Mencukupi

Masalah:Keletihan secara buatan memperlahankan masa 200m

Keputusan:CSS yang dikira menjadilebih pantas daripada realiti, membawa kepada zon yang terlalu terlatih

Penyelesaian:Rehat sehingga HR turun di bawah 120 bpm atau sehingga pernafasan pulih sepenuhnya

Pacing yang lemah pada 400m

Masalah:Bermula terlalu cepat menyebabkan kelembapan dramatik

Keputusan:Masa 400m tidak menggambarkan rentak mampan yang sebenar

Penyelesaian:Bertujuan untuk pemisahan genap atau pemisahan negatif (200m kedua ≤ 200m pertama)

Menggunakan Dive Starts

Masalah:Menambah ~0.5-1.5 saat kelebihan, pengiraan condong

Penyelesaian:Sentiasa gunakan tolak mula dari dinding

🔄 Kekerapan Menguji Semula

Uji semula CSS setiap6-8 mingguuntuk mengemas kini zon latihan apabila kecergasan bertambah baik. Zon anda sepatutnya menjadi lebih pantas apabila anda menyesuaikan diri dengan latihan.

Formula Pengiraan CSS

Formula

CSS (m/s) = (D₂ - D₁) / (T₂ - T₁)

di mana:

  • D₁ = 200 meter
  • D₂ = 400 meter
  • T₁ = Masa untuk 200m (dalam saat)
  • T₂ = Masa untuk 400m (dalam saat)

Dipermudahkan untuk Pace setiap 100m

CSS Pace/100m (saat) = (T₄₀₀ - T₂₀₀) / 2

Contoh Kerja

Keputusan Ujian:

  • 400m masa:6:08(368 saat)
  • 200m masa:2:30(150 saat)

Langkah 1: Kira CSS dalam m/s

CSS = (400 - 200) / (368 - 150)
CSS = 200 / 218
CSS =0.917 m/s

Langkah 2: Tukar kepada rentak setiap 100m

Pace = 100 / 0.917
Pace = 109 saat
Pace =1:49 setiap 100m

🎯 Faedah Latihan Berasaskan Zon

Menggunakan zon berasaskan CSS mengubah latihan "rasa" subjektif kepada latihan yang objektif dan berulang. Setiap zon menyasarkan penyesuaian fisiologi tertentu:

  • Zon 2:Bina enjin aerobik (60-70% daripada volum mingguan)
  • Zon 3:Tingkatkan kecekapan perlumbaan (15-20% daripada volum)
  • Zon 4:Tolak ambang laktat lebih tinggi (10-15% daripada volum)
  • Zon 5:Kembangkan kelajuan dan kuasa atasan (5-10% daripada volum)

🏊 Kumpulan Umur Berdaya saing

🌊 Perenang Membangunkan

<0.9 m/s
>1:51 setiap 100m

Membina asas dan teknik aerobik. Kurang daripada 1-2 tahun latihan yang konsisten.

Pengesahan Saintifik

Wakayoshi et al. (1992-1993) - Penyelidikan Asas

Kajian seminal Kohji Wakayoshi di Universiti Osaka menetapkan CSS sebagai alternatif yang sah dan praktikal untuk ujian laktat makmal:

  • Korelasi kuat dengan VO₂ pada ambang anaerobik(r = 0.818)
  • Kaitan yang sangat baik dengan halaju pada OBLA(r = 0.949)
  • Meramalkan prestasi 400m(r = 0.864)
  • Sepadan dengan laktat darah 4 mmol/L- keadaan mantap laktat maksimum
  • Hubungan linear antara jarak dan masa(r² > 0.998)

Kertas Utama:

  1. Wakayoshi K, et al. (1992). "Penentuan dan kesahihan halaju kritikal sebagai indeks prestasi renang dalam perenang yang kompetitif."Jurnal Fisiologi Gunaan Eropah, 64(2), 153-157.
  2. Wakayoshi K, et al. (1992). "Kaedah mudah untuk menentukan kelajuan kritikal sebagai ambang keletihan berenang dalam renang kompetitif."Jurnal Perubatan Sukan Antarabangsa, 13(5), 367-371.
  3. Wakayoshi K, et al. (1993). "Adakah halaju renang kritikal mewakili intensiti senaman pada keadaan mantap laktat maksimum?"Jurnal Fisiologi Gunaan Eropah, 66(1), 90-95.

🔬 Mengapa CSS Berfungsi

CSS mewakili sempadan antaradomain senaman berat dan teruk. Di bawah CSS, pengeluaran dan pelepasan laktat kekal seimbang—anda boleh berenang untuk tempoh yang lama. Di atas CSS, laktat terkumpul secara progresif sehingga keletihan dalam masa 20-40 minit.

Ini menjadikan CSS keamatan yang sempurna untuk:

  • Menetapkan rentak perlumbaan yang mampan untuk acara 800m-1500m
  • Menetapkan latihan selang ambang
  • Memantau peningkatan kecergasan aerobik
  • Mengira beban latihan dan keperluan pemulihan

Cara Menguji CSS Anda

Langkah 1: Memanaskan Badan dengan Betul

Selesaikan 300-800 meter berenang mudah, termasuk latihan dan binaan progresif. Ini menyediakan badan anda untuk usaha yang maksimum dan mengelakkan kecederaan.

Langkah 2: Lakukan Ujian Masa 400m

Berenang 400 meter pada usaha berterusan maksimum dari permulaan tolak (tiada menyelam). Catat masa anda ke detik. Sasarkan untuk belahan genap atau belahan negatif (200m kedua bersamaan atau lebih laju daripada 200m pertama).

Langkah 3: Pulih Sepenuhnya

Rehat selama 5-10 minit dengan berenang mudah atau rehat sepenuhnya. Pemulihan ini adalah KRITIKAL. Tunggu sehingga degupan jantung anda menurun di bawah 120 bpm dan pernafasan pulih sepenuhnya sebelum meneruskan.

Langkah 4: Lakukan Ujian Masa 200m

Berenang 200 meter pada usaha maksimum dari permulaan tolak. Catat masa anda dengan tepat. Ini sepatutnya lebih pantas setiap 100m daripada rentak 400m anda.

Langkah 5: Kira CSS Anda

Gunakan formula: CSS Pace/100m = (T400 - T200) / 2. Contoh: (368s - 150s) / 2 = 109s = 1:49/100m. Atau gunakan kalkulator pada halaman ini untuk mendapatkan hasil segera dengan zon latihan yang diperibadikan.

Aplikasi Praktikal

1️⃣ Buka Kunci Metrik Beban Latihan

CSS ialah penyebut dalam pengiraan Faktor Intensiti untuk sTSS. Tanpanya, anda tidak boleh mengira tekanan senaman atau menjejaki trend kecergasan/keletihan.

2️⃣ Peribadikan Zon Latihan

Carta rentak generik tidak mengambil kira fisiologi individu. Zon berasaskan CSS memastikan setiap perenang berlatih pada keamatan optimum mereka.

3️⃣ Pantau Perkembangan Kecergasan

Uji semula setiap 6-8 minggu. Menambah baik CSS (laju lebih pantas) menunjukkan penyesuaian aerobik yang berjaya. CSS bertakung mencadangkan latihan memerlukan pelarasan.

4️⃣ Ramalkan Prestasi Perlumbaan

Kepantasan CSS menghampiri rentak perlumbaan 30 minit anda yang mampan. Gunakannya untuk menetapkan matlamat yang realistik untuk acara 800m, 1500m dan air terbuka.

5️⃣ Latihan Ambang Reka Bentuk

Set CSS klasik: 8×100 @ CSS rentak (15s rehat), 5×200 @ 101% CSS (20s rehat), 3×400 @ 103% CSS (30s rehat). Bina kapasiti pelepasan laktat.

6️⃣ Optimumkan Strategi Tirus

Jejaki CSS sebelum dan selepas tirus. Tirus yang berjaya mengekalkan atau menambah baik sedikit CSS sambil mengurangkan keletihan (peningkatan TSB).

Gunakan Pengetahuan CSS Anda

Kini setelah anda memahami Kelajuan Renang Kritikal, ambil langkah seterusnya untuk mengoptimumkan latihan anda:

  • Kira Skor Tekanan Latihan (TSS)berdasarkan CSS anda untuk mengukur intensiti senaman
