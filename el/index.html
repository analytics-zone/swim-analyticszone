Επιστημονικές Μετρήσεις

Η Κρίσιμη Ταχύτητα Κολύμβησης (CSS) καθορίζει το αερόβιο κατώφλι σας, επιτρέποντας τον υπολογισμό του Σκορ Καταπόνησης Προπόνησης (TSS) και την παρακολούθηση απόδοσης CTL/ATL/TSB με βάση αποδεδειγμένη έρευνα αθλητικής επιστήμης. Μάθετε περισσότερα για το δικό μαςμαθηματικούς τύπουςκαιμετρήσεις απόδοσης κολύμβησης.

Ζώνες Προπόνησης

7 εξατομικευμένες ζώνες προπόνησης βαθμονομημένες στο CSS σας. Βελτιστοποιήστε κάθε προπόνηση για αποκατάσταση, αερόβια, κατώφλι ή ανάπτυξη VO₂max.

Έξυπνες Συγκρίσεις

Εβδομαδιαίες, μηνιαίες και ετήσιες συγκρίσεις περιόδων με αυτόματη ανίχνευση τάσεων και ποσοστιαίες αλλαγές για όλες τις μετρήσεις.

Απόλυτη Ιδιωτικότητα

Όλα τα δεδομένα επεξεργάζονται τοπικά στη συσκευή σας. Χωρίς διακομιστές, χωρίς cloud, χωρίς παρακολούθηση. Εσείς έχετε την ιδιοκτησία και τον έλεγχο των δεδομένων κολύμβησης σας.

Εξαγωγή Παντού

Εξάγετε προπονήσεις και αναλύσεις σε μορφές JSON, CSV, HTML ή PDF. Συμβατό με προπονητές, υπολογιστικά φύλλα και πλατφόρμες προπόνησης.

Άμεση Απόδοση

Εκκίνηση εφαρμογής σε λιγότερο από 0,35 δευτερόλεπτα με αρχιτεκτονική local-first. Δείτε τις προπονήσεις σας άμεσα χωρίς να περιμένετε συγχρονισμούς ή λήψεις.

Επαγγελματικές λειτουργίες χωρίς πολυπλοκότητα

Πρωτόκολλο Δοκιμής CSS

Ενσωματωμένο πρωτόκολλο δοκιμής 400μ + 200μ για τον προσδιορισμό της Κρίσιμης Ταχύτητας Κολύμβησης σας. Επαναλάβετε κάθε 6-8 εβδομάδες για να παρακολουθείτε την πρόοδο και να προσαρμόζετε αυτόματα τις ζώνες προπόνησης.

Εγγενής Εμπειρία iOS

Κατασκευασμένο με SwiftUI για ομαλή απόδοση και ενσωμάτωση iOS. Απρόσκοπτος συγχρονισμός με την εφαρμογή Υγεία, υποστήριξη widget και οικεία γλώσσα σχεδιασμού Apple.

Βασισμένο σε Έρευνα

Όλες οι μετρήσεις βασίζονται σε αξιολογημένη έρευνα αθλητικής επιστήμης. CSS από Wakayoshi et al., TSS προσαρμοσμένο για κολύμβηση με τύπο IF³, αποδεδειγμένα μοντέλα CTL/ATL.

Φιλικό προς Προπονητές

Εξάγετε λεπτομερείς αναφορές για προπονητές. Μοιραστείτε περιλήψεις HTML μέσω email, CSV για ανάλυση σε υπολογιστικά φύλλα ή PDF για ημερολόγια προπόνησης.

Λειτουργεί Παντού

Πισίνα ή ανοιχτή θάλασσα, σπριντ ή απόσταση. Το Swim Analytics προσαρμόζεται σε όλους τους τύπους κολύμβησης και ανιχνεύει αυτόματα τα χαρακτηριστικά της προπόνησης.

Συνεχής Βελτίωση

Τακτικές ενημερώσεις με νέες λειτουργίες με βάση τα σχόλια των χρηστών. Πρόσφατες προσθήκες περιλαμβάνουν ετήσιες συγκρίσεις, παρακολούθηση προσωπικών ρεκόρ και βελτιωμένες επιλογές εξαγωγής.

FAQ

Συχνές Ερωτήσεις

Πώς λαμβάνει το Swim Analytics τα δεδομένα κολύμβησης μου;

Το Swim Analytics συγχρονίζεται με το Apple Health για να εισάγει προπονήσεις κολύμβησης που έχουν καταγραφεί από οποιαδήποτε συμβατή συσκευή ή εφαρμογή. Αυτό περιλαμβάνει έξυπνα ρολόγια, fitness trackers και χειροκίνητες καταχωρήσεις. Η εφαρμογή επεξεργάζεται αυτά τα δεδομένα τοπικά για να υπολογίσει προηγμένες μετρήσεις.

Τι είναι το τεστ CSS και πώς το κάνω;

Το CSS (Critical Swim Speed) είναι ένα επιστημονικό πρωτόκολλο που χρησιμοποιεί 2 κολυμβήσεις μέγιστης προσπάθειας: 400μ και 200μ με 10-20 λεπτά ανάπαυσης ενδιάμεσα. Η εφαρμογή υπολογίζει το αερόβιο κατώφλι σας από αυτούς τους χρόνους και προσαρμόζει αυτόματα όλες τις ζώνες προπόνησης. Επαναλάβετε κάθε 6-8 εβδομάδες για να παρακολουθείτε την πρόοδο.

Ανεβαίνουν τα δεδομένα μου στο cloud;

Όχι. Το Swim Analytics επεξεργάζεται όλα τα δεδομένα τοπικά στο iPhone σας. Δεν υπάρχουν εξωτερικοί διακομιστές, λογαριασμοί cloud, μεταφορές δεδομένων. Εσείς ελέγχετε τις εξαγωγές: δημιουργήστε αρχεία JSON, CSV, HTML ή PDF και μοιραστείτε τα όπως θέλετε.

Μπορώ να χρησιμοποιήσω το Swim Analytics για κολύμβηση ανοιχτής θάλασσας;

Ναι. Το Swim Analytics λειτουργεί με οποιαδήποτε προπόνηση κολύμβησης στο Apple Health, συμπεριλαμβανομένης της ανοιχτής θάλασσας. Η εφαρμογή προσαρμόζεται στις διαθέσιμες μετρήσεις είτε είστε σε πισίνα είτε σε ανοιχτή θάλασσα, παρέχοντας σχετική ανάλυση για κάθε περιβάλλον.

Ποια είναι η διαφορά μεταξύ μηνιαίων και ετήσιων προγραμμάτων;

Και τα δύο προγράμματα προσφέρουν πανομοιότυπες λειτουργίες: όλες οι μετρήσεις, απεριόριστες ζώνες, χρονικές συγκρίσεις, πολλαπλές εξαγωγές και δωρεάν ενημερώσεις. Η μόνη διαφορά είναι η τιμή: το ετήσιο εξοικονομεί 18% (ισοδύναμο με €3.25/μήνα έναντι €3.99/μήνα).

Μπορώ να ακυρώσω τη συνδρομή μου ανά πάσα στιγμή;

Ναι. Οι συνδρομές διαχειρίζονται μέσω του App Store, οπότε μπορείτε να ακυρώσετε ανά πάσα στιγμή από Ρυθμίσεις → [Το Όνομά Σας] → Συνδρομές. Εάν ακυρώσετε, θα διατηρήσετε την πρόσβαση μέχρι το τέλος της τρέχουσας περιόδου χρέωσης.

Είστε Έτοιμοι να Μεταμορφώσετε την Κολύμβησή σας;

Γίνετε μέλος χιλιάδων κολυμβητών που χρησιμοποιούν επιστημονικές μετρήσεις για να βελτιώσουν την απόδοση. Ξεκινήστε τη δωρεάν δοκιμή 7 ημερών σήμερα.
