Κρίσιμη ταχύτητα κολύμβησης (CSS) έναντι ρυθμού κατωφλίου: Ποιο έχει σημασία;

Για χρόνια, το "Threshold Pace" (T-Pace) ήταν το χρυσό πρότυπο. Το 2026, η στροφή στην Κρίσιμη Ταχύτητα Κολύμβησης (CSS) επαναπροσδιορίζει τον τρόπο με τον οποίο οι αθλητές ρυθμίζουν τις συνεδρίες τους και προβλέπουν την κόπωση την ημέρα του αγώνα.

🎯 Η Γρήγορη Απάντηση

T-Pace (Threshold Pace) είναι μια γενική εκτίμηση του ρυθμού που μπορείτε να διατηρήσετε για μια ώρα, που χρησιμοποιείται συχνά για τον καθορισμό ευρειών ζωνών προπόνησης. Είναι μια «πρακτική» αλλά λιγότερο αναλυτική μέτρηση.

CSS (Κρίσιμη ταχύτητα κολύμβησης) είναι το «λειτουργικό» κατώφλι που προκύπτει μαθηματικά από δύο διαφορετικές αποστάσεις (π.χ. 400m και 200m). Είναι ο θεωρητικός ρυθμός που μπορείτε να διατηρήσετε χωρίς να χτυπήσετε τον «Τοίχο». Εάν είστε κολυμβητής αντοχής ή αθλητής τριάθλου, το CSS σας είναι ο πιο σημαντικός αριθμός στο προπονητικό σας πρόγραμμα.

Η μετατόπιση του 2026: Από την προσπάθεια στη φυσιολογία

Η διαχείριση της κόπωσης στην πισίνα είναι εμφανώς δύσκολη σε σύγκριση με το ποδήλατο ή το τρέξιμο. Χωρίς μετρητή ισχύος, ο ρυθμός είναι ο μόνος μας αντιπρόσωπος για την ένταση. Ενώ πολλοί κολυμβητές εξακολουθούν να βασίζονται σε παλιούς ρυθμούς "χρονομέτρησης" 1500 μέτρων, η σύγχρονη προπονητική έχει μεταφερθεί στο Μοντέλο CSS δύο αποστάσεων.

Η κύρια διαφορά είναι Ακρίβεια έναντι αντιληπτής προσπάθειας. Ένα T-Pace που βασίζεται σε μια και μόνο μεγάλη προσπάθεια συχνά συλλαμβάνει τη διανοητική σας δύναμη περισσότερο από το μεταβολικό κατώφλι σας. Το CSS, που υπολογίζεται από δύο διακριτά σημεία στην "Καμπύλη ρυθμού-διάρκειας", αποκαλύπτει το πραγματικό όριο αερόβιας λειτουργίας του κινητήρα σας.

Μετρική Σύγκριση

Ρυθμός κατωφλίου (T-Pace)

Η «Απλή» Μετρική

Το T-Pace συνήθως προέρχεται από μία μόνο μέγιστη προσπάθεια 1500m ή 1000m. Αντιπροσωπεύει την "Redline" σας για συνεχές κολύμπι.

  • Πλεονεκτήματα: Απλό στην κατανόηση. Ένα τεστ, ένας αριθμός. Ιδανικό για μεγάλα, συνεχόμενα σετ.
  • Μειονεκτήματα: Υψηλή ευαισθησία σε σφάλματα βηματοδότησης. Δεν υπολογίζει πόσο γρήγορα ξεθωριάζεις από την κούραση. Αντιμετωπίζει κάθε κολυμβητή με τον ίδιο ρυθμό 1500 μέτρων σαν να έχει το ίδιο φυσιολογικό προφίλ.

Κρίσιμη ταχύτητα κολύμβησης (CSS)

Η «Λειτουργική» Μετρική

Το CSS είναι μια μαθηματική παραγωγή που βασίζεται σε δύο αποστάσεις (συνήθως μια μέγιστη προσπάθεια 400m και 200m).

  • Πλεονεκτήματα: Καταργεί την προκατάληψη βηματοδότησης. Προβλέπει καλύτερα τη συμπεριφορά σας κατά διαστήματα. Παρέχει μια αντικειμενική «αερόβια οροφή».
  • Μειονεκτήματα: Απαιτεί δύο ξεχωριστές δοκιμές. Ελαφρώς πιο σύνθετα μαθηματικά (υπολογίζονται αυτόματα στο Swim Analytics). Απαιτεί οι δοκιμές να γίνονται με συνεχή ανάπαυση.

Γιατί το Δέλτα έχει σημασία

Το "Δέλτα" είναι η διαφορά μεταξύ του ρυθμού σας στα 400 μέτρα και του ρυθμού σας στα 200 μέτρα. Δύο κολυμβητές μπορεί να έχουν και οι δύο T-Pace 1:40/100m, αλλά τα προφίλ τους CSS μπορεί να είναι έτσι:

Κολυμβητής Α ("The Diesel")

Μικρό δέλτα μεταξύ 200m και 400m. Αυτός ο κολυμβητής έχει πολύ υψηλή αερόβια απόδοση αλλά χαμηλότερη ταχύτητα σπριντ. Το CSS τους είναι πολύ κοντά στο T-Pace τους.

Κολυμβητής Β ("The Sprinter")

Μεγάλο δέλτα. Μεγάλη ταχύτητα 200 μέτρων, αλλά μειώνεται σημαντικά στα 400 μέτρα. Το CSS τους θα είναι χαμηλότερο από αυτό που προτείνει το T-Pace. Αν προσπαθήσουν να προπονηθούν σε γενικό "T-Pace", θα παραμαγειρεύουν συνεχώς το αναερόβιο σύστημά τους.

📈 Η φόρμουλα CSS

CSS = (D1 - D2) / (T1 - T2)

Όπου D1/D2 είναι οι δύο αποστάσεις και T1/T2 οι χρόνοι σε δευτερόλεπτα. Αυτό αποκαλύπτει την «κλίση» του βιώσιμου ρυθμού σας χωρίς τον θόρυβο της αρχικής αναερόβιας έκρηξης.

Πώς να εκτελέσετε τη δοκιμή CSS (βήμα προς βήμα)

Για να υπολογίσετε το CSS σας, πρέπει να εκτελέσετε δύο μέγιστες δοκιμές σε διαφορετικές ημέρες (ή με πλήρη ανάκτηση ενδιάμεσα) σε μια τυπική ομάδα:

  1. Προθέρμανση: Εκτελέστε μια ενδελεχή, τυποποιημένη προθέρμανση (συνήθως 400m έως 600m μικτών κτυπημάτων και χτίσεων) για να προετοιμάσετε το σώμα σας.
  2. Δοκιμή 1 (μέγιστη προσπάθεια 400μ): Κολυμπήστε 400 μέτρα όσο πιο γρήγορα γίνεται. Καταγράψτε την ακριβή ώρα. Βάλτε ρυθμό για να μην καείτε στα πρώτα 100 μέτρα.
  3. Ενεργή ανάκτηση: Κολυμπήστε 100 μέτρα έως 200 μέτρα πολύ εύκολο να ξεπλύνετε το γαλακτικό οξύ, ακολουθούμενο από 5-10 λεπτά ξεκούρασης.
  4. Δοκιμή 2 (Μέγιστη προσπάθεια 200μ): Κολυμπήστε 200 μέτρα με απόλυτη μέγιστη προσπάθεια. Καταγράψτε την ακριβή ώρα.
  5. Υπολογίστε: Μετατρέψτε τους χρόνους σας σε δευτερόλεπτα και εφαρμόστε τον παραπάνω τύπο ή χρησιμοποιήστε το εργαλείο αριθμομηχανής CSS στο Swim Analytics για να λάβετε τον ρυθμό βηματοδότησης των 100 μέτρων.

Ποια πρέπει να χρησιμοποιήσετε;

Χρησιμοποιήστε το ρυθμό κατωφλίου (T-Pace) εάν:

  • Μόλις ξεκινάτε με προπόνηση με βάση το ρυθμό.
  • Εκτελείτε μόνο μεγάλες, συνεχόμενες κολυμπήσεις (Ανοιχτά νερά 3km+).
  • Δεν θέλετε να ασχοληθείτε με πολλές συνεδρίες δοκιμών.

Χρησιμοποιήστε την κρίσιμη ταχύτητα κολύμβησης (CSS) εάν:

  • Τριαθλητές & Μαραθωνοδρόμοι: Για να αποφύγετε το «κάψιμο του κεριού» πολύ νωρίς σε έναν αγώνα.
  • Διαλειμματική προπόνηση: Για να ορίσετε συγκεκριμένες ζώνες CSS + 2s ή CSS - 1s που λειτουργούν πραγματικά.
  • Παρακολούθηση προόδου: Το CSS είναι πιο ευαίσθητο στις αερόβιες βελτιώσεις από μια γενική δοκιμή 1000 μέτρων.

Συνήθεις ερωτήσεις για το ρυθμό κατωφλίου έναντι του CSS

Είναι το CSS το ίδιο με τον ρυθμό ορίου;

Όχι. Είναι σχετικές έννοιες κατωφλίου, αλλά δεν είναι το ίδιο τεστ. Ο ρυθμός κατωφλίου συνήθως προέρχεται από ένα μεγαλύτερο κολύμπι, ενώ το CSS υπολογίζεται από δύο προσπάθειες και δίνει μια πιο επαναλαμβανόμενη άγκυρα για διαλειμματική εργασία.

Τι είναι καλύτερο για τρίαθλο και βηματισμό ανοιχτού νερού;

Το CSS είναι συνήθως το καλύτερο σημείο εκκίνησης επειδή αντανακλά βιώσιμο ρυθμό χωρίς υπερβολική ανταμοιβή βραχυπρόθεσμης ταχύτητας. Ο ρυθμός κατωφλίου μπορεί ακόμα να είναι χρήσιμος για μεγάλες σταθερές κολυμπήσεις, αλλά το CSS είναι συνήθως πιο εύκολο να μετατραπεί σε στόχους βηματοδότησης και εργασία ζώνης.

Πώς μπορώ να μετακινηθώ από τον ρυθμό κατωφλίου σε ζώνες κολύμβησης που βασίζονται σε CSS;

Χρησιμοποιήστε το επόμενο τεστ 400 μέτρων και 200 μέτρων για να ρυθμίσετε το CSS και, στη συνέχεια, αγκυροβολήστε το διάστημα γύρω από αυτόν τον αριθμό. Εάν προπονείστε ήδη με T-Pace, συγκρίνετε και τους δύο αριθμούς και χρησιμοποιήστε τον οδηγό CSS συν τις ζώνες προπόνησης κολύμβησης για να φτιάξετε ξανά τα σετ σας γύρω από επαναλαμβανόμενο αερόβιο ρυθμό.

Σταματήστε να μαντεύετε τις ζώνες της πισίνας σας.
