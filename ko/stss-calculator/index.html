무료 Swimming TSS 계산기

CSS와 워크아웃 데이터를 입력해 sTSS, IF, 구간 페이스를 자동 계산

Swimming TSS (sTSS)란?

Swimming Training Stress Score (sTSS)는 워크아웃의 강도와 시간을 결합해 훈련 부하를 정량화하는 지표입니다. 사이클링 TSS에서 발전했지만, 수영의 물리적 특성(저항 증가)을 반영해 IF³를 사용합니다.

공식

sTSS = (IF³) × 시간(시간 단위) × 100

Intensity Factor(IF) = 워크아웃 속도 / CSS 속도

⚠️ CSS가 핵심

CSS가 부정확하면 IF, sTSS, CTL/ATL/TSB 모두 잘못됩니다. CSS 테스트를 정확히 수행하세요.

계산기 사용 방법

1) CSS 입력

  • 400m, 200m 기록을 입력해 CSS를 계산하거나
  • 이미 알고 있다면 CSS 페이스/속도를 직접 입력

2) 워크아웃 데이터 입력

  • 총 거리 (m)
  • 이동 시간 (분:초)
  • 휴식 시간은 제외

3) 결과 확인

  • IF (Intensity Factor)
  • sTSS (훈련 부하 점수)
  • 평균 페이스 및 존 페이스

4) 부하 관리

sTSS를 일/주 단위로 합산해 CTL/ATL/TSB를 추적합니다.

예시

입력

  • CSS: 1:33/100m
  • 총 거리: 3000m
  • 이동 시간: 55:00

출력

  • IF ≈ 0.845
  • sTSS ≈ 55
  • 평균 페이스, 존별 페이스 표시

sTSS 강도 가이드

sTSS 해석 예시
< 50 회복 30-45분 회복/드릴
50-100 보통 60-90분 유산소
100-200 강함 임계/VO₂ 세션 포함
200+ 매우 강함 장시간, 고강도 — 회복 계획 필수

자주 묻는 질문

sTSS와 TSS 차이는?

개념은 동일하지만, 수영에서는 물 저항을 반영해 IF³를 사용합니다.

휴식 시간을 포함해야 하나요?

포함하지 않습니다. 이동 시간만 사용해 실제 수영 강도를 반영합니다.

CSS를 모르면?

CSS 테스트를 먼저 수행해 정확한 기준을 설정하세요.

오픈워터에도 적용되나요?

네, CSS 기반이므로 오픈워터 수영에도 동일한 방식으로 sTSS를 계산할 수 있습니다.

sTSS로 훈련 부하를 정량화하세요

CSS와 워크아웃 데이터를 기반으로 강도와 시간을 하나의 값으로.
