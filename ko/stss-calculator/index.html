무료 수영 TSS 계산기

수영 운동의 훈련 스트레스 점수 계산 - 유일한 무료 sTSS 계산기

수영 TSS(sTSS)란 무엇인가요?

수영 훈련 스트레스 점수(sTSS)는 강도와 지속 시간을 결합하여 수영 운동의 훈련 부하를 정량화합니다. 사이클링의 TSS 방법론을 채택하여 임계 수영 속도(CSS)를 임계 페이스로 사용합니다. CSS 페이스로 1시간 운동 = 100 sTSS.

훈련 부하 해석

sTSS 계산 방법

공식

sTSS = (시간 단위 지속 시간) × (강도 계수)² × 100

여기서:

  • 강도 계수 (IF) = CSS 페이스 / 평균 운동 페이스
  • 지속 시간 = 시간 단위의 총 운동 시간
  • CSS 페이스 = CSS 테스트의 임계 페이스

계산 예시

운동 세부 정보:

  • CSS 페이스: 1:49/100m (109초)
  • 운동 시간: 60분 (1시간)
  • 평균 페이스: 2:05/100m (125초)

1단계: 강도 계수 계산

IF = CSS 페이스 / 운동 페이스
IF = 109 / 125
IF = 0.872

2단계: sTSS 계산

sTSS = 1.0 시간 × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

해석: 쉬운 페이스(CSS보다 느림)로 60분간 운동하여 76 sTSS를 생성했습니다. 이는 유산소 기초를 다지기에 적합한 중간 정도의 훈련 부하입니다.

sTSS 값 이해하기

sTSS 범위 훈련 부하 회복 시간 운동 예시
< 50 낮음 당일 쉬운 30분 수영, 기술 훈련
50-100 중간 1일 60분 지구력, 꾸준한 페이스
100-200 높음 1-2일 90분 임계 세트, 레이스 페이스 인터벌
200-300 매우 높음 2-3일 2시간 고강도 훈련, 다중 임계 블록
> 300 극한 3일 이상 장거리 레이스(2시간 이상), 초장거리 지구력

주간 sTSS 가이드라인

목표 주간 sTSS는 훈련 수준과 목표에 따라 다릅니다:

레크리에이션 수영 선수

주간 sTSS: 150-300

주 2-3회 운동, 각 50-100 sTSS. 기술과 유산소 기초 다지기에 집중.

피트니스 수영 선수 / 철인 3종 경기 선수

주간 sTSS: 300-500

주 3-4회 운동, 각 75-125 sTSS. 유산소 지구력과 임계 훈련의 혼합.

경쟁 마스터스 수영 선수

주간 sTSS: 500-800

주 4-6회 운동, 각 80-150 sTSS. 주기화가 포함된 체계적인 훈련.

엘리트 / 대학 수영 선수

주간 sTSS: 800-1200+

주 8-12회 운동, 하루 2회 훈련. 회복 관리가 중요한 고용량 훈련.

⚠️ 중요 참고 사항

  • 정확한 CSS 필요: 정확한 sTSS를 위해서는 CSS가 최신 상태(6-8주 이내 테스트)여야 합니다.
  • 간소화된 계산: 이 계산기는 평균 페이스를 사용합니다. 고급 sTSS는 인터벌 구조를 고려한 정규화 등급 페이스(NGP)를 사용합니다.
  • 기술 훈련용 아님: sTSS는 신체적 훈련 스트레스만 측정하며 기술 개발은 측정하지 않습니다.
  • 개인차: 동일한 sTSS라도 수영 선수마다 다르게 느껴질 수 있습니다. 회복 상태에 따라 가이드라인을 조정하세요.

sTSS가 중요한 이유

훈련 스트레스 점수는 다음의 기초가 됩니다:

  • CTL (만성 훈련 부하): 체력 수준 - 일일 sTSS의 42일 지수 가중 이동 평균
  • ATL (급성 훈련 부하): 피로도 - 일일 sTSS의 7일 지수 가중 이동 평균
  • TSB (훈련 스트레스 균형): 컨디션 - TSB = CTL - ATL (양수 = 신선함, 음수 = 피로함)
  • 주기화: 목표 CTL 진행을 사용하여 훈련 단계(기초, 빌드, 피크, 테이퍼) 계획
  • 회복 관리: TSB를 기반으로 언제 밀어붙이고 언제 휴식할지 파악

프로 팁: CTL 추적하기

스프레드시트나 훈련 일지에 매일 sTSS를 기록하세요. 매주 42일 평균(CTL)을 계산하세요. 기초 단계에서는 주당 5-10 CTL 포인트 증가를 목표로 하세요. 테이퍼 기간(레이스 1-2주 전)에는 CTL을 유지하거나 약간 줄이세요.

자주 묻는 질문

수영 TSS(sTSS)란 무엇인가요?

수영 훈련 스트레스 점수(sTSS)는 강도와 지속 시간을 결합하여 수영 운동의 훈련 부하를 정량화하는 지표입니다. 사이클링의 TSS 방법론을 채택하여 임계 수영 속도(CSS)를 임계 페이스로 사용합니다. CSS 페이스로 1시간 운동하면 100 sTSS가 됩니다.

sTSS는 어떻게 계산하나요?

위의 계산기에 CSS 페이스(CSS 테스트 결과), 총 운동 시간, 운동 중 평균 페이스를 입력하여 사용하세요. 공식은 다음과 같습니다: sTSS = 지속 시간(시간) × 강도 계수² × 100, 여기서 강도 계수 = CSS 페이스 / 평균 운동 페이스.

sTSS를 계산하려면 CSS가 필요한가요?

네, 강도 계수를 계산하려면 임계 수영 속도(CSS)가 필요하며, 이는 sTSS 계산에 필수적입니다. CSS는 임계 페이스를 나타내며 6-8주마다 테스트해야 합니다. CSS 계산기를 사용하여 CSS를 찾을 수 있습니다.

한 번의 운동에 적절한 sTSS 점수는 얼마인가요?

운동 강도에 따라 다릅니다: 쉬운 운동은 일반적으로 50 sTSS 미만, 중간 강도 운동은 50-100 sTSS, 힘든 운동은 100-200 sTSS, 매우 힘든 운동은 200 sTSS 이상입니다. 적절한 점수는 훈련 목표와 현재 체력 수준에 따라 다릅니다.

일주일에 얼마나 많은 sTSS를 해야 하나요?

주간 sTSS 목표는 수준에 따라 다릅니다: 레크리에이션 수영 선수: 150-300, 피트니스 수영 선수/철인 3종 경기 선수: 300-500, 경쟁 마스터스: 500-800, 엘리트/대학 선수: 800-1200+. 오버트레이닝을 피하기 위해 보수적으로 시작하고 점진적으로 늘리세요.

수영 TSS는 사이클링 TSS와 같나요?

개념과 공식은 동일하지만 sTSS는 수영에 맞게 조정되었습니다. 사이클링 TSS처럼 파워(FTP)를 사용하는 대신 sTSS는 CSS를 임계값으로 하는 페이스를 사용합니다. 둘 다 지속 시간 × 강도 계수² × 100을 사용하여 훈련 부하를 측정합니다.

모든 수영 영법에 sTSS를 사용할 수 있나요?

예, 하지만 CSS는 영법별로 특정해야 합니다. 대부분의 수영 선수는 자유형 CSS를 사용합니다. 가장 일반적으로 훈련하는 영법이기 때문입니다. 주로 다른 영법으로 훈련하는 경우 해당 영법으로 CSS 테스트를 수행하고 해당 페이스를 sTSS 계산에 사용하세요.

sTSS와 CTL/ATL/TSB의 차이점은 무엇인가요?

sTSS는 단일 운동의 훈련 부하를 측정합니다. CTL(만성 훈련 부하)은 장기 체력, ATL(급성 훈련 부하)은 최근 피로도, TSB(훈련 스트레스 균형)는 신선도입니다. 이러한 지표는 시간 경과에 따른 sTSS 값을 사용하여 훈련 상태를 추적합니다. 훈련 부하 가이드에서 자세히 알아보세요.
