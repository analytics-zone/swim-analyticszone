SwimAnalytics vs 다른 수영 앱 - 기능 비교

네이티브 수영 특화 지표, 자동 sTSS 계산, 합리적 가격

왜 수영은 특화된 분석이 필요한가?

일반 멀티스포츠 플랫폼은 수영에 대한 네이티브 지원이 부족합니다. SwimAnalytics는 수영 전용 지표(CSS, sTSS, SWOLF 등)와 워치-네이티브 분석으로 정확도와 편의성을 제공합니다.

네이티브 수영 지표

CSS 기반 페이스 존, sTSS, SR/DPS, SWOLF를 수영에 맞게 제공합니다.

자동 sTSS

Apple Watch 워크아웃을 자동 수집, sTSS/CTL/ATL/TSB를 계산해 수동 로그 없이 훈련 부하를 추적합니다.

프라이버시 우선

100% 로컬 처리. 서버 업로드나 계정 없이 데이터 소유권을 유지합니다.

주요 비교 요약

기능 SwimAnalytics TrainingPeaks Garmin FORM

핵심 차별점

1. 자동 sTSS (수영 전용)

  • CSS + 페이스로 Intensity Factor 계산, IF³ 적용
  • 모든 워크아웃에 sTSS, CTL/ATL/TSB 자동 업데이트
  • 수동 로그 없이 훈련 부하 추적

2. 수영에 맞춘 존/페이스

  • CSS 테스트로 7개 존 생성
  • 수영 페이스 특성(저항 지수 성장) 반영
  • 구간별 페이스/시간 자동 표시

3. 프라이버시 & 간결성

  • 계정/로그인 없이 즉시 사용
  • 서버 업로드 없음 → 데이터는 기기에만
  • 간결한 워치-네이티브 UX

4. 합리적 비용

  • 월 $20 TrainingPeaks Premium 대비 낮은 비용
  • 고가 장비 없이 수영 특화 분석 제공
  • 투명한 가격, 부가 비용 없음

FAQ

SwimAnalytics는 Garmin과 함께 사용할 수 있나요?

SwimAnalytics는 Apple Health와 네이티브로 동기화됩니다. Garmin 워크아웃을 Health로 동기화하면 가져올 수 있습니다. 워치-내 sTSS 계산은 Apple Watch에서 지원됩니다.

수동으로 워크아웃을 입력해야 하나요?

아니요. Apple Watch로 수영 워크아웃을 기록하면 자동으로 불러옵니다.

SWOLF/스트로크 분석은 어떻게 다릅니까?

수영 전용 SR/DPS/SWOLF 분석을 제공하며, CSS와 결합해 페이스 존과 sTSS에 반영합니다.

데이터는 어디에 저장되나요?

모든 데이터는 기기 로컬에 저장되며 서버로 전송되지 않습니다.
