FAQ

Perguntas Frequentes

Como o Swim Analytics obtém meus dados de natação?

O Swim Analytics sincroniza com o Apple Health para importar treinos de natação registrados por qualquer dispositivo ou app compatível. Isso inclui smartwatches, rastreadores de fitness e entradas manuais. O app processa esses dados localmente para calcular métricas avançadas.

O que é o teste CSS e como eu faço?

CSS (Critical Swim Speed) é um protocolo científico usando 2 nados de esforço máximo: 400m e 200m com 10-20 minutos de descanso entre eles. O app calcula seu limiar aeróbico a partir desses tempos e ajusta automaticamente todas as zonas de treino. Repita a cada 6-8 semanas para acompanhar o progresso.

Meus dados são enviados para a nuvem?

Não. O Swim Analytics processa todos os dados localmente no seu iPhone. Não há servidores externos, contas na nuvem ou transferências de dados. Você controla as exportações: gere arquivos JSON, CSV, HTML ou PDF e compartilhe como quiser.

Posso usar o Swim Analytics para natação em águas abertas?

Sim. O Swim Analytics funciona com qualquer treino de natação no Apple Health, incluindo águas abertas. O app se adapta às métricas disponíveis, seja você na piscina ou em águas abertas, fornecendo análises relevantes para cada ambiente.

Qual a diferença entre os planos mensal e anual?

Ambos os planos oferecem recursos idênticos: todas as métricas, zonas ilimitadas, comparações temporais, múltiplas exportações e atualizações gratuitas. A única diferença é o preço: o anual economiza 18% (equivalente a €3,25/mês vs €3,99/mês).

Posso cancelar minha assinatura a qualquer momento?

Sim. As assinaturas são gerenciadas pela App Store, então você pode cancelar a qualquer momento em Ajustes → [Seu Nome] → Assinaturas. Se você cancelar, manterá o acesso até o final do seu período de cobrança atual.
