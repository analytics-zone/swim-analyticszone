Iyara We to Lominu (CSS) vs. Pace Ibalẹ: Ewo Ni Pataki?

Fun awọn ọdun, "Iwọn Iwọn" (T-Pace) jẹ idiwọn goolu. Ni ọdun 2026, iyipada si Iyara Swim Critical (CSS) n ṣe atuntu bi awọn elere idaraya ṣe n yara awọn akoko wọn ati asọtẹlẹ rirẹ ọjọ-ije.

🎯 Idahun Yara naa

T-Pace (Iwọn Ilọsiwaju) jẹ iṣiro gbogbogbo ti iyara ti o le ṣetọju fun wakati kan, nigbagbogbo lo fun ṣeto awọn agbegbe ikẹkọ gbooro. O ti wa ni a "wulo" sugbon kere granular metric.

CSS (Iyara we to ṣe pataki) jẹ ala “iṣẹ-ṣiṣe” ti a gba ni mathematiki lati awọn ijinna meji ti o yatọ (fun apẹẹrẹ, 400m ati 200m). O ti wa ni awọn tumq si Pace ti o le fowosowopo lai kọlu awọn "Odi." Ti o ba jẹ oluwẹwẹ ifarada tabi ẹlẹrin-mẹta, CSS rẹ jẹ nọmba pataki julọ ninu eto ikẹkọ rẹ.

Iyipada 2026: Lati Igbiyanju si Ẹkọ-ara

Ṣiṣakoso rirẹ ninu adagun-odo jẹ olokiki ti o nira ni akawe si gigun kẹkẹ tabi ṣiṣe. Laisi mita agbara, iyara jẹ aṣoju nikan wa fun kikankikan. Lakoko ti ọpọlọpọ awọn odo tun gbekele ile-iwe atijọ 1500m “idanwo akoko-akoko”, ikẹkọ ode oni ti lọ si Meji-ijinna CSS Awoṣe.

Iyatọ akọkọ ni Konge vs akitiyan ti a ti fiyesi. T-Pace ti o da lori igbiyanju gigun kan nigbagbogbo n gba grit ọpọlọ rẹ diẹ sii ju ala ti iṣelọpọ rẹ. CSS, ti a ṣe iṣiro lati awọn aaye ọtọtọ meji lori “Pace-Duration Curve” rẹ ṣe afihan opin aerobic gangan ti ẹrọ rẹ.

Ifiwera Metiriki

Iyara Ipele (T-Pace)

Metiriki "rọrun".

T-Pace ti wa ni maa yo lati kan nikan 1500m tabi 1000m o pọju akitiyan . O duro rẹ "Redline" fun a lemọlemọfún we.

  • Aleebu: Rọrun lati ni oye. Idanwo kan, nọmba kan. Nla fun gun, lemọlemọfún tosaaju.
  • Kosi: Ni ifarabalẹ ga si awọn aṣiṣe pacing. Ko ṣe akọọlẹ fun bi o ṣe yara ti o rọ labẹ rirẹ. Ṣe itọju gbogbo oluwẹwẹ pẹlu iyara 1500m kanna bi nini profaili ti ẹkọ iṣe-ara kanna.

Iyara we to ṣe pataki (CSS)

Metiriki "Iṣẹ-ṣiṣe".

CSS jẹ itọsẹ mathematiki ti o da lori awọn ijinna meji (paapaa 400m ati igbiyanju 200m ti o pọju).

  • Aleebu: Yọ aibikita pacing kuro. Dara julọ ṣe asọtẹlẹ ihuwasi rẹ ni awọn aaye arin. Pese ohun idi "aerobic aja."
  • Kosi: Nilo meji lọtọ igbeyewo. Iṣiro-diẹ idiju diẹ sii (iṣiro-laifọwọyi ni Awọn atupale we). Nilo awọn idanwo lati ṣe pẹlu isinmi deede.

Kí nìdí Delta ọrọ

“Delta” jẹ iyatọ laarin iyara 400m rẹ ati iyara 200m rẹ. Awọn oluwẹwẹ meji le mejeeji ni T-Pace ti 1:40/100m, ṣugbọn awọn profaili CSS wọn le dabi eyi:

Swimmer A ("The Diesel")

Delta kekere laarin 200m ati 400m. Oluwẹwẹ yii ni ṣiṣe aerobic ti o ga pupọ ṣugbọn iyara isunmi kekere. CSS wọn wa nitosi T-Pace wọn.

Swimmer B ("The Sprinter")

Delta nla. Iyara 200m nla ṣugbọn o lọ silẹ ni pataki lori 400m. CSS wọn yoo kere ju T-Pace wọn ni imọran. Ti wọn ba gbiyanju lati ṣe ikẹkọ ni “T-Pace” jeneriki, wọn yoo maa se eto anaerobic wọn nigbagbogbo.

📈 Ilana CSS

CSS = (D1 - D2) / (T1 - T2)

Nibo ni D1/D2 jẹ awọn ijinna meji ati T1/T2 jẹ awọn akoko ni iṣẹju-aaya. Eyi ṣe afihan “itẹ” ti iyara alagbero rẹ laisi ariwo ti iṣẹ abẹ anaerobic akọkọ rẹ.

Bii o ṣe le Ṣe Idanwo CSS (Igbese-nipasẹ-Igbese)

Lati ṣe iṣiro CSS rẹ, o nilo lati ṣe awọn idanwo ti o pọju meji ni awọn ọjọ oriṣiriṣi (tabi pẹlu imularada ni kikun laarin) ni adagun-odo kan:

  1. Igbagbo: Ṣe igbona ni kikun, iwọnwọn (nigbagbogbo 400m si 600m ti awọn ikọlu adalu ati kọ) lati ṣeto ara rẹ.
  2. Idanwo 1 (Igbiyanju 400m ti o pọju): We 400m ni yarayara bi o ti ṣee. Ṣe igbasilẹ akoko gangan. Pace ara rẹ ki o ma ba sun ni 100m akọkọ.
  3. Imularada lọwọ: We 100m si 200m rọrun pupọ lati yọ jade lactic acid, atẹle nipa iṣẹju 5-10 ti isinmi.
  4. Idanwo 2 (Igbiyanju ti o pọju 200m): We 200m ni idi ti o pọju akitiyan. Ṣe igbasilẹ akoko gangan.
  5. Ṣe iṣiro: Yipada awọn akoko rẹ si iṣẹju-aaya ki o lo agbekalẹ ti o wa loke, tabi lo ohun elo iṣiro CSS ni Awọn itupalẹ Swim lati gba iyara 100m rẹ.

Ewo Ni O yẹ O Lo?

Lo Pace Ibẹrẹ (T-Pace) ti o ba jẹ:

  • O kan bẹrẹ pẹlu ikẹkọ ti o da lori iyara.
  • Iwọ nikan ṣe gigun, awọn iwẹ lemọlemọfún (Omi Ṣii 3km+).
  • O ko fẹ lati ṣe pẹlu awọn akoko idanwo pupọ.

Lo Iyara we Lominu (CSS) ti o ba jẹ:

  • Mẹta-mẹta & Marathon Swimmers: Lati yago fun “sisun abẹla” ni kutukutu ere-ije kan.
  • Ikẹkọ aarin: Lati ṣeto CSS + 2s kan pato tabi CSS - awọn agbegbe 1s ti o ṣiṣẹ gaan.
  • Ilọsiwaju titele: CSS ṣe akiyesi diẹ sii si awọn ilọsiwaju aerobic ju idanwo jeneriki 1000m lọ.

Ala Pace vs CSS FAQ

Njẹ CSS jẹ kanna bi iyara ala bi?

Rara. Wọn jẹ awọn imọran ala-ilẹ ti o ni ibatan, ṣugbọn wọn kii ṣe idanwo kanna. Pace ala maa n wa lati odo gigun kan, lakoko ti CSS ṣe iṣiro lati awọn akitiyan meji ati pe o funni ni oran atunwi diẹ sii fun iṣẹ aarin.

Ewo ni o dara julọ fun triathlon ati ṣiṣan omi-ìmọ?

CSS maa n jẹ aaye ibẹrẹ ti o dara julọ nitori pe o ṣe afihan iyara alagbero laisi iyara akoko kukuru ti o ni ere pupọ. Iyara ẹnu-ọna tun le wulo fun awọn wiwẹ gigun, ṣugbọn CSS nigbagbogbo rọrun lati yipada si awọn ibi-afẹde pacing ati iṣẹ agbegbe.

Bawo ni MO ṣe gbe lati iyara iloro si awọn agbegbe iwẹ orisun CSS?

Lo idanwo 400m ati 200m atẹle rẹ lati ṣeto CSS, lẹhinna iṣẹ aarin igbaduro ni ayika nọmba yẹn. Ti o ba ti ṣe ikẹkọ tẹlẹ nipasẹ T-Pace, ṣe afiwe awọn nọmba mejeeji ki o lo itọsọna CSS pẹlu awọn agbegbe ikẹkọ wiwẹ lati tun awọn eto rẹ ṣe ni ayika iyara aerobic ti o le tun ṣe.

Duro lafaimo awọn agbegbe adagun-odo rẹ.

Awọn atupale wewe ṣe iranlọwọ fun ọ lati yi CSS pada si awọn agbegbe adagun-odo ti o wulo, fifẹ aarin, ati wiwo ti o han gbangba ti aja aerobic rẹ ni akoko pupọ.
