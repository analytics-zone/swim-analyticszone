Ẹrọ Ìṣirò sTSS Wíwẹ́ Ọ̀fẹ́

Ṣe ìṣirò Ẹru Ìkẹ́kọ̀ọ́ Wíwẹ́ fún àwọn adàṣe - Ẹrọ ìṣirò sTSS ọ̀fẹ́ nìkan

Kí ni sTSS Wíwẹ́?

Ẹru Ìkẹ́kọ̀ọ́ Wíwẹ́ (sTSS) ṣe ìwọ̀n ẹru ìkẹ́kọ̀ọ́ fún adàṣe wíwẹ́ nípa apapọ̀ intensity àti iye àkókò. Ó ni àtúnṣe látara fọ́múlà ìkẹ́kẹ̀ TSS, nípa lílo Iyara Wíwẹ́ Pàtàkì (CSS) rẹ gẹ́gẹ́ bí iyara ìhámọ́. Adàṣe wákàtí 1 ní iyara CSS = 100 sTSS.

Ẹrọ Ìṣirò sTSS Ọ̀fẹ́

Ṣe ìṣirò ẹru ìkẹ́kọ̀ọ́ fún èyíkéyìí adàṣe wíwẹ́. O nílò iyara CSS rẹ.

Iyara ìhámọ́ rẹ látara ìdánwò CSS (fún àpẹrẹ, 1:49)
Lapapọ akoko adaṣe pẹlu isinmi (iṣẹju 1-300)
Apapọ iyara rẹ lakoko adaṣe (fun apẹẹrẹ, 2:05)

Itumọ Fifuye Ikẹkọ

Bawo ni sTSS ṣe Iṣiro

Agbekalẹ

sTSS = (Iye akoko ni awọn wakati) × (Okunfa Kikankikan)² × 100

Nibo:

  • Okunfa Kikankikan (IF) = Iyara CSS / Apapọ Iyara Adaṣe
  • Iye akoko = Lapapọ akoko adaṣe ni awọn wakati
  • Iyara CSS = Iyara ala rẹ lati idanwo CSS

Apẹẹrẹ Ti a Ṣiṣẹ

Awọn alaye Adaṣe:

  • Iyara CSS: 1:49/100m (aaya 109)
  • Iye Akoko Adaṣe: iṣẹju 60 (wakati 1)
  • Apapọ Iyara: 2:05/100m (aaya 125)

Igbesẹ 1: Ṣe iṣiro Okunfa Kikankikan

IF = Iyara CSS / Iyara Adaṣe
IF = 109 / 125
IF = 0.872

Igbesẹ 2: Ṣe iṣiro sTSS

sTSS = wakati 1.0 × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

Itumọ: Adaṣe iṣẹju 60 yii ni iyara irọrun (lọra ju CSS lọ) ṣe ipilẹṣẹ 76 sTSS - fifuye ikẹkọ iwọntunwọnsi ti o dara fun kikọ ipilẹ aerobic.

Loye Awọn iye sTSS

Ibiti sTSS Fifuye Ikẹkọ Akoko Imularada Apẹẹrẹ Adaṣe
< 50 Kekere Ọjọ kanna Odo irọrun 30-min, awọn adaṣe ilana
50-100 Iwọntunwọnsi Ọjọ 1 Ifarada 60-min, iyara iduroṣinṣin
100-200 Giga Ọjọ 1-2 Awọn eto ala 90-min, awọn aaye arin iyara ere-ije
200-300 Giga Pupọ Ọjọ 2-3 Ikẹkọ lile wakati 2, awọn bulọọki ala pupọ
> 300 O pọju Ọjọ 3+ Ere-ije gigun (> wakati 2), ifarada ultra

Awọn Itọsọna sTSS Ọsẹ

Ifojusi sTSS ọsẹ da lori ipele ikẹkọ rẹ ati awọn ibi-afẹde:

Awọn oluwẹwẹ Ere idaraya

sTSS Ọsẹ: 150-300

Awọn adaṣe 2-3 fun ọsẹ kan, 50-100 sTSS kọọkan. Fojusi lori ilana ati kikọ ipilẹ aerobic.

Awọn oluwẹwẹ Amọdaju / Triathletes

sTSS Ọsẹ: 300-500

Awọn adaṣe 3-4 fun ọsẹ kan, 75-125 sTSS kọọkan. Apapo ifarada aerobic ati iṣẹ ala.

Awọn oluwẹwẹ Masters Idije

sTSS Ọsẹ: 500-800

Awọn adaṣe 4-6 fun ọsẹ kan, 80-150 sTSS kọọkan. Ikẹkọ ti a ṣeto pẹlu akoko.

Gbajumo / Awọn oluwẹwẹ Kọlẹji

sTSS Ọsẹ: 800-1200+

Awọn adaṣe 8-12 fun ọsẹ kan, awọn ọjọ meji. Iwọn didun giga pẹlu iṣakoso imularada ṣe pataki.

⚠️ Awọn Akọsilẹ Pataki

  • Nilo CSS deede: CSS rẹ gbọdọ jẹ lọwọlọwọ (idanwo laarin awọn ọsẹ 6-8) fun sTSS deede.
  • Iṣiro irọrun: Ẹrọ iṣiro yii nlo iyara apapọ. sTSS ti ilọsiwaju nlo Iyara Ipele Deede (NGP) eyiti o ṣe akọọlẹ fun eto aarin.
  • Kii ṣe fun iṣẹ ilana: sTSS nikan ṣe iwọn wahala ikẹkọ ti ara, kii ṣe idagbasoke ọgbọn.
  • Iyatọ ẹni kọọkan: sTSS kanna ni rilara oriṣiriṣi fun awọn oluwẹwẹ oriṣiriṣi. Ṣatunṣe awọn itọnisọna ti o da lori imularada rẹ.

Kini idi ti sTSS Ṣe Pataki

Dimegilio Wahala Ikẹkọ jẹ ipilẹ fun:

  • CTL (Fifuye Ikẹkọ Onibaje): Ipele amọdaju rẹ - iwọn iwọn iwọn 42-ọjọ ti sTSS ojoojumọ
  • ATL (Fifuye Ikẹkọ Nla): Arẹ rẹ - iwọn iwọn iwọn 7-ọjọ ti sTSS ojoojumọ
  • TSB (Iwontunwonsi Wahala Ikẹkọ): Fọọmu rẹ - TSB = CTL - ATL (rere = titun, odi = rirẹ)
  • Akoko: Gbero awọn ipele ikẹkọ (ipilẹ, kọ, tente oke, taper) ni lilo awọn ilọsiwaju CTL ibi-afẹde
  • Iṣakoso Imularada: Mọ igba lati Titari ati igba lati sinmi da lori TSB

Imọran Pro: Tọpinpin CTL Rẹ

Gba sTSS ojoojumọ ninu iwe kaunti tabi log ikẹkọ. Ṣe iṣiro apapọ ọjọ-42 rẹ (CTL) ni ọsẹ kan. Fojusi ilosoke awọn aaye CTL 5-10 fun ọsẹ kan lakoko kikọ ipilẹ. Ṣetọju tabi dinku CTL diẹ lakoko taper (awọn ọsẹ 1-2 ṣaaju ere-ije).

Awọn ibeere ti a beere nigbagbogbo

Kini TSS Odo (sTSS)?

Dimegilio Wahala Ikẹkọ Odo (sTSS) jẹ metiriki kan ti o ṣe iwọn fifuye ikẹkọ ti adaṣe odo nipa apapọ mejeeji kikankikan ati iye akoko. O ti ni ibamu lati ilana TSS gigun kẹkẹ, ni lilo Iyara Odo Pataki rẹ (CSS) bi iyara ala. Adaṣe ti wakati 1 ni iyara CSS dọgba 100 sTSS.

Bawo ni MO ṣe ṣe iṣiro sTSS mi?

Lo ẹrọ iṣiro ti o wa loke nipa titẹ iyara CSS rẹ (lati idanwo CSS), apapọ iye akoko adaṣe, ati iyara apapọ lakoko adaṣe. Agbekalẹ jẹ: sTSS = Iye akoko (wakati) × Okunfa Kikankikan² × 100, nibiti Okunfa Kikankikan = Iyara CSS / Apapọ Iyara Adaṣe.

Ṣe Mo nilo CSS lati ṣe iṣiro sTSS?

Bẹẹni, Iyara Odo Pataki rẹ (CSS) nilo lati ṣe iṣiro Okunfa Kikankikan, eyiti o ṣe pataki fun iṣiro sTSS. CSS ṣe aṣoju iyara ala rẹ ati pe o yẹ ki o ni idanwo ni gbogbo ọsẹ 6-8. O le wa CSS rẹ nipa lilo ẹrọ iṣiro CSS wa.

Kini Dimegilio sTSS ti o dara fun adaṣe kan?

O da lori kikankikan adaṣe: Awọn adaṣe irọrun ni igbagbogbo Dimegilio ni isalẹ 50 sTSS, awọn adaṣe iwọntunwọnsi 50-100 sTSS, awọn adaṣe lile 100-200 sTSS, ati awọn adaṣe lile pupọ ju 200 sTSS. Dimegilio ti o yẹ da lori awọn ibi-afẹde ikẹkọ rẹ ati ipele amọdaju lọwọlọwọ.

Elo sTSS ni MO yẹ ki n ṣe fun ọsẹ kan?

Awọn ibi-afẹde sTSS ọsẹ yatọ nipasẹ ipele: Awọn oluwẹwẹ ere idaraya: 150-300, Awọn oluwẹwẹ Amọdaju/Triathletes: 300-500, Awọn Masters Idije: 500-800, Gbajumo/Kọlẹji: 800-1200+. Bẹrẹ ni ilodisi ati pọ si diẹdiẹ lati yago fun ikẹkọ pupọ.

Ṣe TSS wíwé jẹ́ bákan náà bí TSS kẹ̀kẹ́?

Èrò àti àgbékalẹ̀ jẹ́ bákan náà, ṣùgbọ́n sTSS ti ṣe àtúnṣe fún wíwé. Dípò lílo agbára (FTP) bí TSS kẹ̀kẹ́, sTSS ń lo ìyára pẹ̀lú CSS gẹ́gẹ́ bí ààlà. Àwọn méjèèjì ń wọn ẹrù ìkọ́ni nípa lílo Àkókò × Ìpele Líle² × 100.

Ṣe mo lè lo sTSS fún gbogbo ọ̀nà wíwé?

Bẹ́ẹ̀ni, ṣùgbọ́n CSS rẹ yẹ kí ó jẹ́ pàtó fún ọ̀nà. Ọ̀pọ̀ àwọn olùwé ń lo CSS ọ̀nà òmìnira nítorí pé ó jẹ́ ọ̀nà tí wọ́n ń kọ́ jùlọ. Tí o bá ń kọ́ni ní pàtàkì ní ọ̀nà mìíràn, ṣe ìdánwò CSS ní ọ̀nà yẹn kí o sì lo ìyára yẹn fún ìṣirò sTSS.

Kí ni ìyàtọ̀ láàárín sTSS àti CTL/ATL/TSB?

sTSS ń wọn ẹrù ìkọ́ni ti iṣẹ́-ara kan. CTL (Chronic Training Load) ni ìdárayá rẹ fún ìgbà pípẹ́, ATL (Acute Training Load) ni àárẹ̀ rẹ láìpẹ́, àti TSB (Training Stress Balance) ni títún rẹ. Àwọn òṣùwọn wọ̀nyí ń lo àwọn iye sTSS ní àkókò láti tọpinpin ipò ìkọ́ni rẹ. Kọ́ síi nínú ìtọ́sọ́nà Ẹrù Ìkọ́ni wa.
