အခမဲ့ရေကူး TSS ဂဏန်းတွက်စက်

ရေကူးလေ့ကျင့်ခန်းများအတွက် လေ့ကျင့်ရေး စိတ်ဖိစီးမှုရမှတ်ကို တွက်ချက်ပါ - တစ်ခုတည်းသော အခမဲ့ sTSS ဂဏန်းတွက်စက်

Swimming TSS (sTSS) ဆိုတာဘာလဲ။

ရေကူးလေ့ကျင့်ရေး စိတ်ဖိစီးမှုရမှတ် (sTSS)ပြင်းထန်မှုနှင့် ကြာချိန်တို့ကို ပေါင်းစပ်ခြင်းဖြင့် ရေကူးလေ့ကျင့်ခန်း၏ လေ့ကျင့်ရေးဝန်ကို တွက်ချက်သည်။ စက်ဘီးစီးခြင်း၏ TSS နည်းစနစ်မှ လိုက်လျောညီထွေဖြစ်ပြီး သင်၏အရေးပါသော ရေကူးအမြန်နှုန်း (CSS) ကို သတ်မှတ်နှုန်းအဖြစ် အသုံးပြုသည်။ CSS ခြေလှမ်း = 100 sTSS တွင် ၁ နာရီ လေ့ကျင့်ခန်း။

အခမဲ့ sTSS ဂဏန်းတွက်စက်

ရေကူးလေ့ကျင့်ခန်းအတွက် လေ့ကျင့်ရေးစိတ်ဖိစီးမှုကို တွက်ချက်ပါ။ သင်၏ CSS အရှိန်အဟုန် လိုအပ်သည်။

CSS စမ်းသပ်မှုမှ သင့်အဆင့်သတ်မှတ်ချက် (ဥပမာ၊ 1:49)
အနားယူချိန်အပါအဝင် စုစုပေါင်း လေ့ကျင့်ခန်းအချိန် (၁-၃၀၀ မိနစ်)
လေ့ကျင့်ခန်းလုပ်နေစဉ် သင်၏ပျမ်းမျှနှုန်း (ဥပမာ၊ 2:05)

ဖော်မြူလာ

sTSS = (နာရီအတွင်း ကြာချိန်) × (ပြင်းထန်မှုအချက်)³ × 100

  • ပြင်းထန်မှုအချက် (IF)= CSS Pace/Average Workout Pace
  • သင်တန်းကာလ= စုစုပေါင်းလေ့ကျင့်ခန်းအချိန်နာရီ
  • CSS Pace= CSS စမ်းသပ်မှုမှ သင့်အဆင့်သတ်မှတ်ချက်

စကားပြန်-ဤမိနစ် 60 လေ့ကျင့်ခန်းသည် လွယ်ကူသောအရှိန်အဟုန်ဖြင့် (CSS ထက်နှေးသည်) သည် 76 sTSS ကိုထုတ်ပေးသည် - အေရိုးဗစ်အခြေခံအဆောက်အအုံအတွက် သင့်လျော်သောအလယ်အလတ်လေ့ကျင့်ရေးဝန်တစ်ခုဖြစ်သည်။

sTSS တန်ဖိုးများကို နားလည်ခြင်း။

sTSS အတိုင်းအတာလေ့ကျင့်ရေးဝန်ပြန်လည်ရယူချိန်နမူနာလေ့ကျင့်ခန်း
< 50နိမ့်သည်။ထိုနေ့မိနစ် 30 လွယ်ကူသော ရေကူးနည်း၊ နည်းပညာလေ့ကျင့်ခန်းများ
၅၀-၁၀၀တော်ရုံတန်ရုံ၁ ရက်မိနစ် 60 ခံနိုင်ရည်၊ တည်ငြိမ်သောအရှိန်အဟုန်
၁၀၀-၂၀၀မြင့်သည်။1-2 ရက်မိနစ် 90 ကန့်သတ်သတ်မှတ်မှုများ၊ ပြိုင်ပွဲအရှိန်အဟုန်ကြားကာလများ
၂၀၀-၃၀၀အရမ်းမြင့်တယ်။2-3 ရက်၂ နာရီကြာ ပြင်းထန်သော လေ့ကျင့်မှု၊ တံခါးခုံတုံးများစွာ
> ၃၀၀အလွန်အမင်း၃+ ရက်ရှည်လျားသောပြိုင်ပွဲ (> 2 နာရီ)၊ အလွန်ခံနိုင်ရည်ရှိသည်။

အပတ်စဉ် sTSS လမ်းညွှန်ချက်များ

အပတ်စဉ်ပစ်မှတ် sTSS သည် သင်၏လေ့ကျင့်ရေးအဆင့်နှင့် ပန်းတိုင်များပေါ်တွင်မူတည်သည်-

အပန်းဖြေရေကူးသမားများ

အပတ်စဉ် sTSS-၁၅၀-၃၀၀

တစ်ပတ်လျှင် 2-3 လေ့ကျင့်ခန်း၊ 50-100 sTSS တစ်ခုစီ။ နည်းပညာနှင့် အေရိုးဗစ်အခြေခံတည်ဆောက်ခြင်းတို့ကို အာရုံစိုက်ပါ။

ကြံ့ခိုင်ရေး ရေကူးသမားများ / Triathletes

အပတ်စဉ် sTSS-၃၀၀-၅၀၀

တစ်ပတ်လျှင် 3-4 လေ့ကျင့်ခန်း၊ 75-125 sTSS တစ်ခုစီ။ အေရိုးဗစ်ခံနိုင်ရည်နှင့် တံခါးခုံအလုပ်တို့ ရောနှောထားသည်။

အပြိုင်အဆိုင် Masters ရေကူးသမားများ

အပတ်စဉ် sTSS-၅၀၀-၈၀၀

Elite / Collegiate ရေကူးသမားများ

အပတ်စဉ် sTSS-800-1200+

တစ်ပတ်လျှင် 8-12 လေ့ကျင့်ခန်း၊ နှစ်ရက်။ ပြန်လည်ရယူခြင်း စီမံခန့်ခွဲမှုနှင့်အတူ အရေးကြီးသော ပမာဏမြင့်မားသည်။

⚠️ အရေးကြီးသော မှတ်စုများ

  • တိကျသော CSS လိုအပ်သည်-တိကျသော sTSS အတွက် သင်၏ CSS သည် လက်ရှိ (၆-၈ ပတ်အတွင်း) စမ်းသပ်ရပါမည်။
  • ရိုးရှင်းသော တွက်ချက်မှု-ဤဂဏန်းတွက်စက်သည် ပျမ်းမျှနှုန်းကို အသုံးပြုသည်။ Advanced sTSS သည် ကြားကာလဖွဲ့စည်းပုံအတွက် ထည့်သွင်းထားသော Normalized Graded Pace (NGP) ကို အသုံးပြုသည်။
  • နည်းပညာအလုပ်အတွက်မဟုတ်ပါsTSS သည် ကျွမ်းကျင်မှု ဖွံ့ဖြိုးတိုးတက်မှုကို မဟုတ်ဘဲ ရုပ်ပိုင်းဆိုင်ရာ လေ့ကျင့်ရေးဆိုင်ရာ ဖိစီးမှုကိုသာ တိုင်းတာသည်။
  • တစ်ဦးချင်း ကွဲပြားမှု-တူညီသော sTSS သည် မတူညီသော ရေကူးသမားများအတွက် ကွဲပြားခြားနားသည်ဟု ခံစားရသည် ။ သင့်ပြန်လည်ရယူခြင်းအပေါ် အခြေခံ၍ လမ်းညွှန်ချက်များကို ချိန်ညှိပါ။

sTSS သည် အဘယ်ကြောင့် အရေးကြီးသနည်း။

လေ့ကျင့်ရေးစိတ်ဖိစီးမှုရမှတ်သည် အောက်ပါတို့အတွက် အခြေခံအုတ်မြစ်ဖြစ်သည်။

  • CTL (နာတာရှည်လေ့ကျင့်ရေးဝန်)-သင်၏ကြံ့ခိုင်မှုအဆင့် - နေ့စဥ် sTSS ၏ ပျမ်းမျှအားဖြင့် 42 ရက်ကြာ အဆမတန်အလေးချိန်ရှိသည်။
  • ATL (Acute Training Load):သင်၏ ပင်ပန်းနွမ်းနယ်မှု - နေ့စဉ် sTSS ၏ ပျမ်းမျှအားဖြင့် 7-ရက်ကြာ အဆများသော အလေးချိန်
  • TSB (လေ့ကျင့်ရေးစိတ်ဖိစီးမှုဟန်ချက်)သင့်ပုံစံ - TSB = CTL - ATL (အပြုသဘော = လန်းဆန်းခြင်း၊ အနှုတ် = ပင်ပန်းနွမ်းနယ်ခြင်း)
  • ကာလအပိုင်းအခြားသတ်မှတ်ခြင်း-ပစ်မှတ် CTL တိုးတက်မှုများကို အသုံးပြု၍ လေ့ကျင့်ရေးအဆင့်များ (အခြေခံ၊ တည်ဆောက်မှု၊ အထွတ်အထိပ်၊ တိုတို) ကို စီစဉ်ပါ။
  • ပြန်လည်ရယူခြင်း စီမံခန့်ခွဲမှု-TSB ကိုအခြေခံ၍ မည်သည့်အချိန်တွင် တွန်းရမည်နှင့် မည်သည့်အချိန်တွင် အနားယူရမည်ကို သိပါ။

Pro အကြံပြုချက်- သင်၏ CTL ကို ခြေရာခံပါ။

နေ့စဉ် sTSS ကို စာရင်းဇယား သို့မဟုတ် လေ့ကျင့်ရေးမှတ်တမ်းတွင် မှတ်တမ်းတင်ပါ။ သင်၏ 42 ရက်ပျမ်းမျှ (CTL) ကို အပတ်စဉ် တွက်ချက်ပါ။ အခြေစိုက်စခန်းတည်ဆောက်နေစဉ်အတွင်း 5-10 CTL အမှတ်များ တိုးမြှင့်ရန် ပစ်မှတ်ထား။ ကျဉ်းနေချိန်တွင် CTL ကို ထိန်းသိမ်းပါ သို့မဟုတ် အနည်းငယ်လျှော့ချပါ (ပြိုင်ပွဲမစမီ 1-2 ပတ်)။

အမေးများသောမေးခွန်းများ

Swimming TSS (sTSS) ဆိုတာဘာလဲ။

ရေကူးလေ့ကျင့်ရေးစိတ်ဖိစီးမှုရမှတ် (sTSS) သည် ပြင်းထန်မှုနှင့် ကြာချိန်တို့ကို ပေါင်းစပ်ခြင်းဖြင့် ရေကူးလေ့ကျင့်ခန်း၏ လေ့ကျင့်ရေးဝန်ကို တိုင်းတာသည့် မက်ထရစ်တစ်ခုဖြစ်သည်။ စက်ဘီးစီးခြင်း၏ TSS နည်းစနစ်မှ လိုက်လျောညီထွေဖြစ်ပြီး သင်၏အရေးပါသော ရေကူးအမြန်နှုန်း (CSS) ကို သတ်မှတ်နှုန်းအဖြစ် အသုံးပြုသည်။ CSS အရှိန်ဖြင့် ၁ နာရီ လေ့ကျင့်ခန်းသည် 100 sTSS နှင့် ညီမျှသည်။

ကျွန်ုပ်၏ sTSS ကို မည်သို့တွက်ချက်ရမည်နည်း။

သင်၏ CSS အရှိန်အဟုန် (CSS စမ်းသပ်မှုတစ်ခုမှ)၊ စုစုပေါင်းလေ့ကျင့်ခန်းကြာချိန်နှင့် လေ့ကျင့်ခန်းအတွင်း ပျမ်းမျှခြေလှမ်းတို့ကို ထည့်သွင်းခြင်းဖြင့် အထက်ပါဂဏန်းတွက်စက်ကို အသုံးပြုပါ။ ဖော်မြူလာမှာ- sTSS = ကြာချိန် (နာရီ) × Intensity Factor³ × 100 ရှိရာ Intensity Factor = CSS Pace / Average Workout Pace ဖြစ်သည်။

sTSS ကိုတွက်ချက်ရန် CSS လိုအပ်ပါသလား။

ဟုတ်ကဲ့၊ sTSS တွက်ချက်မှုအတွက် မရှိမဖြစ်လိုအပ်သော Intensity Factor ကို တွက်ချက်ရန် သင်၏ Critical Swim Speed (CSS) လိုအပ်ပါသည်။ CSS သည် သင်၏ အဆင့်သတ်မှတ်နှုန်းကို ကိုယ်စားပြုပြီး 6-8 ပတ်တစ်ကြိမ် စမ်းသပ်သင့်သည်။ ကျွန်ုပ်တို့၏ CSS ကိုအသုံးပြု၍ သင်၏ CSS ကိုသင်ရှာဖွေနိုင်သည်။CSS ဂဏန်းတွက်စက်.

လေ့ကျင့်ခန်းတစ်ခုအတွက် sTSS ရမှတ်ကောင်းကဘာလဲ။

လေ့ကျင့်ခန်းပြင်းထန်မှုအပေါ် မူတည်သည်- လွယ်ကူသောလေ့ကျင့်ခန်းများသည် ပုံမှန်အားဖြင့် 50 sTSS အောက်တွင်ရမှတ်၊ အလယ်အလတ်လေ့ကျင့်ခန်း 50-100 sTSS၊ ပြင်းထန်သောလေ့ကျင့်ခန်း 100-200 sTSS နှင့် 200 sTSS အထက်တွင် အလွန်ခက်ခဲသောလေ့ကျင့်ခန်းများ။ သင့်လျော်သောရမှတ်သည် သင်၏လေ့ကျင့်ရေးပန်းတိုင်များနှင့် လက်ရှိကြံ့ခိုင်မှုအဆင့်အပေါ် မူတည်ပါသည်။

တစ်ပတ်ကို sTSS ဘယ်လောက်လုပ်သင့်လဲ။

အပတ်စဉ် sTSS ပစ်မှတ်များသည် အဆင့်အလိုက် ကွဲပြားသည်- အပန်းဖြေရေကူးသမားများ- 150-300၊ ကြံ့ခိုင်ရေး ရေကူးသမားများ/Triathletes- 300-500၊ ယှဉ်ပြိုင်နိုင်သော မာစတာများ- 500-800၊ Elite/Collegiate- 800-1200+။ လေ့ကျင့်မှု လွန်ကဲခြင်းကို ရှောင်ရှားရန် ရှေးရိုးဆန်စွာ စတင်ပြီး တဖြည်းဖြည်း တိုးပါ။

ရေကူးခြင်း TSS သည် စက်ဘီးစီးခြင်း TSS နှင့် အတူတူပင်လား။

အယူအဆနှင့် ဖော်မြူလာသည် တူညီသော်လည်း sTSS သည် ရေကူးရန်အတွက် အဆင်ပြေသည်။ TSS ကဲ့သို့ စက်ဘီးစီးခြင်းကဲ့သို့ ပါဝါ (FTP) ကို အသုံးပြုမည့်အစား sTSS သည် အဆင့်သတ်မှတ်ချက်အဖြစ် CSS နှင့်အတူ အရှိန်အဟုန်ကို အသုံးပြုသည်။ Duration × Intensity Factor³ × 100 ကို အသုံးပြု၍ လေ့ကျင့်ရေးဝန်ကို တိုင်းတာခြင်း။

ရေကူးခြင်းအားလုံးအတွက် sTSS ကို သုံးနိုင်ပါသလား။

ဟုတ်သည်၊ သို့သော် သင်၏ CSS သည် လေဖြတ်ခြင်းဆိုင်ရာ သီးသန့်ဖြစ်သင့်သည်။ ရေကူးသမားအများစုသည် အလွတ်စတိုင် CSS ကို အသုံးပြုလေ့ရှိပြီး ၎င်းသည် လေ့ကျင့်မှုအများဆုံး လေဖြတ်ခြင်းကြောင့်ဖြစ်သည်။ အကယ်၍ သင်သည် အခြားလေဖြတ်ခြင်းတွင် အဓိကလေ့ကျင့်ပါက၊ ထိုလေဖြတ်ခြင်းတွင် CSS စမ်းသပ်မှုပြုလုပ်ပြီး sTSS တွက်ချက်မှုအတွက် ထိုအရှိန်ကို အသုံးပြုပါ။

sTSS နှင့် CTL/ATL/TSB အကြား ကွာခြားချက်မှာ အဘယ်နည်း။

sTSS သည် လေ့ကျင့်ခန်းတစ်ခုတည်း၏ လေ့ကျင့်ရေးဝန်ကို တိုင်းတာသည်။ CTL (Chronic Training Load) သည် သင်၏ ရေရှည်ကြံ့ခိုင်မှု၊ ATL (Acute Training Load) သည် သင်၏ လတ်တလော ပင်ပန်းနွမ်းနယ်မှု၊ နှင့် TSB (Training Stress Balance) သည် သင်၏ လန်းဆန်းမှု ဖြစ်သည်။ ဤမက်ထရစ်များသည် သင်၏လေ့ကျင့်ရေးအခြေအနေကို ခြေရာခံရန် အချိန်ကြာလာသည်နှင့်အမျှ sTSS တန်ဖိုးများကို အသုံးပြုပါသည်။ ကျွန်ုပ်တို့၌ ပိုမိုလေ့လာပါ။Training Load လမ်းညွှန်.
