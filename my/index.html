Swim Analytics App ဆင်းသက်ခြင်းစာမျက်နှာ - ပင်မစာမျက်နှာ အက်ပ်မိတ်ဆက်နှင့်အတူ သူရဲကောင်းကဏ္ဍ
ပရော်ဖက်ရှင်နယ် ရေကူး ပိုင်းခြားစိတ်ဖြာချက်

သင်၏ရေကူးဒေတာကို စွမ်းဆောင်ရည်အဖြစ် ပြောင်းလဲပါ။

သိပ္ပံနည်းကျ တိုင်းတာမှုများ၊ ပုဂ္ဂိုလ်ရေးသီးသန့် လေ့ကျင့်ရေးဇုန်များနှင့် ပြည့်စုံသော စွမ်းဆောင်ရည် ခြေရာခံခြင်း။ သင်၏ iPhone တွင် အလုံးစုံ ကိုယ်ရေးကိုယ်တာ အပြည့်အ၀ဖြင့် စက်တွင်း စီမံဆောင်ရွက်သည်။

ဝန်ဆောင်မှုများကို စူးစမ်းပါ။

✓ 7 ရက် အခမဲ့ အစမ်းသုံးခြင်း ✓ အကောင့် မလိုအပ်ဘဲ ✓ 100% ဒေသတွင်းဒေတာ

အဓိကအင်္ဂါရပ်များကဏ္ဍ
အသွင်အပြင်

တိုးတက်ရန် လိုအပ်သမျှ

အဆင့်တိုင်းတွင် ရေကူးသမားများအတွက် ပရော်ဖက်ရှင်နယ်အဆင့်ခွဲခြမ်းစိတ်ဖြာချက်

သိပ္ပံနည်းကျ တိုင်းတာမှုများ

အရေးကြီးသော ရေကူးအမြန်နှုန်း (CSS)သင်၏ အေရိုးဗစ်အဆင့်ကို သတ်မှတ်ပေးခြင်း၊လေ့ကျင့်ရေးစိတ်ဖိစီးမှုရမှတ် (TSS)တွက်ချက်မှုနှင့်CTL/ATL/TSB စွမ်းဆောင်ရည် ခြေရာခံခြင်း။သက်သေပြထားသော အားကစားသိပ္ပံကို အခြေခံသည်။ သုတေသန။ ကျွန်ုပ်တို့၏အကြောင်းပိုမိုလေ့လာပါ။သင်္ချာဖော်မြူလာများနှင့်ရေကူးထိရောက်မှုတိုင်းတာမှု.

လေ့ကျင့်ရေးဇုန်များ

သင့် CSS တွင် စိတ်ကြိုက်သတ်မှတ်ထားသော လေ့ကျင့်ရေးဇုန် 7 ခုကို ချိန်ညှိထားသည်။ ပြန်လည်ထူထောင်ရေးအတွက် လေ့ကျင့်ခန်းတိုင်းကို အကောင်းဆုံးဖြစ်အောင်၊ အေရိုးဗစ်၊ သတ်မှတ်ချက် သို့မဟုတ် VO₂max ဖွံ့ဖြိုးတိုးတက်မှု။

စမတ်ကျသော နှိုင်းယှဉ်မှုများ

အပတ်စဉ်၊ လစဉ်၊ နှင့် နှစ်အလိုက် အချိန်ကာလများကို အလိုအလျောက် လမ်းကြောင်းရှာဖွေခြင်း နှင့် အားလုံးအတွက် ရာခိုင်နှုန်းပြောင်းလဲမှုများ နှိုင်းယှဉ်မှုများ တိုင်းတာမှုများ။

ပြီးပြည့်စုံသော ကိုယ်ရေးကိုယ်တာ

ဒေတာအားလုံးကို သင့်စက်ပေါ်တွင် စက်တွင်း၌ လုပ်ဆောင်သည်။ ဆာဗာများမရှိ၊ cloud မရှိ၊ ခြေရာခံခြင်းမရှိပါ။ သင်ပိုင်ဆိုင်ပြီး သင့်အား ထိန်းချုပ်ပါ။ ရေကူးဒေတာ။

ဘယ်နေရာမဆို တင်ပို့ပါ။

လေ့ကျင့်ခန်းများနှင့် ခွဲခြမ်းစိတ်ဖြာမှုများကို JSON၊ CSV၊ HTML၊ သို့မဟုတ် PDF ဖော်မတ်များဖြင့် ထုတ်ယူပါ။ နည်းပြများ၊ စာရင်းဇယားများနှင့် လိုက်ဖက်သည်။ လေ့ကျင့်ရေးပလက်ဖောင်းများ။

လက်ငင်းစွမ်းဆောင်ရည်

Sub-0.35s အက်ပ်ကို ဒေသဆိုင်ရာ ပထမဗိသုကာဖြင့် စတင်လိုက်ပါ။ စင့်ခ်လုပ်ခြင်းကို မစောင့်ဆိုင်းဘဲ သို့မဟုတ် သင့်လေ့ကျင့်ခန်းများကို ချက်ချင်းကြည့်ရှုပါ။ ဒေါင်းလုဒ်များ

ရှုပ်ထွေးမှုမရှိဘဲ Professional အသွင်အပြင်

CSS စမ်းသပ်မှု ပရိုတိုကော

သင်၏အရေးပါသောရေကူးအမြန်နှုန်းကိုဆုံးဖြတ်ရန် 400 မီတာ + 200 မီတာစမ်းသပ်မှုပရိုတိုကောကို တပ်ဆင်ထားသည်။ခြေရာခံရန် 6-8 ပတ်တစ်ကြိမ် ပြန်လုပ်ပါ။ တိုးတက်မှုနှင့် လေ့ကျင့်ရေးဇုန်များကို အလိုအလျောက် ချိန်ညှိပါ။

မူရင်း iOS အတွေ့အကြုံ

ချောမွေ့သောစွမ်းဆောင်ရည်နှင့် iOS ပေါင်းစပ်မှုအတွက် SwiftUI ဖြင့်တည်ဆောက်ထားသည်။ ချောမွေ့မှုမရှိသော ကျန်းမာရေးအက်ပ်စင့်ခ်၊ ဝစ်ဂျက်များပံ့ပိုးမှု၊ Apple ဒီဇိုင်းဘာသာစကားနှင့် ရင်းနှီးသည်။

သုတေသနအခြေခံ

သက်တူရွယ်တူများ သုံးသပ်ထားသော အားကစားသိပ္ပံ သုတေသနအပေါ် အခြေခံသည့် မက်ထရစ်များအားလုံး။ Wakayoshi et al., TSS မှ CSS ကို ဆီလျော်အောင် ဘာသာပြန်ထားသည်။ IF³ ဖော်မြူလာဖြင့် ရေကူးခြင်း၊ သက်သေပြထားသော CTL/ATL မော်ဒယ်များ။

နည်းပြ အဆင်ပြေပါတယ်။

နည်းပြများအတွက် အသေးစိတ်အစီရင်ခံစာများကို တင်ပို့ပါ။ HTML အနှစ်ချုပ်များကို အီးမေးလ်မှတစ်ဆင့်၊ ဇယားစာရွက်ခွဲခြမ်းစိတ်ဖြာမှုအတွက် CSV သို့မဟုတ် PDF မျှဝေပါ သင်တန်းမှတ်တမ်းများနှင့် မှတ်တမ်းများအတွက်။

နေရာတိုင်းတွင်အလုပ်လုပ်သည်။

ရေကူးကန် သို့မဟုတ် ရေဖွင့်ခြင်း၊ ပြေးခြင်း သို့မဟုတ် အကွာအဝေး။ Swim Analytics သည် ရေကူးအမျိုးအစားအားလုံးနှင့် အလိုအလျောက် လိုက်လျောညီထွေဖြစ်စေသည်။ လေ့ကျင့်ခန်းသွင်ပြင်လက္ခဏာများကို သိရှိနိုင်သည်။

အမေးများသောမေးခွန်းများ

Swim Analytics သည် ကျွန်ုပ်၏ ရေကူးဒေတာကို မည်သို့ရရှိသနည်း။

Swim Analytics သည် တွဲဖက်အသုံးပြုနိုင်သည့် စက် သို့မဟုတ် အက်ပ်မှမှတ်တမ်းတင်ထားသော ရေကူးလေ့ကျင့်ခန်းများကို တင်သွင်းရန် Apple Health နှင့် ထပ်တူပြုပါသည်။ ၎င်းတွင် စမတ်နာရီများ၊ ကြံ့ခိုင်မှု ခြေရာခံကိရိယာများနှင့် လူကိုယ်တိုင် ထည့်သွင်းမှုများ ပါဝင်သည်။ အပလီကေးရှင်းသည် ဤဒေတာကို စက်အတွင်းမှ လုပ်ဆောင်ပါသည်။ အဆင့်မြင့် မက်ထရစ်များကို တွက်ချက်ပါ။

CSS test ဆိုတာဘာလဲ၊ ဘယ်လိုလုပ်ဆောင်ရမလဲ။

CSS (Critical Swim Speed) သည် အမြင့်ဆုံးအားထုတ်ရေကူးခြင်း 2 ခုကို အသုံးပြုထားသော သိပ္ပံနည်းကျ ပရိုတိုကောတစ်ခုဖြစ်သည်- 400m နှင့် 200m 10-20 ဖြင့် မိနစ်ကြားတွင် အနားယူပါ။ အက်ပ်သည် ဤအချိန်များမှ သင်၏ အေရိုးဗစ်အဆင့်ကို တွက်ချက်ပြီး အားလုံးကို အလိုအလျောက် ချိန်ညှိပေးသည်။ လေ့ကျင့်ရေးဇုန်များ။ တိုးတက်မှုကိုခြေရာခံရန် 6-8 ပတ်တစ်ကြိမ်ပြန်လုပ်ပါ။

ကျွန်ုပ်၏ဒေတာကို cloud သို့ အပ်လုဒ်လုပ်ပါသလား။

နံပါတ်၊ Swim Analytics သည် သင်၏ iPhone တွင် ဒေသအလိုက် ဒေတာအားလုံးကို လုပ်ဆောင်ပါသည်။ ပြင်ပဆာဗာများ ၊ Cloud မရှိပါ။ အကောင့်များ၊ ဒေတာလွှဲပြောင်းမှုများမရှိပါ။ ပို့ကုန်များကို သင်ထိန်းချုပ်သည်- JSON၊ CSV၊ HTML၊ သို့မဟုတ် PDF ဖိုင်များကို ဖန်တီးပြီး ၎င်းတို့ကို မျှဝေပါ သင်အလိုရှိသော်လည်း။

ပွင့်လင်းသောရေကူးရန်အတွက် Swim Analytics ကိုသုံးနိုင်ပါသလား။

ဟုတ်ကဲ့။ Swim Analytics သည် open water အပါအဝင် Apple Health ရှိ မည်သည့်ရေကူးလေ့ကျင့်ခန်းနှင့်မဆို လုပ်ဆောင်ပါသည်။ အက်ပ်သည် လိုက်လျောညီထွေဖြစ်စေသည်။ သင်ရေကူးကန် သို့မဟုတ် ရေထဲတွင်ရှိနေသည်ဖြစ်စေ ပတ်ဝန်းကျင်တစ်ခုစီအတွက် သက်ဆိုင်ရာခွဲခြမ်းစိတ်ဖြာမှုကို ပေးစွမ်းနိုင်သော တိုင်းတာမှုများ။

လစဉ်နှင့် နှစ်အလိုက် အစီအစဉ်များ ကွာခြားချက်မှာ အဘယ်နည်း။

အစီအစဥ်နှစ်ခုစလုံးသည် တူညီသောအင်္ဂါရပ်များကို ပေးဆောင်သည်- မက်ထရစ်များအားလုံး၊ အကန့်အသတ်မဲ့ဇုန်များ၊ ယာယီနှိုင်းယှဉ်မှုများ၊ ပို့ကုန်အများအပြား၊ နှင့် အခမဲ့ အပ်ဒိတ်များ။ တစ်ခုတည်းသော ကွာခြားချက်မှာ စျေးနှုန်းဖြစ်သည်- နှစ်စဉ် 18% သက်သာသည် (တစ်လလျှင် ယူရို 3.25 နှင့် ယူရို 3.99/လ နှင့်ညီမျှသည်)။

ကျွန်ုပ်၏ စာရင်းသွင်းမှုကို အချိန်မရွေး ပယ်ဖျက်နိုင်ပါသလား။

ဟုတ်ကဲ့။ စာရင်းသွင်းမှုများကို App Store မှတဆင့် စီမံခန့်ခွဲသည်၊ ထို့ကြောင့် ဆက်တင်များ → [သင့်အမည်] → အချိန်မရွေး ပယ်ဖျက်နိုင်သည်။ စာရင်းသွင်းမှုများ။ သင်ပယ်ဖျက်ပါက၊ သင်၏လက်ရှိငွေပေးချေမှုကာလ ကုန်ဆုံးသည်အထိ ဝင်ရောက်အသုံးပြုနိုင်မည်ဖြစ်သည်။
