  • бһӮбһҺбһ“бһ¶бһҸбҹҶбһ”бһ“бҹӢбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһЎбһҫбһ„бһңбһ·бһү
  • бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһЎбһҫбһ„бһңбһ·бһү sTSS бһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣ 90 бһҗбҹ’бһ„бҹғбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ
  • бһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһң CTL/ATL/TSB бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶
  • бһ”бһҫбһҖбһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһӣбҹҶбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҸбҹҶбһ”бһ“бҹӢ

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҹбҹ’бһӮбһ¶бһӣбҹӢ CSS бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҡбһҪбһ…бһ бһҫбһҷбһ–бһёбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһ–бһёбһҳбһ»бһ“ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһүбҹ’бһ…бһјбһӣбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ” бһ‘бҹ„бҹҮбһҮбһ¶бһҷбҹүбһ¶бһ„бһҺбһ¶бһҖбҹҸбһҠбҹ„бһҷ бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ”бҹҶбһ•бһ»бһҸ бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһҗбҹ’бһҳбһёбһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢ 6-8 бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбҹ” CSS бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„ (бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӣбһҝбһ“бһҮбһ¶бһ„бһҳбһ»бһ“) бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“бҹ”

бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһҳбһ¶бһҸбҹ’бһҡбһҠбҹ’бһӢбһ¶бһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһӣбҹ’бһ”бһҝбһ“бһ бҹӮбһӣбһ‘бһ№бһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢ (CSS)

бһҸбһҫбһңбһ¶бһҮбһ¶бһўбҹ’бһңбһёбҹ–бһӣбҹ’бһ”бһҝбһ“бһҖбһҳбҹ’бһҡбһ·бһҸ aerobic бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ - бһӣбҹ’бһ”бһҝбһ“бһӣбһҝбһ“бһ”бҹҶбһ•бһ»бһҸбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһ”бһ¶бһ“бһҡбһҷбҹҲбһ–бҹҒбһӣ ~ 30 бһ“бһ¶бһ‘бһёбһҠбҹ„бһҷбһҳбһ·бһ“бһўбһҹбҹӢбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бҹ”

бһҸбһҫбһңбһ¶бһҳбһ¶бһ“бһ“бҹҗбһҷбһҷбҹүбһ¶бһ„бһҺбһ¶бҹ–CSS = 1:49/100m бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһӣбҹ’бһ”бһҝбһ“ 1:49 бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ·бһ…бҹ’бһ…бһҒбһ·бһҸбһҒбҹҶбһ”бҹ’бһҡбһ№бһ„бһ”бҹ’бһҡбҹӮбһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһҠбҹӮбһӣбһ‘бҹ’бһҡбһ‘бҹ’бһҡбһ„бҹӢбҹ”

бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбҹ–бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹҶбһ”бһ“бҹӢбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶ sTSS бҹ” бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢ 6-8 бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбҹ”

бһҸбҹҶбһ”бһ“бҹӢбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ

бһҸбһҫбһ–бһҪбһҖбһӮбҹҒбһҮбһ¶бһўбҹ’бһңбһёбҹ–бһҮбһҪбһҡбһўбһ¶бҹҶбһ„бһҸбһ„бҹӢбһҹбҹҠбһёбһҸбҹҒ 7 бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ CSS бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ–бһёбһҖбһ¶бһҡбһ„бһҫбһ”бһЎбһҫбһ„бһңбһ·бһү (бһҸбҹҶбһ”бһ“бҹӢ 1) бһҠбһӣбҹӢбһҖбһ¶бһҡбһҡбһҸбҹӢ (бһҸбҹҶбһ”бһ“бҹӢ 7) бҹ”

бһҸбһҫбһ–бһҪбһҖбһӮбҹҒбһҳбһ¶бһ“бһ“бҹҗбһҷбһҷбҹүбһ¶бһ„бһҺбһ¶бҹ–бһҸбҹҶбһ”бһ“бҹӢбһ“бһёбһҳбһҪбһҷбҹ—бһҖбҹҶбһҺбһҸбҹӢбһӮбҹ„бһӣбһҠбҹ…бһҹбһҳбҹ’бһҡбһ”бһҒбҹ’бһӣбһҪбһ“бһҸбһ¶бһҳбһҹбһҡбһёбһҡбһңбһ·бһ‘бҹ’бһҷбһ¶бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ (бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ aerobic бһҖбһҳбҹ’бһҡбһ·бһҸ VOвӮӮmax)бҹ”

бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбҹ–бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһңбҹҒбһҮбҹ’бһҮбһ”бһүбҹ’бһҮбһ¶бһҸбҹҶбһ”бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҳбҹүбҹ„бһ„бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺбһ“бһёбһҳбһҪбһҷбҹ—бҹ”

бһҸбҹҶбһ”бһ“бҹӢбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ вҶ’

бһ–бһ·бһ“бҹ’бһ‘бһ»бһҹбҹ’бһҸбҹ’бһҡбҹҒбһҹбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһ бҹӮбһӣбһ‘бһ№бһҖ (sTSS)

бһҸбһҫбһңбһ¶бһҮбһ¶бһўбҹ’бһңбһёбҹ–бһ—бһ¶бһ–бһҸбһ¶бһ“бһҸбһ№бһ„бһ“бҹғбһҖбһ¶бһҡбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺбһҸбһ¶бһҳбһ”бһҡбһ·бһҳбһ¶бһҺбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһўбһ¶бҹҶбһ„бһҸбһ„бҹӢбһҹбҹҠбһёбһҸбҹҒ бһ“бһ·бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбҹ” 1 бһҳбҹүбҹ„бһ„бһ“бҹ… CSS бһӣбҹ’бһ”бһҝбһ“ = 100 sTSS бҹ”

бһҸбһҫбһңбһ¶бһҳбһ¶бһ“бһ“бҹҗбһҷбһҷбҹүбһ¶бһ„бһҺбһ¶бҹ–sTSS 50 = бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ„бһҫбһ”бһЎбһҫбһ„бһңбһ·бһү, sTSS 100 = бһҳбһ’бҹ’бһҷбһҳ, sTSS 200+ = бһңбһӮбҹ’бһӮбһ–бһ·бһ”бһ¶бһҖбһҺбһ¶бһҹбҹӢбҹ”

бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбҹ–бһҸбһ¶бһҳбһҠбһ¶бһ“бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғ/бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ sTSS бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ“бҹ’бһ‘бһ»бһҖбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ” бһӮбҹ„бһӣбһҠбҹ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ 5-10 sTSS бһҖбһҫбһ“бһЎбһҫбһ„бһўбһҸбһ·бһ”бһҡбһҳбһ¶бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбҹ”

sTSS бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бҹҒбһҹбһҖбҹҚ вҶ’

CTL / ATL / TSB

бһҸбһҫбһ–бһҪбһҖбһӮбҹҒбһҮбһ¶бһўбҹ’бһңбһёбҹ–

  • CTLбҹ–бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҡбҹүбһ¶бҹҶбһҡбҹүбҹғ (бһҹбһҳбҹ’бһ”бһ‘бһ¶) - бһҮбһ¶бһҳбһ’бҹ’бһҷбһҳ 42 бһҗбҹ’бһ„бҹғ sTSS
  • ATLбҹ–бһ”бһ“бҹ’бһ‘бһ»бһҖбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһҹбҹ’бһҡбһҪбһ…бһҹбҹ’бһҡбһ¶бһң (бһўбһҹбҹӢбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„) - бһҳбһ’бҹ’бһҷбһҳ 7 бһҗбҹ’бһ„бҹғ sTSS
  • TSBбҹ–бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһ—бһ¶бһ–бһҸбһ¶бһ“бһҸбһ№бһ„ (бһ‘бһҳбҹ’бһҡбһ„бҹӢ) = CTL - ATL

бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбҹ–бһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ TSB = бһҹбҹ’бһҡбһҹбҹӢ/бһҒбҹ’бһӣбһё бһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ TSB = бһўбһҹбҹӢбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бҹ” бһ”бҹ’бһҡбһҺбһ¶бҹҶбһ„бһ“бҹ…бһ–бҹҒбһӣ TSB = +5 бһҠбһӣбҹӢ +25 бҹ”

рҹ“Ҡ бһӮбҹ„бһӣбһҠбҹ…бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ…бһјбһӣ CSS бһ бһҫбһҷбһ”бҹҶбһ–бҹҒбһүбһҖбһ¶бһҡбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺ 3-5бҹ–

  • бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҸбһҳбҹ’бһӣбҹғ sTSSбҹ–бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢвҖӢбһҗбһ¶вҖӢбһ–бһҪбһҖбһӮбҹҒвҖӢбһҸбҹ’бһҡбһјбһңвҖӢбһӮбҹ’бһ“бһ¶вҖӢбһ“бһ№бһ„вҖӢбһҖбһ¶бһҡвҖӢбһҷбһӣбҹӢвҖӢбһғбһҫбһүвҖӢбһ–бһёвҖӢбһҖбһ¶бһҡвҖӢбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳ (бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ ~50, бһҳбһ’бҹ’бһҷбһҳ ~100, бһҡбһ№бһ„ ~150+)
  • бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһҸбҹҶбһ”бһ“бҹӢбҹ–бһҸбһҫбһўбҹ’бһ“бһҖбһ…бҹҶбһҺбһ¶бһҷ 60-70% бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ 2 (бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ aerobic) бһҳбҹӮбһ“бһ‘бҹҒ?
  • бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ CTLбҹ–бһҳбһ’бҹ’бһҷбһҳбһ—бһ¶бһӮ sTSS бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһҒбҹ’бһҹбҹӮбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһ»бһҒбһ—бһ¶бһ–бһҠбҹҶбһ”бһјбһ„
  • бһҖбҹҶбһҺбһҸбҹӢбһӣбҹҶбһ“бһ¶бҹҶбҹ–бһҸбһҫвҖӢбһҖбһ¶бһҡвҖӢбһ бһ¶бһҸбҹӢвҖӢбһ”бҹ’бһҡбһ¶бһҺвҖӢбһҳбһҪбһҷвҖӢбһҺбһ¶вҖӢбһ”бһ„бҹ’бһҖбһҫбһҸвҖӢбһ”бһ¶бһ“вҖӢбһҒбҹ’бһ–бһҹбҹӢвҖӢбһ”бҹҶбһ•бһ»бһҸ sTSS? бһҸбһҫбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ’бһјбһҡбһҹбҹ’бһ”бһҫбһҷбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһ‘бҹҒ?

бһҠбҹҶбһҺбһҫбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бһҳбҹ’бһҳбһҸбһ¶ (8 бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһҠбҹҶбһ”бһјбһ„)

бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһ‘бһё 1-2бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“

  • бһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ CSS бһ бһҫбһҷбһ”бһүбҹ’бһ…бһјбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣ
  • бһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһ’бһҳбҹ’бһҳбһҸбһ¶ 3-5
  • бһҹбһ„бҹ’бһҖбҹҒбһҸбһҳбһҫбһӣбһҸбһҳбҹ’бһӣбҹғ sTSS бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһҸбҹҶбһ”бһ“бҹӢ
  • бһ”бһ„бҹ’бһҖбһҫбһҸ CTL бһҠбҹҶбһ”бһјбһ„ (бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһҳбҹ’бһ”бһ‘бһ¶)
  • бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бҹ–бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһҳбһ¶бһҸбҹ’бһҡбһҠбҹ’бһӢбһ¶бһ“ бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ“бҹ…бһЎбһҫбһҷ

бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһ‘бһё 3-4бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбҹҶбһ”бһ“бҹӢ

  • бһ”бҹ’бһҡбһҫбһҸбҹҶбһ”бһ“бҹӢ CSS бһҖбҹ’бһ“бһ»бһ„бһ•бҹӮбһ“бһҖбһ¶бһҡбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺ
  • бһ…бҹҒбһҸбһ“бһ¶бһ бҹӮбһӣбһ‘бһ№бһҖбһҸбҹҶбһ”бһ“бҹӢбһ‘бһё 2 бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҲбһ»бһҸ aerobic
  • бһҸбһ¶бһҳбһҠбһ¶бһ“бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ sTSS бһҹбһҡбһ»бһ” (бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһ–бһҹбҹҠбһёбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһӮбҹ’бһ“бһ¶)
  • бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ TSB (бһӮбһҪбһҡбһҸбҹӮбһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“бһ”бһ“бҹ’бһҸбһ·бһ… = бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ)
  • бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бҹ–бһҡбһҗбһ—бҹ’бһӣбһҫбһ„бһҸбһ¶бһҳбһҸбҹҶбһ”бһ“бҹӢбһҳбһ·бһ“бһҳбһ¶бһ“бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ

бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһ‘бһё 5-6бҹ– бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһҹбһ…бҹҶбһҺбһ»бҹҮ

  • бһ”бһ„бҹ’бһҖбһҫбһ“ sTSS бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ 5-10% бһ–бһёбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“
  • бһ”бһ“бҹ’бһҗбҹӮбһҳ 1 бһҖбһҳбҹ’бһҡбһ·бһҸ (бһҸбҹҶбһ”бһ“бҹӢ 4) бһңбһӮбҹ’бһӮбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ
  • CTL бһӮбһҪбһҡбһҸбҹӮбһҖбһҫбһ“бһЎбһҫбһ„бһҮбһ¶бһ”бһҺбҹ’бһҸбһҫбһҡбҹ— (бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһҹбһ»бһҒбһ—бһ¶бһ–)
  • ATL бһўбһ¶бһ…бһҖбһҫбһ“бһЎбһҫбһ„бһ“бҹ…бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһҡбһ№бһ„ (бһ’бһҳбҹ’бһҳбһҸбһ¶)
  • бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бҹ–бһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһңбһҢбҹ’бһҚбһ“бһ—бһ¶бһ–бһҹбһҳбҹ’бһ”бһ‘бһ¶

бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһ‘бһё 7-8бҹ– бһҸбҹҒбһҹбҹ’бһҸбһЎбһҫбһ„бһңбһ·бһү & бһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһң

  • бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ CSS бһӣбһҫбһҖбһ‘бһёбһ–бһёбһҡ (бһӮбһҪбһҡбһҸбҹӮбһӣбһҝбһ“бһҮбһ¶бһ„)
  • бһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһҸбҹҶбһ”бһ“бҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё (бһӣбҹ’бһ”бһҝбһ“бһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„)
  • бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ” CTL бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһ‘бһё 1 бһ‘бһӣбҹӢбһ“бһ№бһ„бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһ‘бһё 8 (бһӮбһҪбһҡбһҸбҹӮбһҮбһ¶ +10-20)
  • бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһңбһҢбҹ’бһҚбһ“бһ—бһ¶бһ–бҹ– бһҸбһҫбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бһ»бҹҮбһҳбҹӮбһ“бһ‘бҹҒ? бһҳбһ¶бһ“бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҹбҹ’бһҡбһҪбһӣбһҮбһ¶бһ„?
  • бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бҹ–бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ“бҹғбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ

вң… бһҹбһјбһ…бһ“бһ¶бһҖбһҡбһҮбҹ„бһӮбһҮбҹҗбһҷ

  • бһҖбһ¶бһҡбһ„бһҫбһ”бһЎбһҫбһ„бһңбһ·бһүбһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„бҹ–TSB бһңбһҠбҹ’бһҸбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚ (-10 бһҠбһӣбҹӢ +5)

бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ вҶ’ бһӣбҹ’бһ”бһҝбһ“бһ бҹӮбһӣбһ‘бһ№бһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢ вҶ’ бһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбҹ” бһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ¶бҹҶбһ„ 400m бһ“бһ·бһ„ 200m бһҠбһ„ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ…бһ»бһ… Save бҹ”
