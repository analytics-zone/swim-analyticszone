ហែលទឹកដោយឥតគិតថ្លៃ TSS ម៉ាស៊ីនគិតលេខ

គណនា Stress Stress សម្រាប់ការហាត់ប្រាណហែលទឹក - ម៉ាស៊ីនគិតលេខឥតគិតថ្លៃ sTSS តែមួយគត់

តើអ្វីជាហែលទឹក TSS (sTSS)?

ពិន្ទុស្ត្រេសហ្វឹកហាត់ហែលទឹក (sTSS)កំណត់បរិមាណនៃបន្ទុកហ្វឹកហាត់នៃការហាត់ប្រាណហែលទឹកដោយរួមបញ្ចូលគ្នានូវអាំងតង់ស៊ីតេ និងរយៈពេល។ វាត្រូវបានកែសម្រួលពីវិធីសាស្រ្ត TSS របស់អ្នកជិះកង់ ដោយប្រើល្បឿនហែលទឹកដ៏សំខាន់ (CSS) ជាល្បឿនកំណត់។ ការហាត់ប្រាណ 1 ម៉ោងនៅ CSS ល្បឿន = 100 sTSS ។

ហេតុអ្វី sTSS សំខាន់

Training Stress Score គឺជាមូលដ្ឋានគ្រឹះសម្រាប់៖

  • CTL (ផ្ទុកការបណ្តុះបណ្តាលរ៉ាំរ៉ៃ):កម្រិតសម្បទារបស់អ្នក - ទម្ងន់អិចស្ប៉ូណង់ស្យែលរយៈពេល 42 ថ្ងៃនៃប្រចាំថ្ងៃ sTSS
  • ATL (ផ្ទុកការបណ្តុះបណ្តាលស្រួចស្រាវ):ភាពអស់កម្លាំងរបស់អ្នក - ទម្ងន់អិចស្ប៉ូណង់ស្យែលរយៈពេល 7 ថ្ងៃនៃប្រចាំថ្ងៃ sTSS
  • TSB (Training Stress Balance):ទម្រង់របស់អ្នក - TSB = CTL - ATL (វិជ្ជមាន = ស្រស់ អវិជ្ជមាន = អស់កម្លាំង)
  • សម័យកាលៈផែនការដំណាក់កាលបណ្តុះបណ្តាល (មូលដ្ឋាន ស្ថាបនា កំពូល ខ្លី) ដោយប្រើគោលដៅ CTL វឌ្ឍនភាព
  • ការគ្រប់គ្រងការងើបឡើងវិញ៖ដឹងថាពេលណាត្រូវរុញ និងពេលណាត្រូវសម្រាកដោយផ្អែកលើ TSB

Pro Tip៖ តាមដាន CTL របស់អ្នក

កត់ត្រាប្រចាំថ្ងៃ sTSS នៅក្នុងសៀវភៅបញ្ជី ឬកំណត់ហេតុបណ្តុះបណ្តាល។ គណនាជាមធ្យម 42 ថ្ងៃរបស់អ្នក (CTL) ប្រចាំសប្តាហ៍។ គោលដៅ 5-10 CTL ពិន្ទុកើនឡើងក្នុងមួយសប្តាហ៍កំឡុងពេលសាងសង់មូលដ្ឋាន។ រក្សា ឬកាត់បន្ថយបន្តិច CTL កំឡុងពេលកាត់បន្ថយ (1-2 សប្តាហ៍មុនការប្រណាំង)។

សំនួរដែលសួរញឹកញាប់

តើអ្វីជាហែលទឹក TSS (sTSS)?

ពិន្ទុស្ត្រេសហ្វឹកហ្វឺនហែលទឹក (sTSS) គឺជារង្វាស់ដែលកំណត់បរិមាណនៃការហ្វឹកហាត់នៃការហាត់ប្រាណដោយរួមបញ្ចូលគ្នាទាំងអាំងតង់ស៊ីតេ និងរយៈពេល។ វាត្រូវបានកែសម្រួលពីវិធីសាស្រ្ត TSS របស់អ្នកជិះកង់ ដោយប្រើល្បឿនហែលទឹកដ៏សំខាន់ (CSS) ជាល្បឿនកំណត់។ ការហាត់ប្រាណ 1 ម៉ោងនៅ CSS ល្បឿនស្មើនឹង 100 sTSS ។

តើខ្ញុំគណនា sTSS របស់ខ្ញុំដោយរបៀបណា?

ប្រើម៉ាស៊ីនគិតលេខខាងលើដោយបញ្ចូលល្បឿន CSS របស់អ្នក (ពីការធ្វើតេស្ត CSS) រយៈពេលនៃការហាត់ប្រាណសរុប និងល្បឿនមធ្យមអំឡុងពេលហាត់ប្រាណ។ រូបមន្តគឺ៖ sTSS = រយៈពេល (ម៉ោង) × Intensity Factor³ × 100 ដែល Intensity Factor = CSS Pace / Average Workout Pace ។

តើខ្ញុំត្រូវការ CSS ដើម្បីគណនា sTSS ទេ?

បាទ/ចាស ល្បឿនហែលទឹកសំខាន់ (CSS) របស់អ្នកត្រូវបានទាមទារដើម្បីគណនាអាំងតង់ស៊ីតេ Factor ដែលចាំបាច់សម្រាប់ការគណនា sTSS។ CSS តំណាងឱ្យល្បឿនកំណត់របស់អ្នក ហើយគួរតែត្រូវបានធ្វើតេស្តរៀងរាល់ 6-8 សប្តាហ៍។ អ្នកអាចស្វែងរក CSS របស់អ្នកដោយប្រើរបស់យើង។ CSS ម៉ាស៊ីនគិតលេខ.

តើអ្វីទៅជាពិន្ទុ sTSS ល្អសម្រាប់ការហាត់ប្រាណមួយ?

វាអាស្រ័យលើអាំងតង់ស៊ីតេនៃការហាត់ប្រាណ៖ ការហាត់ប្រាណងាយៗជាធម្មតាទទួលបានពិន្ទុក្រោម 50 sTSS ការហាត់ប្រាណកម្រិតមធ្យម 50-100 sTSS ការហាត់ប្រាណខ្លាំង 100-200 sTSS និងការហាត់ប្រាណខ្លាំងលើសពី 200 ___TECH_13 ពិន្ទុសមរម្យអាស្រ័យលើគោលដៅហ្វឹកហាត់របស់អ្នក និងកម្រិតកាយសម្បទាបច្ចុប្បន្ន។

តើខ្ញុំគួរធ្វើ sTSS ប៉ុន្មានក្នុងមួយសប្តាហ៍?

គោលដៅ sTSS ប្រចាំសប្តាហ៍ប្រែប្រួលតាមកម្រិត៖ អ្នកហែលទឹកកម្សាន្ត៖ 150-300 នាក់ អ្នកហាត់ហែលទឹក/អត្តពលិក 300-500 នាក់ ថ្នាក់អនុបណ្ឌិតប្រកួតប្រជែង៖ 500-800 នាក់ វរជន/មហាវិទ្យាល័យ៖ 800-1200+ ។ ចាប់ផ្តើមបែបអភិរក្ស និងបង្កើនបន្តិចម្តងៗ ដើម្បីជៀសវាងការហ្វឹកហាត់លើសកម្រិត។

តើការហែលទឹក TSS ដូចគ្នានឹងការជិះកង់ TSS ដែរទេ?

គោលគំនិត និងរូបមន្តគឺដូចគ្នា ប៉ុន្តែ sTSS ត្រូវបានកែសម្រួលសម្រាប់ការហែលទឹក។ ជំនួសឱ្យការប្រើថាមពល (FTP) ដូចជាការជិះកង់ TSS, sTSS ប្រើល្បឿនជាមួយ CSS ជាកម្រិតចាប់ផ្ដើម។ ទាំងការវាស់វែងការហ្វឹកហាត់ផ្ទុកដោយប្រើប្រាស់ Duration × Intensity Factor³ × 100.

តើខ្ញុំអាចប្រើ sTSS សម្រាប់ការហែលទឹកទាំងអស់បានទេ?

បាទ/ចាស ប៉ុន្តែ CSS របស់អ្នកគួរតែមានជំងឺដាច់សរសៃឈាមខួរក្បាលជាក់លាក់។ អ្នកហែលទឹកភាគច្រើនប្រើស្ទីលសេរី CSS ចាប់តាំងពីវាជាជំងឺដាច់សរសៃឈាមខួរក្បាលដែលត្រូវបានបណ្តុះបណ្តាលញឹកញាប់បំផុត។ ប្រសិនបើអ្នកហ្វឹកហាត់ជាចម្បងក្នុងជំងឺដាច់សរសៃឈាមខួរក្បាលផ្សេងទៀត អនុវត្តការធ្វើតេស្ត CSS នៅក្នុងការដាច់សរសៃឈាមខួរក្បាលនោះ ហើយប្រើល្បឿននោះសម្រាប់ការគណនា sTSS ។

តើអ្វីជាភាពខុសគ្នារវាង sTSS និង CTL/ATL/TSB?

sTSS វាស់បន្ទុកនៃការហ្វឹកហាត់តែមួយ។ CTL (ការផ្ទុកការបណ្តុះបណ្តាលរ៉ាំរ៉ៃ) គឺជាកាយសម្បទារយៈពេលវែងរបស់អ្នក ATL (ការផ្ទុកការបណ្តុះបណ្តាលស្រួចស្រាវ) គឺជាភាពអស់កម្លាំងថ្មីៗរបស់អ្នក ហើយ TSB (តុល្យភាពភាពតានតឹងការបណ្តុះបណ្តាល) គឺជាភាពស្រស់ស្រាយរបស់អ្នក។ ម៉ែត្រទាំងនេះប្រើតម្លៃ sTSS តាមពេលវេលាដើម្បីតាមដានស្ថានភាពហ្វឹកហាត់របស់អ្នក។ ស្វែងយល់បន្ថែមនៅក្នុងរបស់យើង។ ការណែនាំអំពីការបណ្តុះបណ្តាល.
