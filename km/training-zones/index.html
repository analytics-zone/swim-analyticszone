бһҹбҹҶбһҺбһҪбһҡбһӮбҹҒбһҹбһҪбһҡбһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҸбҹҶбһ”бһ“бҹӢбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ

бһҸбһҫбһҒбҹ’бһүбһ»бҹҶбһӮбһҪбһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ CSS бһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһЎбһҫбһ„бһңбһ·бһүбһүбһ№бһҖбһүбһ¶бһ”бҹӢбһ”бҹүбһ»бһҺбҹ’бһҺбһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҸбҹҶбһ”бһ“бҹӢ?

бһҡбһ¶бһӣбҹӢ6-8 бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһҖбҹҶбһЎбһ»бһ„бһ–бҹҒбһӣбһҹбһ¶бһ„бһҹбһ„бҹӢ бһ“бһ·бһ„бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бҹ” CSS бһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖвҖӢбһӮбһҪбһҡвҖӢбһҸбҹӮвҖӢбһ’бҹ’бһңбһҫвҖӢбһұбҹ’бһҷвҖӢбһ”бҹ’бһҡбһҹбһҫбһҡвҖӢбһЎбһҫбһ„ (бһҖбһ¶бһ“бҹӢвҖӢбһҸбҹӮвҖӢбһӣбһҝбһ“) бһҒбһҺбҹҲвҖӢбһҠбҹӮбһӣвҖӢбһҹбһҳбҹ’бһ”бһ‘бһ¶вҖӢбһҳбһ¶бһ“вҖӢбһҖбһ¶бһҡвҖӢбһҖбһҫбһ“вҖӢбһЎбһҫбһ„ бһ‘бһ¶бһҳбһ‘бһ¶бһҡвҖӢбһұбҹ’бһҷвҖӢбһҳбһ¶бһ“вҖӢбһҖбһ¶бһҡвҖӢбһҖбҹӮвҖӢбһҹбһҳбҹ’бһҡбһҪбһӣвҖӢбһҸбҹҶбһ”бһ“бҹӢбҹ” бһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһЎбһҫбһ„бһңбһ·бһүбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҮбҹҶбһ„бһә бһҡбһ”бһҪбһҹ бһ¬бһҹбһҳбҹ’бһҡбһ¶бһҖбһҷбһјбһҡбҹ”

бһҸбһҫбһҒбҹ’бһүбһ»бҹҶбһўбһ¶бһ…бһӣбһ¶бһҷбһҸбҹҶбһ”бһ“бҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺбһҸбҹӮбһҳбһҪбһҷбһ”бһ¶бһ“бһ‘бҹҒ?

бһ”бһ¶бһ‘- бһҖбһ¶бһҡбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһӮбһәбһ–бһ бһ»бһҸбҹҶбһ”бһ“бҹӢбҹ”бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ– 400m бһҸбҹҶбһ”бһ“бҹӢ 1 warm-up + 8Г—100 Zone 4 threshold + 300m Zone 1 cool-downбҹ” бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢбһӮбһәбһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҸбҹҶбһ”бһ“бҹӢбһҠбҹ„бһҷбһ…бҹҒбһҸбһ“бһ¶ бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҖбһ¶бһҡбһ бҹӮбһӣбһ‘бһ№бһҖ "бһҸбҹҶбһ”бһ“бҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣ" бһҠбҹ„бһҷбһ…бҹғбһҠбһ“бҹ’бһҷбһ“бҹ„бҹҮбһ‘бҹҒбҹ”

бһ…бһ»бҹҮбһ”бһҫбһҒбҹ’бһүбһ»бҹҶбһҳбһ·бһ“бһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһӣбҹ’бһ”бһҝбһ“бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ?

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһӣбҹ’бһ”бһҝбһ“бһҸбҹҶбһ”бһ“бҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ‘бҹҒ бһ‘бһ¶бҹҶбһ„бҹ– (1) CSS бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ бһҪбһҹбһҹбһҳбҹҗбһҷ (бһӣбһҝбһ“бһ–бҹҒбһҖ) (2) бһўбҹ’бһ“бһҖбһўбһҹбҹӢбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„ (бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ TSB) бһ¬ (3) бһҖбһ¶бһҡбһ„бһҫбһ”бһЎбһҫбһ„бһңбһ·бһүбһҳбһ·бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҡбһңбһ¶бһ„бһ…бһ“бҹ’бһӣбҹ„бҹҮбһ–бҹҒбһӣбҹ” бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ CSS бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҡбһҝбһ„бһ“бҹҒбҹҮбһҖбһҫбһҸбһЎбһҫбһ„бһҮбһ¶бһ”бҹӢбһӣбһ¶бһ”бҹӢбҹ”

бһҸбһҫбһҸбҹҶбһ”бһ“бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ…бҹҶбһ–бҹ„бҹҮбһҮбҹҶбһ„бһәбһҠбһ¶бһ…бҹӢбһҹбһҡбһҹбҹғбһҲбһ¶бһҳбһҒбһҪбһҡбһҖбҹ’бһ”бһ¶бһӣбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһҡбһ¬бһ‘бҹҒ?

CSS бһҮбһ¶бһ’бһҳбҹ’бһҳбһҸбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„freestyle. бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһўбҹ’бһ“бһҖвҖӢбһ бҹӮбһӣвҖӢбһ‘бһ№бһҖвҖӢбһҠбҹӮбһӣвҖӢбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢвҖӢбһҠбһ¶бһ…бҹӢвҖӢбһҹбһҡбһҹбҹғвҖӢбһҲбһ¶бһҳвҖӢбһҖбҹ’бһ“бһ»бһ„вҖӢбһҒбһҪбһҡбһҖбҹ’бһ”бһ¶бһӣвҖӢбһҮбһ¶вҖӢбһ…бһҳбҹ’бһ”бһ„вҖӢбһ•бҹ’бһҹбҹҒбһ„вҖӢбһ–бһёвҖӢбһҖбһ¶бһҡвҖӢбһ бҹӮбһӣвҖӢбһҹбҹҒбһҡбһё бһҹбһјбһҳвҖӢбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶вҖӢбһ’бҹ’бһңбһҫвҖӢбһҸбҹҒбһҹбҹ’бһҸ CSS бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢвҖӢбһ“бҹғвҖӢбһҮбҹҶбһ„бһәвҖӢбһҠбһ¶бһ…бҹӢвҖӢбһҹбһҡбһҹбҹғвҖӢбһҲбһ¶бһҳвҖӢбһҖбҹ’бһ“бһ»бһ„вҖӢбһҒбһҪбһҡбһҖбҹ’бһ”бһ¶бһӣвҖӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһңбҹҒбһҮбҹ’бһҮбһ”бһүбҹ’бһҮбһ¶вҖӢбһҸбҹҶбһ”бһ“бҹӢвҖӢбһҠбҹӮбһӣвҖӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ” бһўбҹ’бһ“бһҖбһ бҹӮбһӣбһ‘бһ№бһҖбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ”бҹ’бһҡбһҫбһҸбҹҶбһ”бһ“бҹӢбһҹбҹҒбһҡбһё CSS бһ бһҫбһҷбһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһңбһҸбһ¶бһҳбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ IM/backstroke/breaststrokeбҹ”

бһҸбһҫвҖӢбһҸбҹҶбһ”бһ“бҹӢвҖӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„вҖӢбһ“бһ№бһ„вҖӢбһ–бһ·бһ“бҹ’бһ‘бһ»вҖӢбһҹбҹ’бһҸбҹ’бһҡбҹҒбһҹвҖӢбһҖбһ¶бһҡвҖӢбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ (sTSS) бһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…?
