FAQ

Pertanyaan yang Sering Diajukan

Bagaimana cara Analitik Renang mengambil data renang saya?

Analitik Renang disinkronkan dengan Apple Health untuk mengimpor latihan renang yang direkam oleh perangkat atau aplikasi apa pun yang kompatibel. Ini termasuk jam tangan pintar, pelacak kebugaran, dan entri manual. Aplikasi ini memproses data tersebut secara lokal untuk menghitung metrik tingkat lanjut.

Apa itu tes CSS dan bagaimana cara melakukannya?

CSS (*Critical Swim Speed*) adalah protokol ilmiah yang menggunakan 2 renang upaya maksimum: 400m dan 200m dengan istirahat 10-20 menit di antaranya. Aplikasi menghitung ambang batas aerobik Anda dari waktu-waktu tersebut dan secara otomatis menyesuaikan semua zona latihan. Ulangi setiap 6-8 minggu untuk melacak kemajuan Anda.

Apakah data saya diunggah ke cloud?

Tidak. Analitik Renang memproses semua data secara lokal di iPhone Anda. Tidak ada server eksternal, tidak ada akun cloud, tidak ada transfer data. Anda memegang kendali atas ekspor: hasilkan file JSON, CSV, HTML, atau PDF dan bagikan sesuka Anda.

Dapatkah saya menggunakan Analitik Renang untuk renang di perairan terbuka?

Ya. Analitik Renang berfungsi dengan latihan renang apa pun di Apple Health, termasuk perairan terbuka. Aplikasi beradaptasi dengan metrik yang tersedia baik saat Anda di kolam renang atau perairan terbuka, memberikan analisis yang relevan untuk setiap lingkungan.

Apa perbedaan antara paket bulanan dan tahunan?

Kedua paket menawarkan fitur yang identik: semua metrik, zona tak terbatas, perbandingan temporal, berbagai opsi ekspor, dan pembaruan gratis. Satu-satunya perbedaan adalah harga: paket tahunan menghemat 18% (setara dengan ‚Ç¨3,25/bulan vs ‚Ç¨3,99/bulan).

Dapatkah saya membatalkan langganan kapan saja?

Ya. Langganan dikelola melalui App Store, jadi Anda dapat membatalkannya kapan saja dari Pengaturan ‚Üí [Nama Anda] ‚Üí Langganan. Jika Anda membatalkan, Anda tetap memiliki akses hingga akhir periode penagihan saat ini.
