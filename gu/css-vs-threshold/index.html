ક્રિટિકલ સ્વિમ સ્પીડ (CSS) વિ. થ્રેશોલ્ડ પેસ: કયું એક મહત્ત્વનું છે?

વર્ષો સુધી, "થ્રેશોલ્ડ પેસ" (ટી-પેસ) એ સુવર્ણ ધોરણ હતું. 2026 માં, ક્રિટિકલ સ્વિમ સ્પીડ (CSS) પર શિફ્ટ એ ફરીથી વ્યાખ્યાયિત કરી રહ્યું છે કે કેવી રીતે એથ્લેટ્સ તેમના સત્રોને ગતિ આપે છે અને રેસ-ડેના થાકની આગાહી કરે છે.

🎯 ઝડપી જવાબ

ટી-પેસ (થ્રેશોલ્ડ પેસ) એ ગતિનો સામાન્ય અંદાજ છે જે તમે એક કલાક માટે જાળવી શકો છો, જેનો ઉપયોગ વ્યાપક તાલીમ ઝોન સેટ કરવા માટે થાય છે. તે "વ્યવહારિક" પરંતુ ઓછા દાણાદાર મેટ્રિક છે.

CSS (ક્રિટીકલ સ્વિમ સ્પીડ) બે અલગ અલગ અંતરો (દા.ત., 400m અને 200m) પરથી ગાણિતિક રીતે મેળવેલ "કાર્યકારી" થ્રેશોલ્ડ છે. તે સૈદ્ધાંતિક ગતિ છે જે તમે "દિવાલ" ને ફટકાર્યા વિના ટકાવી શકો છો. જો તમે સહનશક્તિ તરવૈયા અથવા ટ્રાયથ્લેટ છો, તો તમારા તાલીમ કાર્યક્રમમાં તમારો CSS સૌથી મહત્વપૂર્ણ નંબર છે.

2026 શિફ્ટ: પ્રયત્નોથી શરીરવિજ્ઞાન તરફ

પૂલમાં થાકનું સંચાલન કરવું સાઇકલિંગ અથવા દોડવાની સરખામણીમાં ખૂબ જ મુશ્કેલ છે. પાવર મીટર વિના, ગતિ એ તીવ્રતા માટે અમારી એકમાત્ર પ્રોક્સી છે. જ્યારે ઘણા તરવૈયાઓ હજુ પણ જૂની શાળાની 1500m "ટાઇમ-ટ્રાયલ" ગતિ પર આધાર રાખે છે, આધુનિક કોચિંગ બે-અંતરનું CSS મોડલ.

મુખ્ય તફાવત છે પ્રિસિઝન વિ. ધારેલા પ્રયત્નો. એક લાંબા પ્રયત્નો પર આધારિત ટી-પેસ ઘણીવાર તમારા મેટાબોલિક થ્રેશોલ્ડ કરતાં વધુ તમારી માનસિક સંવેદનાને પકડે છે. CSS, જે તમારા "પેસ-ડ્યુરેશન કર્વ" પરના બે અલગ-અલગ બિંદુઓથી ગણવામાં આવે છે, તે તમારા એન્જિનની વાસ્તવિક એરોબિક મર્યાદા દર્શાવે છે.

મેટ્રિક સરખામણી

થ્રેશોલ્ડ પેસ (ટી-પેસ)

"સરળ" મેટ્રિક

ટી-પેસ સામાન્ય રીતે એક જ 1500m અથવા 1000m મહત્તમ પ્રયત્નોમાંથી મેળવવામાં આવે છે. તે સતત તરવા માટે તમારી "રેડલાઇન" દર્શાવે છે.

  • ગુણ: સમજવામાં સરળ. એક ટેસ્ટ, એક નંબર. લાંબા, સતત સેટ માટે સરસ.
  • વિપક્ષ: પેસિંગ ભૂલો માટે અત્યંત સંવેદનશીલ. થાક હેઠળ તમે કેટલી ઝડપથી ઝાંખા પડો છો તેનો હિસાબ નથી. સમાન શારીરિક રૂપરેખા ધરાવતા દરેક તરવૈયાને સમાન 1500m ગતિ સાથે વર્તે છે.

ક્રિટિકલ સ્વિમ સ્પીડ (CSS)

"કાર્યકારી" મેટ્રિક

CSS એ બે અંતર (સામાન્ય રીતે 400m અને 200m મહત્તમ પ્રયાસ) પર આધારિત ગાણિતિક વ્યુત્પન્ન છે.

  • ગુણ: પેસિંગ પૂર્વગ્રહ દૂર કરે છે. અંતરાલોમાં તમારા વર્તનની વધુ સારી રીતે આગાહી કરે છે. એક ઉદ્દેશ્ય "એરોબિક ટોચમર્યાદા" પ્રદાન કરે છે.
  • વિપક્ષ: બે અલગ-અલગ ટેસ્ટની જરૂર છે. થોડું વધુ જટિલ ગણિત (સ્વિમ ઍનલિટિક્સમાં સ્વતઃ ગણતરી). સતત આરામ સાથે પરીક્ષણો કરવાની જરૂર છે.

શા માટે ડેલ્ટા બાબતો

"ડેલ્ટા" એ તમારી 400m ગતિ અને તમારી 200m ગતિ વચ્ચેનો તફાવત છે. બે તરવૈયાઓ બંનેની ટી-પેસ 1:40/100m હોઈ શકે છે, પરંતુ તેમની CSS પ્રોફાઇલ આના જેવી દેખાઈ શકે છે:

સ્વિમર એ ("ધ ડીઝલ")

200m અને 400m વચ્ચેનો નાનો ડેલ્ટા. આ તરવૈયાની એરોબિક કાર્યક્ષમતા ખૂબ ઊંચી છે પરંતુ સ્પ્રિન્ટની ઝડપ ઓછી છે. તેમનું CSS તેમના T-Paceની ખૂબ નજીક છે.

તરવૈયા બી ("ધ દોડવીર")

વિશાળ ડેલ્ટા. મહાન 200m ઝડપ પરંતુ 400m પર નોંધપાત્ર રીતે ઘટી જાય છે. તેમનું CSS તેમના T-Pace સૂચવે છે તેના કરતા ઓછું હશે. જો તેઓ સામાન્ય "ટી-પેસ" પર તાલીમ આપવાનો પ્રયાસ કરે છે, તો તેઓ સતત તેમની એનારોબિક સિસ્ટમને વધુ પડતી રાંધશે.

📈 CSS ફોર્મ્યુલા

CSS = (D1 - D2) / (T1 - T2)

જ્યાં D1/D2 એ બે અંતર છે અને T1/T2 એ સેકન્ડમાં સમય છે. આ તમારા પ્રારંભિક એનારોબિક ઉછાળાના ઘોંઘાટ વિના તમારી ટકાઉ ગતિનો "ઢાળ" દર્શાવે છે.

CSS ટેસ્ટ કેવી રીતે કરવી (સ્ટેપ-બાય-સ્ટેપ)

તમારા CSSની ગણતરી કરવા માટે, તમારે પ્રમાણભૂત પૂલમાં અલગ-અલગ દિવસોમાં (અથવા તેની વચ્ચે સંપૂર્ણ પુનઃપ્રાપ્તિ સાથે) બે મહત્તમ અજમાયશ કરવાની જરૂર છે:

  1. વોર્મ-અપ: તમારા શરીરને તૈયાર કરવા માટે સંપૂર્ણ, પ્રમાણિત વોર્મ-અપ (સામાન્ય રીતે 400m થી 600m મિશ્ર સ્ટ્રોક અને બિલ્ડ્સ) કરો.
  2. ટ્રાયલ 1 (400m મહત્તમ પ્રયાસ): શક્ય તેટલી ઝડપથી 400 મીટર તરવું. ચોક્કસ સમય રેકોર્ડ કરો. તમારી જાતને ગતિ આપો જેથી તમે પ્રથમ 100 મીટરમાં બળી ન જાઓ.
  3. સક્રિય પુનઃપ્રાપ્તિ: લેક્ટિક એસિડને બહાર કાઢવા માટે 100m થી 200m સુધી તરવું ખૂબ જ સરળ છે, ત્યારબાદ 5-10 મિનિટનો આરામ કરો.
  4. ટ્રાયલ 2 (200m મહત્તમ પ્રયાસ): સંપૂર્ણ મહત્તમ પ્રયત્નો પર 200 મીટર તરવું. ચોક્કસ સમય રેકોર્ડ કરો.
  5. ગણતરી કરો: તમારા સમયને સેકન્ડમાં રૂપાંતરિત કરો અને ઉપરોક્ત સૂત્ર લાગુ કરો અથવા તમારી 100m પેસિંગ ગતિ મેળવવા માટે સ્વિમ ઍનલિટિક્સમાં CSS કેલ્ક્યુલેટર ટૂલનો ઉપયોગ કરો.

તમારે કયો ઉપયોગ કરવો જોઈએ?

થ્રેશોલ્ડ પેસ (ટી-પેસ) નો ઉપયોગ કરો જો:

  • તમે માત્ર ગતિ-આધારિત તાલીમ સાથે પ્રારંભ કરી રહ્યાં છો.
  • તમે માત્ર લાંબા, સતત સ્વિમિંગ કરો છો (ઓપન વોટર 3km+).
  • તમે બહુવિધ પરીક્ષણ સત્રો સાથે વ્યવહાર કરવા માંગતા નથી.

ક્રિટિકલ સ્વિમ સ્પીડ (CSS) નો ઉપયોગ કરો જો:

  • ટ્રાયથ્લેટ્સ અને મેરેથોન તરવૈયાઓ: રેસમાં ખૂબ વહેલા "મીણબત્તી બાળી" ટાળવા માટે.
  • અંતરાલ તાલીમ: ચોક્કસ CSS + 2s અથવા CSS - 1s ઝોન સેટ કરવા જે ખરેખર કામ કરે છે.
  • ટ્રેકિંગ પ્રોગ્રેસ: CSS સામાન્ય 1000m પરીક્ષણ કરતાં એરોબિક સુધારણા માટે વધુ સંવેદનશીલ છે.

થ્રેશોલ્ડ પેસ વિ CSS FAQ

શું સીએસએસ થ્રેશોલ્ડ ગતિ સમાન છે?

ના. તે સંબંધિત થ્રેશોલ્ડ ખ્યાલો છે, પરંતુ તે સમાન પરીક્ષણ નથી. થ્રેશોલ્ડ ગતિ સામાન્ય રીતે એક લાંબા તરીને આવે છે, જ્યારે CSS ની ગણતરી બે પ્રયત્નોથી કરવામાં આવે છે અને અંતરાલ કાર્ય માટે વધુ પુનરાવર્તિત એન્કર આપે છે.

ટ્રાયથલોન અને ઓપન-વોટર પેસિંગ માટે કયું સારું છે?

CSS એ સામાન્ય રીતે બહેતર પ્રારંભિક બિંદુ છે કારણ કે તે ટૂંકા ગાળાની ઝડપને વધુ લાભદાયી કર્યા વિના ટકાઉ ગતિને પ્રતિબિંબિત કરે છે. થ્રેશોલ્ડ ગતિ હજુ પણ લાંબા સ્થિર તરવા માટે ઉપયોગી હોઈ શકે છે, પરંતુ CSS સામાન્ય રીતે પેસિંગ લક્ષ્યો અને ઝોન કાર્યમાં રૂપાંતરિત કરવું સરળ છે.

હું થ્રેશોલ્ડ ગતિથી CSS-આધારિત સ્વિમ ઝોનમાં કેવી રીતે જઈ શકું?

CSS સેટ કરવા માટે તમારી આગામી 400m અને 200m ટેસ્ટનો ઉપયોગ કરો, પછી તે નંબરની આસપાસ એન્કર ઇન્ટરવલ વર્ક કરો. જો તમે પહેલાથી જ ટી-પેસ દ્વારા તાલીમ લીધી હોય, તો બંને નંબરોની સરખામણી કરો અને પુનરાવર્તિત એરોબિક ગતિની આસપાસ તમારા સેટને ફરીથી બનાવવા માટે CSS માર્ગદર્શિકા વત્તા સ્વિમ તાલીમ ઝોનનો ઉપયોગ કરો.

તમારા પૂલ ઝોનનું અનુમાન લગાવવાનું બંધ કરો.
