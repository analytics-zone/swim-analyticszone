Wat is Zwem TSS (sTSS)?

Swimming Training Stress Score (sTSS) kwantificeert de trainingslast van een zwemsessie door intensiteit en duur te combineren. Het is aangepast van de wielrennen TSS methodologie, waarbij je Critical Swim Speed (CSS) gebruikt wordt als drempeltempo. Een training van 1 uur op CSS tempo = 100 sTSS.

Trainingslast Interpretatie

Hoe sTSS Berekend Wordt

Formule

sTSS = (Duur in uren) × (Intensity Factor)² × 100

Waarbij:

  • Intensity Factor (IF) = CSS Tempo / Gemiddeld Trainingstempo
  • Duur = Totale trainingstijd in uren
  • CSS Tempo = Je drempeltempo van CSS test

Praktisch Voorbeeld

Trainingsdetails:

  • CSS Tempo: 1:49/100m (109 seconden)
  • Trainingsduur: 60 minuten (1 uur)
  • Gemiddeld Tempo: 2:05/100m (125 seconden)

Stap 1: Bereken Intensity Factor

IF = CSS Tempo / Trainingstempo
IF = 109 / 125
IF = 0.872

Stap 2: Bereken sTSS

sTSS = 1.0 uur × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

Interpretatie: Deze 60 minuten training op rustig tempo (langzamer dan CSS) genereerde 76 sTSS - een gematigde trainingslast geschikt voor het opbouwen van aërobe basis.

sTSS Waarden Interpreteren

sTSS Bereik Trainingslast Hersteltijd Voorbeeldtraining
< 50 Laag Zelfde dag 30 min rustig zwemmen, technische oefeningen
50-100 Gematigd 1 dag 60 min duur, steady tempo
100-200 Hoog 1-2 dagen 90 min drempel sets, wedstrijdtempo intervallen
200-300 Zeer Hoog 2-3 dagen 2 uur harde training, meerdere drempel blokken
> 300 Extreem 3+ dagen Lange wedstrijd (>2 uur), ultra-duur

Wekelijkse sTSS Gids

Doel wekelijkse sTSS hangt af van je trainingsniveau en doelen:

Recreatieve Zwemmers

Wekelijkse sTSS: 150-300

2-3 trainingen per week, 50-100 sTSS elk. Focus op techniek en aërobe basis opbouw.

Fitness Zwemmers / Triathleten

Wekelijkse sTSS: 300-500

3-4 trainingen per week, 75-125 sTSS elk. Mix van aërobe duur en drempelwerk.

Masters Competitieve Zwemmers

Wekelijkse sTSS: 500-800

4-6 trainingen per week, 80-150 sTSS elk. Gestructureerde training met periodisering.

Elite / Universiteit Zwemmers

Wekelijkse sTSS: 800-1200+

8-12 trainingen per week, dubbele sessies. Hoog volume met kritisch herstelmanagement.

⚠️ Belangrijke Opmerkingen

  • Vereist accurate CSS: Je CSS moet up-to-date zijn (getest binnen 6-8 weken) voor nauwkeurige sTSS.
  • Vereenvoudigde berekening: Deze calculator gebruikt gemiddeld tempo. Geavanceerde sTSS gebruikt Normalized Graded Pace (NGP) die intervalstructuur meeneemt.
  • Niet voor technisch werk: sTSS meet alleen fysieke trainingsstress, niet vaardigheidsontwikkeling.
  • Individuele variatie: Dezelfde sTSS voelt anders voor verschillende zwemmers. Pas richtlijnen aan op je herstel.

Waarom sTSS Belangrijk Is

Training Stress Score is de basis voor:

  • CTL (Chronic Training Load): Je fitnessniveau - exponentieel gewogen 42-daags gemiddelde van dagelijkse sTSS
  • ATL (Acute Training Load): Je vermoeidheid - exponentieel gewogen 7-daags gemiddelde van dagelijkse sTSS
  • TSB (Training Stress Balance): Je vorm toestand - TSB = CTL - ATL (positief = fris, negatief = vermoeid)
  • Periodisering: Trainingsfasen plannen (basis, ontwikkeling, piek, taper) met doel CTL progressies
  • Herstelmanagement: Weten wanneer te pushen en wanneer te rusten gebaseerd op TSB

Pro Tip: Volg je CTL

Log dagelijkse sTSS in een spreadsheet of trainingsdagboek. Bereken je 42-daags gemiddelde (CTL) wekelijks. Doel voor 5-10 punt CTL stijging per week tijdens basis opbouw. Handhaaf of verlaag CTL licht tijdens taper (1-2 weken voor wedstrijd).

Veelgestelde Vragen

Wat is Zwem TSS (sTSS)?

Swimming Training Stress Score (sTSS) is een metriek die de trainingslast van een zwemsessie kwantificeert door zowel intensiteit als duur te combineren. Het is aangepast van de wielrennen TSS methodologie, waarbij je Critical Swim Speed (CSS) wordt gebruikt als drempeltempo. Een training van 1 uur op CSS tempo is gelijk aan 100 sTSS.

Hoe bereken ik mijn sTSS?

Gebruik de calculator hierboven door je CSS tempo (van een CSS test), totale trainingsduur en gemiddelde tempo tijdens de training in te voeren. De formule is: sTSS = Duur (uren) × Intensity Factor² × 100, waarbij Intensity Factor = CSS Tempo / Gemiddeld Trainingstempo.

Heb ik CSS nodig om sTSS te berekenen?

Ja, je Critical Swim Speed (CSS) is vereist om de Intensity Factor te berekenen, wat essentieel is voor sTSS berekening. CSS vertegenwoordigt je drempeltempo en moet elke 6-8 weken getest worden. Je kunt je CSS vinden met onze CSS calculator.

Wat is een goede sTSS score voor één training?

Het hangt af van de trainingsintensiteit: Rustige trainingen scoren meestal onder 50 sTSS, gematigde trainingen 50-100 sTSS, harde trainingen 100-200 sTSS, en zeer harde trainingen boven 200 sTSS. De juiste score hangt af van je trainingsdoelen en huidige fitnessniveau.

Hoeveel sTSS moet ik per week doen?

Wekelijkse sTSS doelen variëren per niveau: Recreatieve zwemmers: 150-300, Fitness zwemmers/Triathleten: 300-500, Masters Competitieve zwemmers: 500-800, Elite/Universiteit: 800-1200+. Begin conservatief en bouw geleidelijk op om overtraining te voorkomen.

Is zwem-TSS hetzelfde als fiets-TSS?

Het concept en de formule zijn hetzelfde, maar sTSS is aangepast voor zwemmen. In plaats van vermogen (FTP) te gebruiken zoals fiets-TSS, gebruikt sTSS tempo met CSS als drempel. Beide meten trainingsbelasting met Duur × Intensiteitsfactor² × 100.

Kan ik sTSS gebruiken voor alle zwemstijlen?

Ja, maar je CSS moet stijlspecifiek zijn. De meeste zwemmers gebruiken vrije slag CSS omdat dit de meest getrainde slag is. Als je voornamelijk in een andere slag traint, voer dan een CSS-test uit in die slag en gebruik dat tempo voor sTSS-berekening.

Wat is het verschil tussen sTSS en CTL/ATL/TSB?

sTSS meet de trainingsbelasting van één training. CTL (Chronic Training Load) is je langetermijnfitness, ATL (Acute Training Load) is je recente vermoeidheid, en TSB (Training Stress Balance) is je frisheid. Deze statistieken gebruiken sTSS-waarden over tijd om je trainingsstatus te volgen. Lees meer in onze Trainingsbelasting gids.
