Бесплатный Калькулятор TSS для Плавания

Рассчитайте Training Stress Score для тренировок по плаванию - Единственный бесплатный калькулятор sTSS

Что такое sTSS (TSS для Плавания)?

Swimming Training Stress Score (sTSS) количественно определяет тренировочную нагрузку тренировки по плаванию, объединяя интенсивность и продолжительность. Это адаптация методологии TSS из велоспорта с использованием вашей Critical Swim Speed (CSS) в качестве порогового темпа. Тренировка в течение 1 часа в темпе CSS = 100 sTSS.

Бесплатный Калькулятор sTSS

Интерпретация Тренировочной Нагрузки

Как Рассчитывается sTSS

Формула

sTSS = (Продолжительность в часах) × (Фактор Интенсивности)² × 100

Где:

  • Фактор Интенсивности (IF) = Темп CSS / Средний Темп Тренировки
  • Продолжительность = Общее время тренировки в часах
  • Темп CSS = Ваш пороговый темп из теста CSS

Практический Пример

Детали Тренировки:

  • Темп CSS: 1:49/100м (109 секунд)
  • Продолжительность Тренировки: 60 минут (1 час)
  • Средний Темп: 2:05/100м (125 секунд)

Шаг 1: Рассчитать Фактор Интенсивности

IF = Темп CSS / Темп Тренировки
IF = 109 / 125
IF = 0.872

Шаг 2: Рассчитать sTSS

sTSS = 1.0 часа × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

Интерпретация: Эта 60-минутная тренировка в легком темпе (медленнее CSS) создала 76 sTSS - умеренную тренировочную нагрузку, подходящую для построения аэробной базы.

Понимание Значений sTSS

Диапазон sTSS Тренировочная Нагрузка Время Восстановления Пример Тренировки
< 50 Низкая В тот же день Легкое плавание 30 мин, технические упражнения
50-100 Умеренная 1 день Выносливость 60 мин, стабильный темп
100-200 Высокая 1-2 дня Пороговые серии 90 мин, интервалы в темпе соревнований
200-300 Очень Высокая 2-3 дня Интенсивная тренировка 2 часа, несколько пороговых блоков
> 300 Экстремальная 3+ дня Длинный заплыв (>2 часов), ультравыносливость

Руководство по Недельному sTSS

Целевой недельный sTSS зависит от вашего уровня тренировок и целей:

Любители Плавания

Недельный sTSS: 150-300

2-3 тренировки в неделю, 50-100 sTSS каждая. Фокус на технике и построении аэробной базы.

Фитнес-Пловцы / Триатлеты

Недельный sTSS: 300-500

3-4 тренировки в неделю, 75-125 sTSS каждая. Смесь аэробной выносливости и пороговой работы.

Соревновательные Мастера

Недельный sTSS: 500-800

4-6 тренировок в неделю, 80-150 sTSS каждая. Структурированные тренировки с периодизацией.

Элитные / Университетские Пловцы

Недельный sTSS: 800-1200+

8-12 тренировок в неделю, двойные тренировки. Высокий объем с критическим управлением восстановлением.

⚠️ Важные Заметки

  • Требуется точный CSS: Ваш CSS должен быть актуальным (протестирован в течение 6-8 недель) для точного sTSS.
  • Упрощенный расчет: Этот калькулятор использует средний темп. Продвинутый sTSS использует Normalized Graded Pace (NGP), который учитывает структуру интервалов.
  • Не для технической работы: sTSS измеряет только физический тренировочный стресс, а не развитие навыков.
  • Индивидуальная вариация: Одинаковый sTSS ощущается по-разному для разных пловцов. Корректируйте руководящие принципы в соответствии с вашим восстановлением.

Почему sTSS Важен

Training Stress Score является основой для:

  • CTL (Chronic Training Load): Ваш уровень физической формы - экспоненциально взвешенное 42-дневное среднее дневного sTSS
  • ATL (Acute Training Load): Ваша усталость - экспоненциально взвешенное 7-дневное среднее дневного sTSS
  • TSB (Training Stress Balance): Ваша форма - TSB = CTL - ATL (положительный = свежий, отрицательный = усталый)
  • Периодизация: Планирование тренировочных фаз (база, развитие, пик, снижение нагрузки) с использованием целевых прогрессий CTL
  • Управление Восстановлением: Знание, когда нажимать и когда отдыхать на основе TSB

Профессиональный Совет: Отслеживайте Ваш CTL

Записывайте дневной sTSS в таблицу или тренировочный журнал. Рассчитывайте ваше 42-дневное среднее (CTL) еженедельно. Целью является увеличение CTL на 5-10 пунктов в неделю во время построения базы. Поддерживайте или слегка снижайте CTL во время снижения нагрузки (1-2 недели перед соревнованиями).

Часто Задаваемые Вопросы

Что такое sTSS (TSS для Плавания)?

Swimming Training Stress Score (sTSS) количественно определяет тренировочную нагрузку тренировки по плаванию, объединяя интенсивность и продолжительность. Это адаптация методологии TSS из велоспорта с использованием вашей Critical Swim Speed (CSS) в качестве порогового темпа. Тренировка в течение 1 часа в темпе CSS равна 100 sTSS.

Как мне рассчитать свой sTSS?

Используйте калькулятор выше, введя ваш темп CSS (из теста CSS), общую продолжительность тренировки и средний темп во время тренировки. Формула: sTSS = Продолжительность (часы) × Фактор Интенсивности² × 100, где Фактор Интенсивности = Темп CSS / Средний Темп Тренировки.

Нужен ли мне CSS для расчета sTSS?

Да, ваша Critical Swim Speed (CSS) необходима для расчета Фактора Интенсивности, который важен для расчета sTSS. CSS представляет собой ваш пороговый темп и должен тестироваться каждые 6-8 недель. Вы можете найти свой CSS, используя наш калькулятор CSS.

Какой хороший показатель sTSS для одной тренировки?

Это зависит от интенсивности тренировки: легкие тренировки обычно набирают менее 50 sTSS, умеренные тренировки 50-100 sTSS, тяжелые тренировки 100-200 sTSS, а очень тяжелые тренировки более 200 sTSS. Подходящий показатель зависит от ваших тренировочных целей и текущего уровня физической подготовки.

Сколько sTSS я должен делать в неделю?

Целевые показатели недельного sTSS варьируются в зависимости от уровня: Любители плавания: 150-300, Фитнес-пловцы/Триатлеты: 300-500, Соревновательные мастера: 500-800, Элита/Университетские пловцы: 800-1200+. Начинайте консервативно и увеличивайте постепенно, чтобы избежать перетренированности.

Плавательный TSS такой же, как велосипедный TSS?

Концепция и формула одинаковы, но sTSS адаптирован для плавания. Вместо использования мощности (FTP), как велосипедный TSS, sTSS использует темп с CSS в качестве порога. Оба измеряют тренировочную нагрузку, используя Продолжительность × Фактор Интенсивности² × 100.

Могу ли я использовать sTSS для всех стилей плавания?

Да, но ваш CSS должен быть специфичным для стиля. Большинство пловцов используют CSS вольного стиля, поскольку это наиболее часто тренируемый стиль. Если вы в основном тренируетесь в другом стиле, выполните тест CSS в этом стиле и используйте этот темп для расчета sTSS.

В чем разница между sTSS и CTL/ATL/TSB?

sTSS измеряет тренировочную нагрузку одной тренировки. CTL (Chronic Training Load) — это ваша долгосрочная форма, ATL (Acute Training Load) — ваша недавняя усталость, а TSB (Training Stress Balance) — ваша свежесть. Эти показатели используют значения sTSS с течением времени для отслеживания вашего тренировочного состояния. Узнайте больше в нашем руководстве по Тренировочной Нагрузке.
