Kritiskais peldēšanas ātrums (CSS)

Uz datiem balstītas peldēšanas apmācības pamats

Kas ir kritiskais peldēšanas ātrums (CSS)?

Kritiskais peldēšanas ātrums (CSS)ir teorētiskais maksimālais peldēšanas ātrums, ko varat uzturēt bez pārguruma. Tas atspoguļo jūsu aerobā sliekšņa tempu, kas parasti atbilst 4 mmol/l laktātam asinīs un ir noturīgs aptuveni 30 minūtes. CSS tiek aprēķināts, izmantojot 400 m un 200 m laika izmēģinājumu, lai noteiktu personalizētas treniņu zonas.

Kritiskais peldēšanas ātrums (CSS) ir teorētiskais maksimālais peldēšanas ātrums, ko varat uzturēt nepārtraukti bez pārguruma. Tas ir jūsu aerobais slieksnis ūdenī — intensitāte, kurā laktāta ražošana ir vienāda ar laktāta klīrensu.

🎯 Fizioloģiskā nozīme

CSS cieši atbilst:

  • Laktāta slieksnis 2 (LT2)- Otrais ventilācijas slieksnis
  • Maksimālais laktāta līdzsvara stāvoklis (MLSS)- Augstākais ilgtspējīgais laktāta līmenis
  • Funkcionālais sliekšņa temps (FTP)- Peldēšana līdzvērtīga riteņbraukšanai FTP
  • ~4 mmol/L laktāts asinīs- Tradicionālais OBLA marķieris

Kāpēc CSS ir svarīga

CSS irpamata metrikakas atbloķē visu uzlaboto treniņu slodzes analīzi:

  • Treniņu zonas:Personalizē intensitātes zonas, pamatojoties uz jūsu fizioloģiju
  • sTSS aprēķins:Ļauj precīzi noteikt treniņu stresa rādītājus
  • CTL/ATL/TSB:Nepieciešams veiktspējas pārvaldības diagrammas metrikai
  • Progresa izsekošana:Objektīvs aerobās sagatavotības uzlabošanas pasākums
⚠️ Kritiskā atkarība:Bez derīga CSS testa uzlabotas treniņu slodzes metriku (sTSS, CTL, ATL, TSB) aprēķināt nevar. Neprecīza CSS sabojās visu turpmāko apmācību analīzi.

Kā aprēķināt CSS 4 vienkāršās darbībās

📋 Standarta protokols

  1. Iesildīšanās

    300–800 m viegla peldēšana, vingrinājumi un pakāpeniska slodze, lai sagatavotos maksimālai piepūlei.

  2. 400 m Time Trial

    Maksimālā ilgstoša piepūle no stumšanas sākuma (bez niršanas). Ierakstiet laiku līdz sekundei. Mērķis: ātrākais ilgtspējīgais 400 m.

  3. Pilnīga atveseļošanās

    5-10 minūtesviegla peldēšana vai pilnīga atpūta. Tas ir KRITiski svarīgi precīziem rezultātiem.

  4. 200 m Time Trial

    Maksimālā piepūle no stumšanas sākuma. Precīzi ierakstiet laiku. Tam vajadzētu būt ātrākam uz 100 m nekā 400 m.

🔄 Atkārtotas pārbaudes biežums

Katru reizi atkārtoti pārbaudiet CSS6-8 nedēļaslai atjauninātu treniņu zonas, uzlabojoties fitnesam. Pielāgojoties treniņam, jūsu zonām vajadzētu pakāpeniski kļūt ātrākām.

CSS aprēķina formula

Formula

CSS (m/s) = (D2–D₁) / (T2–T₁)

Kur:

  • D₁ = 200 metri
  • D₂ = 400 metri
  • T₁ = laiks 200 m (sekundēs)
  • T₂ = laiks 400 m (sekundēs)

Vienkāršots tempam uz 100 m

CSS temps/100 m (sekundes) = (T₄00 - T₂0₀) / 2

Nostrādāts piemērs

Testa rezultāti:

  • 400 m laiks:6:08(368 sekundes)
  • 200 m laiks:2:30(150 sekundes)

🎯 Zonās balstītas apmācības priekšrocības

Izmantojot CSS balstītas zonas, subjektīvie "jūtas" treniņi tiek pārveidoti par objektīviem, atkārtojamiem treniņiem. Katra zona ir vērsta uz īpašiem fizioloģiskiem pielāgojumiem:

  • 2. zona:Izveidojiet aerobo dzinēju (60–70% no nedēļas apjoma)
  • 3. zona:Uzlabojiet sacensību tempa efektivitāti (15-20% no apjoma)
  • 4. zona:Palieliniet laktāta slieksni (10-15% no tilpuma)
  • 5. zona:Attīstīt augstākā līmeņa ātrumu un jaudu (5–10% no skaļuma)

Zinātniskā apstiprināšana

Wakayoshi et al. (1992-1993) - Pamatpētījums

Kohji Wakayoshi pamatpētījumi Osakas universitātē noteica CSS kā derīgu, praktisku alternatīvu laboratorijas laktāta testēšanai:

  • Spēcīga korelācija ar VO₂ pie anaerobā sliekšņa(r = 0,818)
  • Lieliska korelācija ar ātrumu pie OBLA(r = 0,949)
  • Prognozē 400m veiktspēju(r = 0,864)
  • Atbilst 4 mmol/l laktātam asinīs- maksimālais laktāta līdzsvara stāvoklis
  • Lineāra attiecība starp attālumu un laiku(r² > 0,998)

🔬 Kāpēc CSS darbojas

CSS ir robeža starpsmagas un smagas slodzes jomās. Zem CSS laktāta ražošana un klīrenss paliek līdzsvaroti — jūs varat peldēt ilgu laiku. Virs CSS laktāts uzkrājas pakāpeniski līdz izsīkumam 20–40 minūšu laikā.

Tas padara CSS par ideālu intensitāti:

  • Ilgtspējīga sacensību tempa iestatīšana 800–1500 m distancēm
  • Sliekšņa intervāla treniņu izrakstīšana
  • Aerobās sagatavotības uzlabojumu uzraudzība
  • Treniņu slodzes un atveseļošanās vajadzību aprēķināšana

Kā pārbaudīt savu CSS

1. darbība: pareizi iesildieties

Veiciet 300–800 metrus vieglas peldēšanas, ieskaitot treniņus un progresīvus vingrinājumus. Tas sagatavo jūsu ķermeni maksimālai piepūlei un novērš traumas.

2. darbība: veiciet 400 m laika izmēģinājumu

Peldiet 400 metrus ar maksimālu noturīgu piepūli no grūšanas sākuma (bez niršanas). Pierakstiet savu laiku līdz sekundei. Mērķējiet uz vienmērīgu sadalījumu vai negatīvu sadalījumu (otrie 200 m ir vienādi vai ātrāki par pirmajiem 200 m).

3. darbība. Pilnīga atkopšana

Atpūtieties 5-10 minūtes ar vieglu peldēšanu vai pilnīgu atpūtu. Šī atkopšana ir KRITISKI. Pirms turpināt, pagaidiet, līdz sirdsdarbība nokrītas zem 120 sitieniem minūtē un elpošana ir pilnībā atjaunota.

4. darbība: veiciet 200 m laika izmēģinājumu

Peldējiet 200 metrus ar maksimālu piepūli no grūšanas sākuma. Precīzi pierakstiet savu laiku. Tam vajadzētu būt ātrākam uz 100 m nekā jūsu 400 m tempam.

5. darbība. Aprēķiniet savu CSS

Izmantojiet formulu: CSS Pace/100m = (T400 - T200) / 2. Piemērs: (368s - 150s) / 2 = 109s = 1:49/100m. Vai arī izmantojiet šajā lapā esošo kalkulatoru, lai iegūtu tūlītējus rezultātus ar personalizētām treniņu zonām.

Praktiski pielietojumi

1️⃣ Atbloķējiet treniņu slodzes metriku

CSS ir saucējs sTSS intensitātes faktora aprēķinā. Bez tā jūs nevarat kvantitatīvi noteikt treniņu stresu vai izsekot fitnesa/noguruma tendencēm.

2️⃣ Personalizējiet treniņu zonas

Vispārējās tempa diagrammas neņem vērā individuālo fizioloģiju. Uz CSS balstītas zonas nodrošina, ka katrs peldētājs trenējas optimālā intensitātē.

3️⃣ Pārraugiet fitnesa progresu

Atkārtoti pārbaudiet ik pēc 6-8 nedēļām. CSS (ātrāka tempa) uzlabošana liecina par veiksmīgu aerobikas adaptāciju. Stagnējošais CSS iesaka pielāgot apmācības vajadzības.

4️⃣ Paredzēt sacīkšu sniegumu

CSS temps aptuveni atbilst jūsu ilgtspējīgajam 30 minūšu sacensību tempam. Izmantojiet to, lai uzstādītu reālus mērķus 800 m, 1500 m un atklātā ūdens notikumiem.

5️⃣ Dizaina sliekšņa treniņi

Klasiskie CSS komplekti: 8×100 @ CSS temps (15s atpūta), 5×200 @ 101% CSS (20s atpūta), 3×400 @ 103% CSS (30s atpūta). Palieliniet laktāta klīrensa spēju.

6️⃣ Optimizējiet konusveida stratēģiju

Izsekot CSS pirms un pēc sašaurinājuma. Veiksmīgs konuss saglabā vai nedaudz uzlabo CSS, vienlaikus samazinot nogurumu (palielināts TSB).

Izmantojiet savas CSS zināšanas

Tagad, kad esat sapratis kritisko peldēšanas ātrumu, veiciet šādas darbības, lai optimizētu treniņu:

  • Aprēķināt treniņa stresa rādītāju (TSS)pamatojoties uz jūsu CSS, lai noteiktu treniņa intensitāti
  • Izpētiet 7 treniņu zonasun uzziniet, kā strukturēt treniņus īpašiem pielāgojumiem
  • Skatīt visas formulasizmanto Swim Analytics, lai nodrošinātu pilnīgu pārskatāmību
  • Lejupielādēt Swim Analyticslai automātiski izsekotu CSS, TSS un veiktspējas tendences
