مفت تیراکی TSS کیلکولیٹر

تیراکی کی ورزشوں کے لیے ٹریننگ اسٹریس اسکور کا حساب لگائیں - واحد مفت sTSS کیلکولیٹر

تیراکی TSS (sTSS) کیا ہے؟

سویمنگ ٹریننگ اسٹریس اسکور (sTSS) شدت اور دورانیہ کو ملا کر تیراکی کی ورزش کے ٹریننگ لوڈ کی مقدار کا تعین کرتا ہے۔ یہ سائیکلنگ کے TSS طریقہ کار سے اخذ کیا گیا ہے، جو آپ کی کرٹیکل سویم اسپیڈ (CSS) کو تھریشولڈ رفتار کے طور پر استعمال کرتا ہے۔ CSS رفتار پر 1 گھنٹے کی ورزش = 100 sTSS۔

مفت sTSS کیلکولیٹر

کسی بھی تیراکی کے سیشن کے لیے تربیتی تناؤ کا حساب لگائیں۔ آپ کی CSS رفتار کی ضرورت ہے۔

CSS ٹیسٹ سے آپ کی تھریشولڈ رفتار (مثلاً، 1:49)
آرام سمیت کل ورزش کا وقت (1-300 منٹ)
ورزش کے دوران آپ کی اوسط رفتار (مثلاً، 2:05)

ٹریننگ لوڈ کی تشریح

sTSS کا حساب کیسے لگایا جاتا ہے

فارمولا

sTSS = (گھنٹوں میں دورانیہ) × (شدت کا عنصر)² × 100

جہاں:

  • شدت کا عنصر (IF) = CSS رفتار / اوسط ورزش کی رفتار
  • دورانیہ = گھنٹوں میں کل ورزش کا وقت
  • CSS رفتار = CSS ٹیسٹ سے آپ کی تھریشولڈ رفتار

حل شدہ مثال

ورزش کی تفصیلات:

  • CSS رفتار: 1:49/100m (109 سیکنڈ)
  • ورزش کا دورانیہ: 60 منٹ (1 گھنٹہ)
  • اوسط رفتار: 2:05/100m (125 سیکنڈ)

مرحلہ 1: شدت کے عنصر کا حساب لگائیں

IF = CSS رفتار / ورزش کی رفتار
IF = 109 / 125
IF = 0.872

مرحلہ 2: sTSS کا حساب لگائیں

sTSS = 1.0 گھنٹے × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

تشریح: آسان رفتار (CSS سے سست) پر اس 60 منٹ کی ورزش نے 76 sTSS پیدا کیا - ایروبک بیس بنانے کے لیے موزوں ایک اعتدال پسند ٹریننگ لوڈ۔

sTSS اقدار کو سمجھنا

sTSS رینجٹریننگ لوڈبحالی کا وقتمثال ورزش
< 50کماسی دنآسان 30 منٹ کی تیراکی، تکنیک کی مشقیں
50-100اعتدال پسند1 دن60 منٹ برداشت، مستحکم رفتار
100-200زیادہ1-2 دن90 منٹ تھریشولڈ سیٹ، ریس کی رفتار کے وقفے
200-300بہت زیادہ2-3 دن2 گھنٹے سخت ٹریننگ، متعدد تھریشولڈ بلاکس
> 300انتہائی3+ دنلمبی ریس (>2 گھنٹے)، الٹرا برداشت

ہفتہ وار sTSS رہنما خطوط

ہدف ہفتہ وار sTSS آپ کی ٹریننگ کی سطح اور اہداف پر منحصر ہے:

تفریحی تیراک

ہفتہ وار sTSS: 150-300

ہفتے میں 2-3 ورزشیں، ہر ایک 50-100 sTSS۔ تکنیک اور ایروبک بیس بنانے پر توجہ دیں۔

فٹنس تیراک / ٹرائیتھلیٹس

ہفتہ وار sTSS: 300-500

ہفتے میں 3-4 ورزشیں، ہر ایک 75-125 sTSS۔ ایروبک برداشت اور تھریشولڈ کام کا مرکب۔

مسابقتی ماسٹرز تیراک

ہفتہ وار sTSS: 500-800

ہفتے میں 4-6 ورزشیں، ہر ایک 80-150 sTSS۔ مدت کے ساتھ منظم ٹریننگ۔

ایلیٹ / کالجیٹ تیراک

ہفتہ وار sTSS: 800-1200+

ہفتے میں 8-12 ورزشیں، ڈبل دن۔ زیادہ حجم اور بحالی کا انتظام اہم ہے۔

⚠️ اہم نوٹس

  • درست CSS کی ضرورت ہے: درست sTSS کے لیے آپ کی CSS موجودہ (6-8 ہفتوں کے اندر ٹیسٹ کی گئی) ہونی چاہیے۔
  • آسان حساب کتاب: یہ کیلکولیٹر اوسط رفتار کا استعمال کرتا ہے۔ اعلی درجے کی sTSS نارملائزڈ گریڈڈ پیس (NGP) کا استعمال کرتی ہے جو وقفہ کی ساخت کا حساب رکھتی ہے۔
  • تکنیک کے کام کے لیے نہیں: sTSS صرف جسمانی ٹریننگ کے دباؤ کی پیمائش کرتا ہے، مہارت کی نشوونما نہیں۔
  • انفرادی تغیر: ایک ہی sTSS مختلف تیراکوں کے لیے مختلف محسوس ہوتا ہے۔ اپنی بحالی کی بنیاد پر رہنما خطوط کو ایڈجسٹ کریں۔

sTSS کیوں اہم ہے

Training Stress Score درج ذیل کی بنیاد ہے:

  • CTL (Chronic Training Load): آپ کی فٹنس کی سطح - روزانہ sTSS کی 42 دن کی تیز رفتار وزنی اوسط
  • ATL (Acute Training Load): آپ کی تھکاوٹ - روزانہ sTSS کی 7 دن کی تیز رفتار وزنی اوسط
  • TSB (Training Stress Balance): آپ کی فارم کی حالت - TSB = CTL - ATL (مثبت = تازہ، منفی = تھکا ہوا)
  • دورانیہ بندی: CTL کے ہدف کی ترقی استعمال کرتے ہوئے تربیتی مراحل (بنیاد، ترقی، عروج، کمی) کی منصوبہ بندی
  • بحالی کا انتظام: TSB کی بنیاد پر جاننا کہ کب دھکا دینا ہے اور کب آرام کرنا ہے

پرو ٹپ: اپنی CTL کو ٹریک کریں

اسپریڈ شیٹ یا تربیتی ڈائری میں روزانہ sTSS کو ریکارڈ کریں۔ ہفتہ وار اپنی 42 دن کی اوسط (CTL) کا حساب لگائیں۔ بنیاد کی تعمیر کے دوران ہفتے میں 5-10 پوائنٹس CTL میں اضافے کا ہدف رکھیں۔ کمی کے دوران CTL کو برقرار رکھیں یا تھوڑا کم کریں (مقابلے سے 1-2 ہفتے پہلے)۔

اکثر پوچھے گئے سوالات

تیراکی TSS (sTSS) کیا ہے؟

سویمنگ ٹریننگ اسٹریس اسکور (sTSS) ایک میٹرک ہے جو شدت اور دورانیہ دونوں کو ملا کر تیراکی کی ورزش کے ٹریننگ لوڈ کی مقدار کا تعین کرتا ہے۔ یہ سائیکلنگ کے TSS طریقہ کار سے اخذ کیا گیا ہے، جو آپ کی کرٹیکل سویم اسپیڈ (CSS) کو تھریشولڈ رفتار کے طور پر استعمال کرتا ہے۔ CSS رفتار پر 1 گھنٹے کی ورزش 100 sTSS کے برابر ہے۔

میں اپنے sTSS کا حساب کیسے لگاؤں؟

اپنی CSS رفتار (CSS ٹیسٹ سے)، کل ورزش کا دورانیہ، اور ورزش کے دوران اوسط رفتار درج کرکے اوپر والا کیلکولیٹر استعمال کریں۔ فارمولا ہے: sTSS = دورانیہ (گھنٹے) × شدت کا عنصر² × 100، جہاں شدت کا عنصر = CSS رفتار / اوسط ورزش کی رفتار۔

کیا مجھے sTSS کا حساب لگانے کے لیے CSS کی ضرورت ہے؟

ہاں، آپ کی کرٹیکل سویم اسپیڈ (CSS) شدت کے عنصر کا حساب لگانے کے لیے ضروری ہے، جو sTSS کے حساب کتاب کے لیے ضروری ہے۔ CSS آپ کی تھریشولڈ رفتار کی نمائندگی کرتا ہے اور اسے ہر 6-8 ہفتوں میں ٹیسٹ کیا جانا چاہیے۔ آپ ہمارے CSS کیلکولیٹر کا استعمال کرکے اپنا CSS تلاش کرسکتے ہیں۔

ایک ورزش کے لیے اچھا sTSS اسکور کیا ہے؟

یہ ورزش کی شدت پر منحصر ہے: آسان ورزشیں عام طور پر 50 sTSS سے کم، اعتدال پسند ورزشیں 50-100 sTSS، سخت ورزشیں 100-200 sTSS، اور بہت سخت ورزشیں 200 sTSS سے زیادہ اسکور کرتی ہیں۔ مناسب اسکور آپ کے ٹریننگ کے اہداف اور موجودہ فٹنس کی سطح پر منحصر ہے۔

مجھے فی ہفتہ کتنا sTSS کرنا چاہیے؟

ہفتہ وار sTSS کے اہداف سطح کے لحاظ سے مختلف ہوتے ہیں: تفریحی تیراک: 150-300، فٹنس تیراک/ٹرائیتھلیٹس: 300-500، مسابقتی ماسٹرز: 500-800، ایلیٹ/کالجیٹ: 800-1200+۔ اوور ٹریننگ سے بچنے کے لیے قدامت پسندانہ طور پر شروع کریں اور آہستہ آہستہ بڑھائیں۔

کیا تیراکی TSS سائیکلنگ TSS کی طرح ہے؟

تصور اور فارمولا ایک جیسے ہیں، لیکن sTSS تیراکی کے لیے موافق ہے۔ سائیکلنگ TSS کی طرح طاقت (FTP) استعمال کرنے کے بجائے، sTSS رفتار کو CSS کے ساتھ حد کے طور پر استعمال کرتا ہے۔ دونوں مدت × شدت کا عنصر² × 100 استعمال کرتے ہوئے تربیتی بوجھ کی پیمائش کرتے ہیں۔

کیا میں تمام تیراکی کے اسٹروکس کے لیے sTSS استعمال کر سکتا ہوں؟

ہاں، لیکن آپ کا CSS اسٹروک کے لیے مخصوص ہونا چاہیے۔ زیادہ تر تیراک فری اسٹائل CSS استعمال کرتے ہیں کیونکہ یہ سب سے زیادہ تربیت یافتہ اسٹروک ہے۔ اگر آپ بنیادی طور پر کسی اور اسٹروک میں تربیت کرتے ہیں، تو اس اسٹروک میں CSS ٹیسٹ کریں اور sTSS کی گنتی کے لیے اس رفتار کو استعمال کریں۔

sTSS اور CTL/ATL/TSB میں کیا فرق ہے؟

sTSS ایک ورزش کے تربیتی بوجھ کی پیمائش کرتا ہے۔ CTL (دائمی تربیتی بوجھ) آپ کی طویل مدتی فٹنس ہے، ATL (شدید تربیتی بوجھ) آپ کی حالیہ تھکاوٹ ہے، اور TSB (تربیتی دباؤ کا توازن) آپ کی تازگی ہے۔ یہ میٹرکس وقت کے ساتھ sTSS کی قدروں کو استعمال کرتے ہوئے آپ کی تربیتی حالت کو ٹریک کرتے ہیں۔ ہماری ٹریننگ لوڈ گائیڈ میں مزید جانیں۔
