Ingyenes Úszás TSS Kalkulátor

Számítsa ki az edzési stressz pontszámot (sTSS) úszóedzésekhez - Az egyetlen ingyenes sTSS kalkulátor

Mi az az Úszás TSS (sTSS)?

Az Úszás Edzési Stressz Pontszám (Swimming Training Stress Score - sTSS) az úszóedzés terhelését számszerűsíti az intenzitás és az időtartam kombinálásával. A kerékpározás TSS módszertanából lett átvéve, a Kritikus Úszósebességet (CSS) használva küszöbtempóként. Egy 1 órás edzés CSS tempóban = 100 sTSS.

Ingyenes sTSS Kalkulátor

Edzésterhelés Értelmezése

Hogyan Számolják az sTSS-t

Képlet

sTSS = (Időtartam órában) × (Intenzitási Faktor)² × 100

Ahol:

  • Intenzitási Faktor (IF) = CSS Tempó / Átlagos Edzéstempó
  • Időtartam = Teljes edzésidő órában
  • CSS Tempó = Az Ön küszöbtempója a CSS tesztből

Kidolgozott Példa

Edzés Részletei:

  • CSS Tempó: 1:49/100m (109 másodperc)
  • Edzés Időtartama: 60 perc (1 óra)
  • Átlagos Tempó: 2:05/100m (125 másodperc)

1. Lépés: Intenzitási Faktor Számítása

IF = CSS Tempó / Edzéstempó
IF = 109 / 125
IF = 0.872

2. Lépés: sTSS Számítása

sTSS = 1.0 óra × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

Értelmezés: Ez a 60 perces edzés könnyű tempóban (lassabb, mint a CSS) 76 sTSS-t eredményezett - mérsékelt edzésterhelés, amely alkalmas az aerob alapozásra.

Az sTSS Értékek Megértése

sTSS Tartomány Edzésterhelés Regenerációs Idő Példa Edzés
< 50 Alacsony Azonos nap Könnyű 30 perces úszás, technikai gyakorlatok

Heti sTSS Irányelvek

A heti cél sTSS az edzettségi szintjétől és céljaitól függ:

Hobbiúszók

Heti sTSS: 150-300

Heti 2-3 edzés, egyenként 50-100 sTSS. Fókusz a technikán és az aerob alapok építésén.

Fitnesz Úszók / Triatlonisták

Heti sTSS: 300-500

Heti 3-4 edzés, egyenként 75-125 sTSS. Aerob állóképesség és küszöbmunka keveréke.

Versenyző Masters Úszók

Heti sTSS: 500-800

Heti 4-6 edzés, egyenként 80-150 sTSS. Strukturált edzés periodizációval.

Elite / Egyetemi Úszók

Heti sTSS: 800-1200+

Heti 8-12 edzés, dupla napok. Nagy volumen, ahol a regeneráció menedzselése kritikus.

⚠️ Fontos Megjegyzések

  • Pontos CSS szükséges: A CSS-nek aktuálisnak kell lennie (6-8 héten belül tesztelve) a pontos sTSS érdekében.
  • Egyszerűsített számítás: Ez a kalkulátor átlagos tempót használ. A fejlett sTSS a Normalized Graded Pace (NGP) értéket használja, amely figyelembe veszi az intervallum szerkezetét.
  • Nem technikai munkához: Az sTSS csak a fizikai edzésterhelést méri, nem a készségfejlesztést.
  • Egyéni eltérés: Ugyanaz az sTSS másképp érződhet különböző úszók számára. Igazítsa az irányelveket a regenerációja alapján.

Miért Fontos az sTSS

Az Edzési Stressz Pontszám az alapja a következőknek:

  • CTL (Chronic Training Load): Az Ön edzettségi szintje - a napi sTSS 42 napos exponenciálisan súlyozott átlaga
  • ATL (Acute Training Load): Az Ön fáradtsága - a napi sTSS 7 napos exponenciálisan súlyozott átlaga
  • TSB (Training Stress Balance): Az Ön formája - TSB = CTL - ATL (pozitív = friss, negatív = fáradt)
  • Periodizáció: Edzési fázisok tervezése (alapozás, építés, csúcsot érés, levezetés) CTL célok használatával
  • Regeneráció Menedzsment: Tudja, mikor kell nyomni és mikor kell pihenni a TSB alapján

Pro Tipp: Kövesse Nyomon a CTL-t

Rögzítse a napi sTSS-t egy táblázatban vagy edzésnaplóban. Számítsa ki a 42 napos átlagát (CTL) hetente. Célozzon meg heti 5-10 CTL pont növekedést az alapozás során. Tartsa vagy enyhén csökkentse a CTL-t a levezetés alatt (1-2 héttel a verseny előtt).

Gyakran Ismételt Kérdések

Mi az az Úszás TSS (sTSS)?

Az Úszás Edzési Stressz Pontszám (sTSS) egy mérőszám, amely az úszóedzés terhelését számszerűsíti az intenzitás és az időtartam kombinálásával. A kerékpározás TSS módszertanából lett átvéve, a Kritikus Úszósebességet (CSS) használva küszöbtempóként. Egy 1 órás edzés CSS tempóban 100 sTSS-nek felel meg.

Hogyan számíthatom ki az sTSS-emet?

Használja a fenti kalkulátort a CSS tempója (CSS tesztből), a teljes edzésidő és az edzés átlagos tempójának megadásával. A képlet: sTSS = Időtartam (óra) × Intenzitási Faktor² × 100, ahol Intenzitási Faktor = CSS Tempó / Átlagos Edzéstempó.

Szükségem van CSS-re az sTSS számításához?

Igen, a Kritikus Úszósebessége (CSS) szükséges az Intenzitási Faktor kiszámításához, amely elengedhetetlen az sTSS számításához. A CSS a küszöbtempóját képviseli, és 6-8 hetente tesztelni kell. Megtalálhatja a CSS-ét a CSS kalkulátorunk használatával.

Mi a jó sTSS pontszám egy edzéshez?

Ez az edzés intenzitásától függ: A könnyű edzések általában 50 sTSS alatt, a mérsékelt edzések 50-100 sTSS, a kemény edzések 100-200 sTSS, a nagyon kemény edzések pedig 200 sTSS felett pontoznak. A megfelelő pontszám az edzési céljaitól és a jelenlegi edzettségi szintjétől függ.

Mennyi sTSS-t kellene csinálnom hetente?

A heti sTSS célok szintenként változnak: Hobbiúszók: 150-300, Fitnesz úszók/Triatlonisták: 300-500, Versenyző Masters: 500-800, Elite/Egyetemi: 800-1200+. Kezdje konzervatívan, és fokozatosan növelje a tú edzés elkerülése érdekében.

Az úszás TSS ugyanaz, mint a kerékpározás TSS?

A koncepció és a képlet ugyanaz, de az sTSS úszásra van adaptálva. Ahelyett, hogy teljesítményt (FTP) használna, mint a kerékpározás TSS, az sTSS tempót használ a CSS-sel mint küszöbérték. Mindkettő az Edzésterhelést méri az Időtartam × Intenzitási Faktor² × 100 képlettel.

Használhatom az sTSS-t minden úszásnemhez?

Igen, de a CSS-ének úszásnem-specifikusnak kell lennie. A legtöbb úszó gyorsúszás CSS-t használ, mivel ez a leggyakrabban edzett úszásnem. Ha elsősorban másik úszásnemben edz, végezzen CSS tesztet abban az úszásnemben, és használja azt a tempót az sTSS számításához.

Mi a különbség az sTSS és a CTL/ATL/TSB között?

Az sTSS egyetlen edzés terhelését méri. A CTL (Krónikus Edzésterhelés) az Ön hosszú távú edzettsége, az ATL (Akut Edzésterhelés) az Ön közelmúltbeli fáradtsága, és a TSB (Edzési Stressz Egyensúly) az Ön frissessége. Ezek a mérőszámok sTSS értékeket használnak idővel az edzési állapota nyomon követésére. Tudjon meg többet a Edzésterhelés útmutatónkban.
