SwimAnalytics vs. más úszó appok

Úszó-specifikus metrikák, automatikus sTSS, kedvező ár

Miért kell speciális úszóanalitika?

A multisport platformok gyakran hiányosan támogatják az úszást. A SwimAnalytics natív úszó metrikákat (CSS, sTSS, SWOLF) és watch-native elemzést ad a pontosságért és egyszerűségért.

Natív úszó metrikák

CSS alapú zónák, sTSS, SR/DPS, SWOLF.

Automatikus sTSS

Apple Watch pasok importja után sTSS/CTL/ATL/TSB kézi log nélkül.

Adatvédelem

100% eszközön, szerver/konto/tracking nélkül.

Kulcskülönbségek

1. Automatikus sTSS

  • CSS + tempó → IF³ → sTSS minden pasra
  • CTL/ATL/TSB automatikusan frissül
  • Nincs manuális log

2. Úszóhoz igazított zónák

  • 7 zóna CSS tesztből
  • Köbös intenzitássúlyozás (víz ellenállása)
  • Szegmens tempó automatikusan zónákhoz kötve

3. Adatvédelem & egyszerűség

  • Nincs konto, nincs szerver
  • Watch-native, egyszerű UI

4. Ár

  • Alacsonyabb, mint pl. TrainingPeaks Premium
  • Nincs drága hardverigény
  • Átlátható díj, rejtett költség nincs

GYIK

Garminnal működik?

Apple Health-szinkron kell. Ha a Garmin pasok Health-be kerülnek, importálhatóak. Watch-on sTSS: Apple Watch.

Kell manuálisan logolni?

Nem. Apple Watch-on rögzített pasok automatikusan importálódnak.

SWOLF/stroke elemzés?

SR/DPS/SWOLF úszó-specifikusan, CSS/zónák/sTSS integrációval.

Hol tárolódnak az adatok?

Csak az eszközön, nincs feltöltés.

CSS zónák, automatikus sTSS, on-device adatvédelem.
