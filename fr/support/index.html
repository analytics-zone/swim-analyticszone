Foire aux questions

Comment synchroniser mes entraînements ?

L'app se synchronise automatiquement avec Apple Health pour importer les entraînements de natation enregistrés par tout appareil ou application compatible. Assurez-vous d'avoir accordé les autorisations nécessaires à l'app Santé dans les réglages iOS.

Mes données sont-elles privées ?

Oui, toutes les données sont traitées localement sur votre appareil. Nous ne collectons, ne stockons ni ne transmettons aucune de vos informations personnelles. Consultez notre politique de confidentialité complète.

Comment exporter mes données ?

Vous pouvez exporter vos données d'entraînement et vos analyses dans plusieurs formats (JSON, CSV, HTML, PDF) directement depuis l'app. Tous les exports sont générés localement sur votre appareil.

Ai-je besoin d'une connexion Internet ?

Non, l'app fonctionne entièrement hors ligne. Tous les calculs et le traitement des données sont effectués localement sur votre appareil.

Puis-je utiliser l'app sur plusieurs appareils ?

L'app peut être installée sur tous vos appareils iOS utilisant le même identifiant Apple. Toutefois, les données sont stockées localement sur chaque appareil, sauf si vous activez les sauvegardes des apps iOS via iCloud.

Besoin d'aide supplémentaire ?

Vous ne trouvez pas ce que vous cherchez ? Écrivez-nous à swimanalytics@onmedic.org et nous vous répondrons dès que possible.
