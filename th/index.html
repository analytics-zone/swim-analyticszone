ตัวชี้วัดทางวิทยาศาสตร์

Critical Swim Speed (CSS) กำหนดค่าขีดจำกัดความสามารถแบบใช้ออกซิเจน (Aerobic threshold) ของคุณ ทำให้สามารถคำนวณ Training Stress Score (TSS) และ ติดตามสมรรถนะแบบ CTL/ATL/TSB โดยอิงจากงานวิจัยด้านวิทยาศาสตร์การกีฬาระดับสากล เรียนรู้เพิ่มเติมเกี่ยวกับเราสูตรทางคณิตศาสตร์และการวัดประสิทธิภาพการว่ายน้ำ.

โซนการฝึกซ้อม

7 โซนการฝึกซ้อมส่วนบุคคลที่ปรับแต่งตามค่า CSS ของคุณ เพิ่มประสิทธิภาพสูงสุดให้กับทุกการฝึกซ้อม ทั้งช่วงพักฟื้น, พื้นฐานแอโรบิก, ขีดจำกัดความอดทน หรือการพัฒนา VO₂max

การเปรียบเทียบอัจฉริยะ

เปรียบเทียบข้อมูลรายสัปดาห์ รายเดือน และรายปี พร้อมระบบตรวจจับแนวโน้มอัตโนมัติและการเปลี่ยนแปลงเป็นเปอร์เซ็นต์สำหรับตัวชี้วัดทั้งหมด

ความเป็นส่วนตัวสมบูรณ์แบบ

ประมวลผลข้อมูลทั้งหมดภายในอุปกรณ์ของคุณ ไม่มีการส่งข้อมูลผ่านเซิร์ฟเวอร์หรือคลาวด์ และไม่มีการติดตามใดๆ คุณเป็นเจ้าของและเป็นผู้ควบคุมข้อมูลการว่ายน้ำของคุณเอง

ส่งออกข้อมูลได้ทุกที่

ส่งออกผลการฝึกซ้อมและการวิเคราะห์ในรูปแบบ JSON, CSV, HTML หรือ PDF สามารถทำงานร่วมกับผู้ฝึกสอน, สเปรดชีต และแพลตฟอร์มการฝึกซ้อมชั้นนำได้

สมรรถนะที่รวดเร็วทันใจ

ตัวแอปเปิดใช้งานได้เร็วกว่า 0.35 วินาที ด้วยสถาปัตยกรรมที่เน้นการประมวลผลภายในเครื่อง เข้าดูผลการฝึกซ้อมได้ทันทีโดยไม่ต้องรอซิงค์หรือดาวน์โหลด

ระบบทดสอบ 400ม. + 200ม. ในตัวเพื่อกำหนดค่า Critical Swim Speed ของคุณ ควรทดสอบซ้ำทุก 6-8 สัปดาห์เพื่อติดตามความก้าวหน้าและปรับโซนการฝึกซ้อมโดยอัตโนมัติ

ประสบการณ์ iOS แท้ๆ

สร้างด้วย SwiftUI เพื่อการทำงานที่ลื่นไหลและการผสานรวมกับ iOS อย่างสมบูรณ์ ซิงค์กับแอป Health ได้อย่างไร้รอยต่อ รองรับวิดเจ็ต และใช้ภาษาการออกแบบที่คุ้นเคยของ Apple

อิงตามงานวิจัย

ตัวชี้วัดทั้งหมดอิงตามงานวิจัยวิทยาศาตร์การกีฬาที่ผ่านการรับรอง CSS จาก Wakayoshi และคณะ, TSS ที่ปรับแต่งเพื่อการว่ายน้ำด้วยสูตร IF³ และโมเดล CTL/ATL ที่ผ่านการพิสูจน์แล้ว

เป็นมิตรต่อผู้ฝึกสอน

ใช้งานได้ทุกที่

ไม่ว่าจะเป็นสระว่ายน้ำหรือแหล่งน้ำธรรมชาติ การว่ายน้ำระยะสั้นหรือระยะไกล Swim Analytics ปรับใช้ได้กับการว่ายน้ำทุกประเภทและตรวจจับลักษณะการฝึกซ้อมได้โดยอัตโนมัติ

คำถามที่พบบ่อย

Swim Analytics รับข้อมูลการว่ายน้ำของฉันได้อย่างไร?

Swim Analytics จะซิงค์ข้อมูลกับแอป Apple Health เพื่อนำเข้าข้อมูลการว่ายน้ำที่บันทึกโดยอุปกรณ์หรือแอปใดๆ ที่รองรับ ซึ่งรวมถึงสมาร์ทวอทช์, อุปกรณ์ติดตามการออกกำลังกาย และการบันทึกด้วยตนเอง โดยตัวแอปจะประมวลผลข้อมูลเหล่านี้ภายในเครื่องเพื่อคำนวณตัวชี้วัดขั้นสูง

การทดสอบ CSS คืออะไร และต้องทำอย่างไร?

CSS (Critical Swim Speed) คือโปรโตคอลทางวิทยาศาสตร์ที่ใช้การว่ายน้ำแบบเต็มกำลัง 2 ระยะ ได้แก่ 400 เมตร และ 200 เมตร โดยมีการพัก 10-20 นาทีระหว่างรอบ ตัวแอปจะคำนวณขีดจำกัดความสามารถแบบใช้ออกซิเจนจากเวลาที่ทำได้ และปรับโซนการฝึกซ้อมทั้งหมดให้โดยอัตโนมัติ ควรทดสอบซ้ำทุก 6-8 สัปดาห์เพื่อติดตามพัฒนาการ

ข้อมูลของฉันถูกอัปโหลดขึ้นคลาวด์หรือไม่?

ไม่ ข้อมูลทั้งหมดใน Swim Analytics จะถูกประมวลผลภายใน iPhone ของคุณเท่านั้น ไม่มีการส่งข้อมูลไปยังเซิร์ฟเวอร์ภายนอก ไม่มีบัญชีคลาวด์ และไม่มีการโอนถ่ายข้อมูล คุณเป็นผู้ควบคุมการส่งออกข้อมูลเอง ทั้งในรูปแบบ JSON, CSV, HTML หรือ PDF และสามารถแชร์ข้อมูลได้ตามที่ต้องการ

ฉันสามารถใช้ Swim Analytics กับการว่ายน้ำในแหล่งน้ำธรรมชาติ (Open Water) ได้หรือไม่?

ได้ Swim Analytics รองรับข้อมูลการว่ายน้ำทุกประเภทใน Apple Health รวมถึงการว่ายน้ำในแหล่งน้ำธรรมชาติ ตัวแอปจะปรับตัวชี้วัดที่มีให้ตามความเหมาะสม ไม่ว่าคุณจะว่ายในสระหรือแหล่งน้ำธรรมชาติ เพื่อให้การวิเคราะห์ที่ตรงจุดสำหรับแต่ละสภาพแวดล้อม

แผนรายเดือนและรายปีแตกต่างกันอย่างไร?

ทั้งสองแผนมีคุณสมบัติที่เหมือนกันทุกประการ ทั้งตัวชี้วัดทั้งหมด, โซนไม่จำกัด, การเปรียบเทียบตามช่วงเวลา, การส่งออกหลายรูปแบบ และการอัปเดตฟรี ความแตกต่างเพียงอย่างเดียวคือราคา โดยแผนรายปีจะช่วยคุณประหยัดถึง 18% (เฉลี่ยเพียง €3.25/เดือน เทียบกับ €3.99/เดือน)

ฉันสามารถยกเลิกการสมัครใช้งานได้ตลอดเวลาหรือไม่?

ได้ การสมัครใช้งานจะถูกจัดการผ่าน App Store ดังนั้นคุณจึงสามารถยกเลิกได้ทุกเมื่อผ่านเมนู การตั้งค่า → [ชื่อของคุณ] → การสมัครรับ หากคุณยกเลิก คุณจะยังสามารถใช้งานได้จนกว่าจะสิ้นสุดรอบบิลปัจจุบัน

พร้อมที่จะเปลี่ยนการว่ายน้ำของคุณแล้วหรือยัง?

เข้าร่วมกับนักว่ายน้ำหลายพันคนที่ใช้ค่าตัวชี้วัดทางวิทยาศาสตร์เพื่อพัฒนาสมรรถนะ เริ่มต้นทดลองใช้ฟรี 7 วันได้ตั้งแต่วันนี้
