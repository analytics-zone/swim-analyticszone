ความเร็วการว่ายน้ำที่สำคัญ (CSS) กับอัตราการก้าวตามเกณฑ์: สิ่งไหนสำคัญ?

หลายปีที่ผ่านมา "Threshold Pace" (T-Pace) คือมาตรฐานทองคำ ในปี 2026 การเปลี่ยนไปใช้ Critical Swim Speed ​​(CSS) ถือเป็นการกำหนดจังหวะของนักกีฬาและคาดการณ์ความเหนื่อยล้าในวันแข่งขัน

🎯 คำตอบด่วน

T-Pace (ก้าวเกณฑ์) เป็นการประมาณความเร็วโดยทั่วไปที่คุณสามารถรักษาได้เป็นเวลาหนึ่งชั่วโมง ซึ่งมักใช้สำหรับการกำหนดโซนการฝึกแบบกว้างๆ เป็นเมตริกที่ "ใช้งานได้จริง" แต่มีรายละเอียดน้อยกว่า

CSS (ความเร็วการว่ายน้ำวิกฤต) คือเกณฑ์ "เชิงหน้าที่" ซึ่งได้มาทางคณิตศาสตร์จากระยะทางที่แตกต่างกันสองระยะ (เช่น 400 ม. และ 200 ม.) เป็นความเร็วตามทฤษฎีที่คุณสามารถรักษาได้โดยไม่ต้องชน "กำแพง" หากคุณเป็นนักว่ายน้ำหรือนักไตรกีฬาที่มีความอดทน CSS ของคุณคือตัวเลขที่สำคัญที่สุดในโปรแกรมการฝึกของคุณ

การเปลี่ยนแปลงปี 2026: จากความพยายามสู่สรีรวิทยา

การจัดการความเหนื่อยล้าในสระน้ำเป็นเรื่องยากมากเมื่อเทียบกับการปั่นจักรยานหรือการวิ่ง หากไม่มีเครื่องวัดกำลัง การก้าวเป็นเพียงตัวแทนของความเข้มข้นเท่านั้น ในขณะที่นักว่ายน้ำหลายคนยังคงพึ่งพาฝีก้าวแบบ "จับเวลา" 1,500 เมตรแบบเก่า การฝึกสอนสมัยใหม่ได้ย้ายไปที่ โมเดล CSS แบบสองทาง.

ความแตกต่างที่สำคัญคือ ความแม่นยำกับความพยายามในการรับรู้. T-Pace ที่อิงจากความพยายามที่ยาวนานเพียงครั้งเดียวมักจะจับกรวดทางจิตของคุณมากกว่าเกณฑ์การเผาผลาญของคุณ CSS ซึ่งคำนวณจากจุดที่แตกต่างกันสองจุดบน "Pace-Duration Curve" จะแสดงขีดจำกัดแอโรบิกที่แท้จริงของเครื่องยนต์

การเปรียบเทียบเมตริก

ก้าวเกณฑ์ (T-Pace)

ตัวชี้วัด "แบบง่าย"

โดยทั่วไปแล้ว T-Pace จะมาจากความพยายามสูงสุด 1,500 ม. หรือ 1,000 ม. เพียงครั้งเดียว มันแสดงถึง "เส้นสีแดง" ของคุณสำหรับการว่ายน้ำอย่างต่อเนื่อง

  • ข้อดี: ง่ายต่อการเข้าใจ หนึ่งการทดสอบ หนึ่งหมายเลข เหมาะสำหรับชุดที่ยาวและต่อเนื่อง
  • จุดด้อย: มีความไวสูงต่อข้อผิดพลาดในการเว้นจังหวะ ไม่ได้คำนึงถึงความเร็วที่คุณจางหายไปภายใต้ความเหนื่อยล้า ปฏิบัติต่อนักว่ายน้ำทุกคนที่มีฝีก้าว 1,500 เมตรเท่ากันโดยมีลักษณะทางสรีรวิทยาเหมือนกัน

ความเร็วการว่ายน้ำวิกฤต (CSS)

ตัวชี้วัด "การทำงาน"

CSS เป็นรากศัพท์ทางคณิตศาสตร์ที่อิงจากระยะทางสองระยะ (โดยทั่วไปคือความพยายามสูงสุด 400 ม. และ 200 ม.)

  • ข้อดี: ขจัดอคติในการเว้นจังหวะ ทำนายพฤติกรรมของคุณตามช่วงเวลาได้ดีขึ้น ให้วัตถุประสงค์ "เพดานแอโรบิก"
  • จุดด้อย: ต้องมีการทดสอบแยกกันสองครั้ง คณิตศาสตร์ที่ซับซ้อนกว่าเล็กน้อย (คำนวณอัตโนมัติใน Swim Analytics) กำหนดให้ทำการทดสอบโดยพักผ่อนอย่างสม่ำเสมอ

เหตุใดเดลต้าจึงมีความสำคัญ

"เดลต้า" คือความแตกต่างระหว่างเพซ 400 ม. และเพซ 200 ม. ของคุณ นักว่ายน้ำสองคนอาจมี T-Pace อยู่ที่ 1:40/100 ม. แต่โปรไฟล์ CSS ของพวกเขาอาจมีลักษณะดังนี้:

นักว่ายน้ำ A ("The Diesel")

สามเหลี่ยมปากแม่น้ำขนาดเล็กระหว่าง 200 ม. ถึง 400 ม. นักว่ายน้ำคนนี้มีประสิทธิภาพแอโรบิกสูงมากแต่มีความเร็วในการวิ่งต่ำกว่า CSS ของพวกเขาอยู่ใกล้กับ T-Pace มาก

นักว่ายน้ำ B ("The Sprinter")

เดลต้าขนาดใหญ่ ความเร็วที่ยอดเยี่ยม 200 ม. แต่ลดลงอย่างเห็นได้ชัดที่ 400 ม. CSS ของพวกเขาจะต่ำกว่าที่ T-Pace แนะนำ หากพวกเขาพยายามฝึกแบบ "T-Pace" ทั่วไป พวกเขาจะปรุงระบบแอนแอโรบิกมากเกินไปอย่างต่อเนื่อง

📈 สูตร CSS

ซีเอสเอส = (D1 - D2) / (T1 - T2)

โดยที่ D1/D2 คือระยะทางทั้งสอง และ T1/T2 คือเวลาเป็นวินาที สิ่งนี้เผยให้เห็น "ความชัน" ของอัตราการก้าวที่ยั่งยืนของคุณโดยปราศจากเสียงรบกวนจากกระแสแอนแอโรบิกในช่วงแรก

วิธีดำเนินการทดสอบ CSS (ทีละขั้นตอน)

หากต้องการคำนวณ CSS คุณต้องดำเนินการทดลองสูงสุดสองครั้งในวันที่ต่างกัน (หรือด้วยการกู้คืนทั้งหมดในระหว่างนั้น) ในพูลมาตรฐาน:

  1. อุ่นเครื่อง: วอร์มอัพที่ได้มาตรฐานอย่างละเอียดถี่ถ้วน (โดยทั่วไปคือ 400 ม. ถึง 600 ม. ของจังหวะและท่าผสม) เพื่อเตรียมร่างกายของคุณ
  2. การทดลองที่ 1 (ความพยายามสูงสุด 400 เมตร): ว่ายน้ำ 400 เมตรให้เร็วที่สุด บันทึกเวลาที่แน่นอน ก้าวตัวเองเพื่อไม่ให้คุณเหนื่อยหน่ายใน 100 เมตรแรก
  3. การกู้คืนที่ใช้งานอยู่: ว่ายน้ำ 100 ม. ถึง 200 ม. เพื่อล้างกรดแลคติคออกอย่างง่ายดาย ตามด้วยการพัก 5-10 นาที
  4. การทดลองที่ 2 (ความพยายามสูงสุด 200 ม.): ว่ายน้ำ 200 ม. ด้วยความพยายามสูงสุด บันทึกเวลาที่แน่นอน
  5. คำนวณ: แปลงเวลาของคุณเป็นวินาทีแล้วใช้สูตรด้านบน หรือใช้เครื่องมือคำนวณ CSS ใน Swim Analytics เพื่อให้ได้อัตราการก้าว 100 เมตรของคุณ

คุณควรใช้อันไหน?

ใช้ Threshold Pace (T-Pace) หาก:

  • คุณเพิ่งเริ่มต้นด้วยการฝึกตามจังหวะก้าว
  • คุณจะว่ายน้ำต่อเนื่องเป็นเวลานานเท่านั้น (น้ำเปิด 3 กม. ขึ้นไป)
  • คุณไม่ต้องการจัดการกับเซสชันการทดสอบหลายเซสชัน

ใช้ความเร็ววิกฤตว่ายน้ำ (CSS) หาก:

  • นักไตรกีฬาและนักว่ายน้ำมาราธอน: เพื่อหลีกเลี่ยง "การจุดเทียน" เร็วเกินไปในการแข่งขัน
  • การฝึกแบบช่วง: เพื่อตั้งค่าโซน CSS + 2s หรือ CSS - 1s เฉพาะที่ใช้งานได้จริง
  • ติดตามความคืบหน้า: CSS มีความไวต่อการปรับปรุงแอโรบิกมากกว่าการทดสอบ 1,000 ม. ทั่วไป

เกณฑ์ก้าวเทียบกับคำถามที่พบบ่อย CSS

CSS เหมือนกับก้าวของเกณฑ์หรือไม่

ไม่ เป็นแนวคิดเกี่ยวกับเกณฑ์ที่เกี่ยวข้องกัน แต่ไม่ใช่การทดสอบเดียวกัน ความเร็วเกณฑ์มักจะมาจากการว่ายน้ำอีกต่อไปหนึ่งครั้ง ในขณะที่ CSS คำนวณจากความพยายามสองครั้งและให้จุดยึดที่ทำซ้ำได้มากขึ้นสำหรับการทำงานตามช่วงเวลา

อะไรจะดีไปกว่าสำหรับไตรกีฬาและการเดินในแหล่งน้ำเปิด?

โดยปกติแล้ว CSS จะเป็นจุดเริ่มต้นที่ดีกว่า เนื่องจากสะท้อนถึงก้าวที่ยั่งยืนโดยไม่ต้องให้รางวัลกับความเร็วระยะสั้นมากเกินไป อัตราก้าวตามเกณฑ์ยังคงมีประโยชน์สำหรับการว่ายน้ำอย่างต่อเนื่องเป็นเวลานาน แต่โดยปกติแล้ว CSS จะแปลงเป็นเป้าหมายการเว้นจังหวะและการทำงานของโซนได้ง่ายกว่า

ฉันจะย้ายจากเกณฑ์ก้าวไปยังโซนว่ายน้ำที่ใช้ CSS ได้อย่างไร

ใช้การทดสอบ 400m และ 200m ถัดไปของคุณเพื่อตั้งค่า CSS จากนั้นช่วงจุดยึดจะแก้ไขตัวเลขนั้น หากคุณฝึกด้วย T-Pace อยู่แล้ว ให้เปรียบเทียบตัวเลขทั้งสองและใช้คำแนะนำ CSS และโซนการฝึกว่ายน้ำเพื่อสร้างเซตของคุณใหม่ตามอัตราการก้าวแบบแอโรบิกที่ทำซ้ำได้
