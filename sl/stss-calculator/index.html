Kaj je TSS za Plavanje (sTSS)?

Swimming Training Stress Score (sTSS) kvantificira obremenitev plavalnega treninga s kombiniranjem intenzivnosti in trajanja. Prilagojen je iz metodologije TSS kolesarjenja, pri čemer uporablja vašo Critical Swim Speed (CSS) kot mejno hitrost. Trening 1 ure pri hitrosti CSS = 100 sTSS.

Interpretacija Trenažne Obremenitve

Kako se Izračuna sTSS

Formula

sTSS = (Trajanje v urah) × (Faktor Intenzivnosti)² × 100

Kjer:

  • Faktor Intenzivnosti (IF) = CSS Hitrost / Povprečna Hitrost Treninga
  • Trajanje = Skupni čas treninga v urah
  • CSS Hitrost = Vaša mejna hitrost iz CSS testa

Praktični Primer

Podrobnosti Treninga:

  • CSS Hitrost: 1:49/100m (109 sekund)
  • Trajanje Treninga: 60 minut (1 ura)
  • Povprečna Hitrost: 2:05/100m (125 sekund)

Korak 1: Izračun Faktorja Intenzivnosti

IF = CSS Hitrost / Hitrost Treninga
IF = 109 / 125
IF = 0.872

Korak 2: Izračun sTSS

sTSS = 1.0 ure × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

Interpretacija: Ta 60-minutni trening pri lahki hitrosti (počasneje kot CSS) je ustvaril 76 sTSS - zmerna trenažna obremenitev, primerna za gradnjo aerobne baze.

Interpretacija Vrednosti sTSS

Razpon sTSS Trenažna Obremenitev Čas Okrevanja Primer Treninga
< 50 Nizka Isti dan Lahko plavanje 30 min, tehnične vaje
50-100 Zmerna 1 dan Vzdržljivost 60 min, enakomerna hitrost
100-200 Visoka 1-2 dni Pragovne serije 90 min, intervali v tekmovalni hitrosti
200-300 Zelo Visoka 2-3 dni Težek trening 2 uri, več pragovnih blokov
> 300 Ekstremna 3+ dni Dolga tekma (>2 uri), ultra-vzdržljivost

Tedenske Smernice sTSS

Ciljni tedenski sTSS je odvisen od vaše ravni treninga in ciljev:

Rekreativni Plavalci

Tedenski sTSS: 150-300

2-3 treningi na teden, 50-100 sTSS vsak. Fokus na tehniki in gradnji aerobne baze.

Fitnes Plavalci / Triatlonci

Tedenski sTSS: 300-500

3-4 treningi na teden, 75-125 sTSS vsak. Mešanica aerobne vzdržljivosti in pragovnega dela.

Tekmovalni Masters Plavalci

Tedenski sTSS: 500-800

4-6 treningov na teden, 80-150 sTSS vsak. Strukturiran trening s periodizacijo.

Elitni / Univerzitetni Plavalci

Tedenski sTSS: 800-1200+

8-12 treningov na teden, dvojni treningi. Visok volumen s kritičnim upravljanjem okrevanja.

⚠️ Pomembne Opombe

  • Zahteva natančen CSS: Vaš CSS mora biti aktualen (testiran v zadnjih 6-8 tednih) za natančen sTSS.
  • Poenostavljen izračun: Ta kalkulator uporablja povprečno hitrost. Napredni sTSS uporablja Normalized Graded Pace (NGP), ki upošteva strukturo intervalov.
  • Ni za tehnično delo: sTSS meri le fizični stres treninga, ne razvoja veščin.
  • Individualna variacija: Enak sTSS se različnim plavalcem zdi drugačen. Prilagodite smernice glede na vaše okrevanje.

Zakaj je sTSS Pomemben

Training Stress Score je temelj za:

  • CTL (Chronic Training Load): Vaša raven kondicije - eksponentno uteženo 42-dnevno povprečje dnevnega sTSS
  • ATL (Acute Training Load): Vaša utrujenost - eksponentno uteženo 7-dnevno povprečje dnevnega sTSS
  • TSB (Training Stress Balance): Vaša forma - TSB = CTL - ATL (pozitivno = svež, negativno = utrujen)
  • Periodizacija: Načrtovanje faz treninga (baza, gradnja, vrh, taper) z uporabo ciljnih CTL progresij
  • Upravljanje Okrevanja: Vedeti, kdaj pritisniti in kdaj počivati na podlagi TSB

Pro Nasvet: Sledite svojemu CTL

Beležite dnevni sTSS v preglednico ali dnevnik treninga. Tedensko izračunajte svoje 42-dnevno povprečje (CTL). Ciljajte na povečanje CTL za 5-10 točk na teden med gradnjo baze. Ohranite ali rahlo zmanjšajte CTL med taperjem (1-2 tedna pred tekmo).

Pogosto Zastavljena Vprašanja

Kaj je Swimming TSS (sTSS)?

Swimming Training Stress Score (sTSS) je metrika, ki kvantificira obremenitev treninga plavanja s kombiniranjem intenzivnosti in trajanja. Prilagojena je iz metodologije TSS kolesarjenja, pri čemer uporablja vašo kritično hitrost plavanja (CSS) kot mejno hitrost. Trening 1 ure pri hitrosti CSS je enak 100 sTSS.

Kako si izračunam sTSS?

Uporabite zgornji kalkulator z vnosom vaše CSS hitrosti (iz CSS testa), skupnega trajanja treninga in povprečne hitrosti med treningom. Formula je: sTSS = Trajanje (ure) × Faktor Intenzivnosti² × 100, kjer Faktor Intenzivnosti = CSS Hitrost / Povprečna Hitrost Treninga.

Ali potrebujem CSS za izračun sTSS?

Da, vaša kritična hitrost plavanja (CSS) je potrebna za izračun Faktorja Intenzivnosti, ki je bistven za izračun sTSS. CSS predstavlja vašo mejno hitrost in bi morala biti testirana vsakih 6-8 tednov. Svoj CSS lahko najdete z uporabo našega CSS kalkulatorja.

Kaj je dober sTSS rezultat za en trening?

Odvisno je od intenzivnosti treninga: Lahki treningi običajno dosežejo pod 50 sTSS, zmerni treningi 50-100 sTSS, težki treningi 100-200 sTSS in zelo težki treningi nad 200 sTSS. Ustrezen rezultat je odvisen od vaših ciljev treninga in trenutne ravni kondicije.

Koliko sTSS naj naredim na teden?

Tedenski cilji sTSS se razlikujejo glede na raven: Rekreativni plavalci: 150-300, Fitnes plavalci/Triatlonci: 300-500, Tekmovalni Masters: 500-800, Elitni/Univerzitetni: 800-1200+. Začnite konzervativno in postopoma povečujte, da se izognete prekomernemu treningu.

Je plavalni TSS enak kot kolesarski TSS?

Koncept in formula sta enaka, vendar je sTSS prilagojen plavanju. Namesto uporabe moči (FTP) kot kolesarski TSS, sTSS uporablja tempo s CSS kot prag. Oba merita trenažno obremenitev z uporabo Trajanje × Faktor Intenzivnosti² × 100.

Ali lahko uporabljam sTSS za vse plavalne sloge?

Da, vendar mora biti vaš CSS specifičen za slog. Večina plavalcev uporablja CSS prosto plavanje, saj je to najpogosteje trenirani slog. Če trenirate predvsem v drugem slogu, opravite CSS test v tem slogu in uporabite ta tempo za izračun sTSS.

Kakšna je razlika med sTSS in CTL/ATL/TSB?

sTSS meri trenažno obremenitev enega treninga. CTL (Chronic Training Load) je vaša dolgoročna kondicija, ATL (Acute Training Load) je vaša nedavna utrujenost, TSB (Training Stress Balance) pa je vaša svežina. Te metrike uporabljajo vrednosti sTSS skozi čas za sledenje vašega trenažnega stanja. Več informacij v našem vodniku Trenažne Obremenitve.
