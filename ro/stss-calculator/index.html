Ce este TSS pentru Înot (sTSS)?

Swimming Training Stress Score (sTSS) cuantifică sarcina de antrenament dintr-o sesiune de înot combinând intensitatea și durata. Este adaptat din metodologia TSS a ciclismului, folosind Critical Swim Speed (CSS) ca tempo de prag. Un antrenament de 1 oră la tempo CSS = 100 sTSS.

Interpretarea Sarcinii de Antrenament

Cum se Calculează sTSS

Formula

sTSS = (Durată în ore) × (Factor Intensitate)² × 100

Unde:

  • Factor de Intensitate (IF) = Tempo CSS / Tempo Mediu Antrenament
  • Durată = Timp total antrenament în ore
  • Tempo CSS = Tempoul tău de prag din testul CSS

Exemplu Practic

Detalii Antrenament:

  • Tempo CSS: 1:49/100m (109 secunde)
  • Durată Antrenament: 60 minute (1 oră)
  • Tempo Mediu: 2:05/100m (125 secunde)

Pasul 1: Calculează Factorul de Intensitate

IF = Tempo CSS / Tempo Antrenament
IF = 109 / 125
IF = 0.872

Pasul 2: Calculează sTSS

sTSS = 1.0 ore × (0.872)² × 100
sTSS = 1.0 × 0.760 × 100
sTSS = 76

Interpretare: Acest antrenament de 60 minute la tempo ușor (mai lent decât CSS) a generat 76 sTSS - o sarcină de antrenament moderată potrivită pentru construirea bazei aerobice.

Interpretarea Valorilor sTSS

Ghid sTSS Săptămânal

Ținta de sTSS săptămânal depinde de nivelul tău de antrenament și obiective:

Înotători Recreativi

sTSS Săptămânal: 150-300

2-3 antrenamente pe săptămână, 50-100 sTSS fiecare. Focus pe tehnică și construire bază aerobică.

Înotători Fitness / Triatloniști

sTSS Săptămânal: 300-500

3-4 antrenamente pe săptămână, 75-125 sTSS fiecare. Mix de rezistență aerobică și lucru de prag.

Înotători Master Competitivi

sTSS Săptămânal: 500-800

4-6 antrenamente pe săptămână, 80-150 sTSS fiecare. Antrenament structurat cu periodizare.

Înotători Elită / Universitari

sTSS Săptămânal: 800-1200+

8-12 antrenamente pe săptămână, sesiuni duble. Volum mare cu gestionare critică a recuperării.

⚠️ Note Importante

  • Necesită CSS precis: CSS-ul tău trebuie actualizat (testat în ultimele 6-8 săptămâni) pentru un sTSS precis.
  • Calcul simplificat: Acest calculator folosește tempoul mediu. sTSS-ul avansat folosește Normalized Graded Pace (NGP) care ia în considerare structura intervalelor.
  • Nu pentru lucru tehnic: sTSS-ul măsoară doar stresul fizic al antrenamentului, nu dezvoltarea abilităților.
  • Variație individuală: Același sTSS se simte diferit pentru înotători diferiți. Ajustează ghidurile conform recuperării tale.

De Ce Contează sTSS

Training Stress Score este baza pentru:

  • CTL (Chronic Training Load): Nivelul tău de formă fizică - medie ponderată exponențial pe 42 zile a sTSS-ului zilnic
  • ATL (Acute Training Load): Oboseala ta - medie ponderată exponențial pe 7 zile a sTSS-ului zilnic
  • TSB (Training Stress Balance): Starea ta de formă - TSB = CTL - ATL (pozitiv = proaspăt, negativ = obosit)
  • Periodizare: Planificare faze antrenament (bază, dezvoltare, vârf, coborâre) folosind progresii CTL țintă
  • Gestionare Recuperare: A ști când să împingi și când să te odihnești conform TSB

Sfat Pro: Urmărește-ți CTL-ul

Înregistrează sTSS-ul zilnic într-o foaie de calcul sau jurnal de antrenament. Calculează media ta pe 42 zile (CTL) săptămânal. Țintește creștere 5-10 puncte CTL pe săptămână în timpul construirii bazei. Menține sau reduce ușor CTL-ul în timpul coborârii (1-2 săptămâni înainte de competiție).

Întrebări Frecvente

Ce este TSS pentru Înot (sTSS)?

Swimming Training Stress Score (sTSS) este o metrică care cuantifică sarcina de antrenament dintr-o sesiune de înot combinând atât intensitatea, cât și durata. Este adaptat din metodologia TSS a ciclismului, folosind Critical Swim Speed (CSS) ca tempo de prag. Un antrenament de 1 oră la tempo CSS este egal cu 100 sTSS.

Cum îmi calculez sTSS-ul?

Folosește calculatorul de mai sus introducând tempoul tău CSS (dintr-un test CSS), durata totală a antrenamentului și tempoul mediu. Formula este: sTSS = Durată (ore) × Factor Intensitate² × 100, unde Factor Intensitate = Tempo CSS / Tempo Mediu Antrenament.

Am nevoie de CSS pentru a calcula sTSS?

Da, Critical Swim Speed (CSS) este necesar pentru a calcula Factorul de Intensitate, care este esențial pentru calculul sTSS. CSS reprezintă tempoul tău de prag și ar trebui testat la fiecare 6-8 săptămâni. Poți găsi CSS-ul tău folosind calculatorul nostru CSS.

Ce este un scor sTSS bun pentru un antrenament?

Depinde de intensitatea antrenamentului: Antrenamentele ușoare au de obicei sub 50 sTSS, cele moderate 50-100 sTSS, cele grele 100-200 sTSS, iar cele foarte grele peste 200 sTSS. Scorul adecvat depinde de obiectivele tale de antrenament și nivelul actual de formă fizică.

Cât sTSS ar trebui să fac pe săptămână?

Țintele săptămânale de sTSS variază în funcție de nivel: Înotători recreativi: 150-300, Înotători fitness/Triatloniști: 300-500, Master Competitivi: 500-800, Elită/Universitari: 800-1200+. Începe conservator și crește gradual pentru a evita supraantrenamentul.

TSS-ul de înot este același cu TSS-ul de ciclism?

Conceptul și formula sunt aceleași, dar sTSS este adaptat pentru înot. În loc să folosească puterea (FTP) ca TSS-ul de ciclism, sTSS folosește ritmul cu CSS ca prag. Ambele măsoară sarcina de antrenament folosind Durată × Factor de Intensitate² × 100.

Pot folosi sTSS pentru toate stilurile de înot?

Da, dar CSS-ul tău ar trebui să fie specific stilului. Majoritatea înotătorilor folosesc CSS crawl deoarece este stilul cel mai frecvent antrenat. Dacă antrenezi în principal un alt stil, efectuează un test CSS în acel stil și folosește acel ritm pentru calculul sTSS.

Care este diferența dintre sTSS și CTL/ATL/TSB?

sTSS măsoară sarcina de antrenament a unui singur antrenament. CTL (Chronic Training Load) este forma ta pe termen lung, ATL (Acute Training Load) este oboseala ta recentă, iar TSB (Training Stress Balance) este prospețimea ta. Aceste valori folosesc valorile sTSS în timp pentru a urmări starea ta de antrenament. Află mai multe în ghidul nostru de Sarcină de Antrenament.
