Безкоштовний калькулятор для плавання TSS

Розрахунок показника стресу під час тренування з плавання - єдиний безкоштовний калькулятор sTSS

Що таке Swimming TSS (sTSS)?

Оцінка стресу під час тренування з плавання (sTSS)кількісно визначає навчальне навантаження під час тренування з плавання шляхом поєднання інтенсивності та тривалості. Він адаптований на основі методології TSS для велоспорту, використовуючи вашу критичну швидкість плавання (CSS) як пороговий темп. 1-годинне тренування в темпі CSS = 100 sTSS.

Інтерпретація тренувального навантаження

Заповнено JavaScript

Як розраховується sTSS

Формула

sTSS = (Тривалість у годинах) × (Коефіцієнт інтенсивності)³ × 100

Де:

  • Фактор інтенсивності (IF)= Темп CSS / Середній темп тренування
  • Тривалість= Загальний час тренування в годинах
  • Темп CSS= Ваш пороговий темп із тесту CSS

Спрацьований приклад

Деталі тренування:

  • Темп CSS:1:49/100м(109 секунд)
  • Тривалість тренування:60 хвилин(1 година)
  • Середній темп:2:05/100м(125 секунд)

Крок 1: Обчисліть коефіцієнт інтенсивності

IF = Темп CSS / Темп тренування
ЗАХИСТИТИ15X = 109 / 125
ЗАХИСТИТИ15X =0,872

Крок 2: обчисліть sTSS

sTSS = 1,0 години × (0,872)² × 100
sTSS = 1,0 × 0,760 × 100
ЗАХИСТИТИ13X =76

Інтерпретація:Це 60-хвилинне тренування в легкому темпі (повільніше, ніж CSS) створило 76 sTSS - помірне тренувальне навантаження, придатне для аеробного формування бази.

Розуміння значень sTSS

Асортимент sTSS Тренувальне навантаження Час відновлення Приклад тренування
< 50 Низький Той самий день Легке 30-хвилинне плавання, технічне вправляння
50-100 Помірний 1 день 60-хвилинна витривалість, рівномірний темп
100-200 Високий 1-2 дні 90-хвилинні порогові набори, інтервали гоночного темпу
200-300 Дуже висока 2-3 дні 2-годинне важке тренування, кілька порогових блоків
> 300 Екстрім 3+ днів Довга гонка (>2 годин), ультра-витривалість

Щотижневі рекомендації sTSS

Тижнева мета sTSS залежить від вашого рівня підготовки та цілей:

Рекреаційні плавці

Щотижня sTSS:150-300

2-3 тренування на тиждень, 50-100 sTSS кожне. Зосередьтеся на техніці та формуванні аеробної бази.

Фітнес-плавці / триатлоністи

Щотижня sTSS:300-500

3-4 тренування на тиждень, 75-125 sTSS кожне. Поєднання аеробної витривалості та порогової роботи.

Змагання плавців-майстрів

Щотижня sTSS:500-800

4-6 тренувань на тиждень, 80-150 sTSS кожна. Структуроване навчання з періодизацією.

Елітні/Університетські плавці

Щотижня sTSS:800-1200+

8-12 тренувань на тиждень, дводенні. Великий обсяг із критичним керуванням відновленням.

⚠️ Важливі примітки

  • Потрібен точний CSS:Ваш CSS має бути актуальним (перевірено протягом 6-8 тижнів) для точного sTSS.
  • Спрощений розрахунок:Цей калькулятор використовує середній темп. Advanced sTSS використовує нормалізований темп (NGP), який враховує структуру інтервалів.
  • Не для роботи з технікою:sTSS вимірює лише фізичне тренування, а не розвиток навичок.
  • Індивідуальна варіація:Один і той самий sTSS відчувається по-різному для різних плавців. Відкоригуйте вказівки залежно від свого одужання.

Чому sTSS важливий

Рейтинг тренувального стресу є основою для:

  • CTL (хронічне навантаження під час тренувань):Ваш рівень фізичної підготовки - 42-денне експоненціально зважене середнє щоденне значення sTSS
  • ATL (Гостре тренувальне навантаження):Ваша втома – 7-денне експоненціально зважене середнє щоденне значення sTSS
  • TSB (Баланс тренувального стресу):Ваша форма - TSB = CTL - ATL (позитив = свіжий, негатив = втомлений)
  • Періодизація:Плануйте фази навчання (базова, нарощування, пік, звуження), використовуючи цільові прогресії CTL
  • Управління відновленням:Знайте, коли наполягати, а коли відпочивати на основі TSB

Професійна порада: відстежуйте свій CTL

Записуйте щоденний sTSS в електронну таблицю або журнал навчання. Обчислюйте середнє значення за 42 дні (CTL) щотижня. Цільове збільшення 5-10 очок CTL на тиждень під час будівництва бази. Збережіть або трохи зменшіть CTL під час звуження (за 1-2 тижні до гонки).

Часті запитання

Що таке Swimming TSS (sTSS)?

Swimming Training Stress Score (sTSS) — це показник, який визначає кількісне навантаження під час тренування з плавання шляхом поєднання інтенсивності та тривалості. Він адаптований на основі методології TSS для велоспорту, використовуючи вашу критичну швидкість плавання (CSS) як пороговий темп. Тренування тривалістю 1 година в темпі CSS дорівнює 100 sTSS.

Як розрахувати свій sTSS?

Скористайтеся калькулятором вище, ввівши свій темп CSS (з тесту CSS), загальну тривалість тренування та середній темп під час тренування. Формула: sTSS = тривалість (години) × коефіцієнт інтенсивності³ × 100, де коефіцієнт інтенсивності = темп CSS / середній темп тренування.

Чи потрібен мені CSS для обчислення sTSS?

Так, ваша критична швидкість плавання (CSS) потрібна для розрахунку коефіцієнта інтенсивності, який є важливим для розрахунку sTSS. CSS представляє ваш пороговий темп, і його слід перевіряти кожні 6-8 тижнів. Ви можете знайти свій CSS за допомогою нашогоКалькулятор CSS.

Яка хороша оцінка sTSS для одного тренування?

Це залежить від інтенсивності тренувань: легкі тренування зазвичай мають оцінку нижче 50 sTSS, помірні тренування 50-100 sTSS, важкі тренування 100-200 sTSS і дуже важкі тренування вище 200 sTSS. Відповідний бал залежить від ваших тренувальних цілей і поточного рівня фізичної підготовки.

Скільки sTSS я повинен робити на тиждень?

Щотижневі цілі sTSS відрізняються залежно від рівня: плавці-любителі: 150-300, плавці-тренажери/триатлоністи: 300-500, змагальні майстри: 500-800, еліта/університет: 800-1200+. Почніть консервативно і поступово збільшуйте, щоб уникнути перетренованості.

Чи плавання TSS те саме, що їзда на велосипеді TSS?

Концепція та формула ті самі, але sTSS адаптований для плавання. Замість використання потужності (FTP), наприклад циклічного TSS, sTSS використовує швидкість із CSS як порогове значення. Обидва вимірювання тренувального навантаження використовують Тривалість × Коефіцієнт інтенсивності³ × 100.

Чи можу я використовувати sTSS для всіх видів плавання?

Так, але ваш CSS має бути специфічним для штрихів. Більшість плавців використовують CSS у вільному стилі, оскільки це гребок, який найчастіше тренують. Якщо ви тренуєтеся переважно в іншому гребку, виконайте тест CSS у цьому гребку та використовуйте цей темп для розрахунку sTSS.

Яка різниця між sTSS і CTL/ATL/TSB?

sTSS вимірює навантаження під час одного тренування. CTL (хронічне навантаження під час тренування) — це ваша довгострокова фізична форма, ATL (гостре навантаження під час тренування) — це ваша нещодавня втома, а TSB (баланс стресу під час тренування) — ваша свіжість. Ці показники використовують значення sTSS протягом певного часу, щоб відстежувати стан вашого тренування. Дізнайтеся більше в нашомуІнструкція з тренувального навантаження.
